物理公式汇总

力学 · 热学 · 光学 · 电学 · 公式大全

使用说明：公式中所有物理量均使用国际单位制（SI），注意单位换算。计算时先统一单位再代入。

力学基础公式

速度
v = s/t
s 路程(m)，t 时间(s)，v 速度(m/s)
速度换算
1 m/s = 3.6 km/h
米每秒与千米每小时换算
密度
ρ = m/V
m 质量(kg)，V 体积(m³)，ρ 密度(kg/m³)
重力
G = mg (g=9.8N/kg)
质量 × 重力加速度，粗略计算 g 取 10
压强
p = F/S
F 压力(N)，S 受力面积(m²)
液体压强
p = ρgh
ρ 密度(kg/m³)，h 深度(m)
大气压
p₀ = 1.013×10⁵ Pa
标准大气压值，76cmHg

浮力公式

阿基米德原理
F浮 = ρ液gV排
浮力等于排开液体所受重力
称重法
F浮 = G - F示
空气中重力减去弹簧测力计示数
漂浮/悬浮
F浮 = G (物体漂浮或悬浮时)
二力平衡，浮力等于重力
浮沉条件
ρ物 < ρ液 上浮
密度关系决定物体浮沉

简单机械

杠杆平衡条件
F₁L₁ = F₂L₂
动力 × 动力臂 = 阻力 × 阻力臂
滑轮组拉力
F = G总/n
n 为承担重物的绳子段数
滑轮组距离
s = nh
绳子自由端移动距离 = n × 物体升高高度
斜面公式
FL = Gh
理想斜面，不计摩擦
机械效率
η = W有/W总 × 100%
有用功与总功的百分比

功与能公式

功
W = Fs (F 与 s 同向)
力 × 沿力方向移动的距离
功率
P = W/t = Fv
功与做功时间之比
动能
Eₖ = ½ mv²
质量越大、速度越大，动能越大
重力势能
Eₚ = mgh
质量越大、高度越高，势能越大
弹性势能
Eₘ = ½ kx²
k 劲度系数，x 形变量
机械能守恒
Eₖ₁ + Eₚ₁ = Eₖ₂ + Eₚ₂
只有重力/弹力做功时守恒

热学公式

比热容
Q = cmΔt
c 比热容(J/(kg·℃))，Δt 温度变化量
热值
Q = mq (固体/液体)
q 热值(J/kg)
热值（气体）
Q = Vq
V 体积(m³)
热机效率
η = W/Q放 × 100%
有用功与燃料释放热量之比

电学公式

欧姆定律
I = U/R
电流与电压成正比，与电阻成反比
串联总电阻
R总 = R₁ + R₂ + ...
串联电阻越串越大
并联总电阻
1/R总 = 1/R₁ + 1/R₂ + ...
并联电阻越并越小
电功
W = UIt = I²Rt = U²t/R
电流做功的多少
电功率
P = UI = I²R = U²/R
电流做功快慢
焦耳定律
Q = I²Rt
电流通过导体产生的热量
串联分压
U₁/U₂ = R₁/R₂
串联电路电压与电阻成正比
并联分流
I₁/I₂ = R₂/R₁
并联电路电流与电阻成反比
电学注意事项：（1）电流表要串联在电路中，电压表要并联；（2）家庭电路电压 220V，安全电压不高于 36V；（3）电能表读数最后一位为小数。

光学公式

反射定律
∠入 = ∠反
三线共面、两线分居、两角相等
平面镜成像
物距 = 像距
等大、等距、正立、虚像，对称
折射定律
n = sini/sinr
折射率 = 入射角正弦 / 折射角正弦
凸透镜成像规律
1/f = 1/u + 1/v
f 焦距，u 物距，v 像距
放大率
m = |v/u| = 像高/物高
像与物的大小之比
