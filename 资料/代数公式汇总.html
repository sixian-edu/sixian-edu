专题一：幂的运算法则

计算基础

幂运算是代数运算的根基，整式的乘除、方程变换都离不开幂的运算。记住：幂运算的核心是"底数不变、指数操作"。

幂运算层级关系与示例
同底数幂相乘
aᵐ · aⁿ = aᵐ⁺ⁿ
底数不变，指数相加。如 x³·x² = x⁵
同底数幂相除
aᵐ ÷ aⁿ = aᵐ⁻ⁿ (a≠0)
底数不变，指数相减。如 x⁵÷x² = x³
幂的乘方
(aᵐ)ⁿ = aᵐⁿ
底数不变，指数相乘。如 (x²)³ = x⁶
积的乘方
(ab)ⁿ = aⁿbⁿ
分别乘方。如 (2x)³ = 8x³
零指数幂
a⁰ = 1 (a≠0)
任何非零数的0次幂=1。0⁰无意义
负整数指数
a⁻ⁿ = 1/aⁿ (a≠0)
负指数=正指数的倒数。如 2⁻²=¼
分数指数（根式）
a^(m/n) = ⁿ√aᵐ
分子为指数、分母为根指数
记忆技巧：同底数幂乘除对比记——"相乘指数加、相除指数减"；幂的乘方"指数相乘"。注意区分(aᵐ)ⁿ（幂的乘方）与aᵐ·aⁿ（同底数幂相乘）！
【例题】计算：(1) x⁵·x³ (2) (a²)⁴ (3) (3a²b)² (4) 2⁻³
解：
(1) x⁵·x³ = x⁵⁺³ = x⁸（同底数幂相乘，指数相加）
(2) (a²)⁴ = a²ˣ⁴ = a⁸（幂的乘方，指数相乘）
(3) (3a²b)² = 3²·a⁴·b² = 9a⁴b²（积的乘方，每个因式分别乘方）
(4) 2⁻³ = 1/2³ = 1/8（负指数化为正指数的倒数）
幂运算最容易错的是"(aᵐ)ⁿ"和"aᵐ·aⁿ"的混淆。口诀："外面有括号是指数相乘，没括号是指数相加"。
• 计算：(1) x²·x⁴ (2) (x²)⁴ (3) x⁸÷x² (4) (-2a³b²)³

专题二：乘法公式与因式分解

核心运算

乘法公式是代数运算的工具，而因式分解是乘法公式的逆运算。掌握它们之间的互逆关系，能大幅提升计算速度和准确率。

(a+b)² = a²+2ab+b² 的几何意义——面积分割法

七个核心乘法公式

平方差公式
(a+b)(a-b)=a²-b²
"两项平方作差"
完全平方和
(a+b)²=a²+2ab+b²
"首平方+尾平方+二倍积"
完全平方差
(a-b)²=a²-2ab+b²
中间符号为负
立方和
(a+b)(a²-ab+b²)=a³+b³
注意ab前的符号是负
立方差
(a-b)(a²+ab+b²)=a³-b³
与立方和对比记忆
完全立方和
(a+b)³=a³+3a²b+3ab²+b³
二项式展开
完全立方差
(a-b)³=a³-3a²b+3ab²-b³
注意符号交替

因式分解五种方法

提公因式法
ma+mb+mc=m(a+b+c)
先看是否有公因式可提
公式法
逆用乘法公式分解
平方差/完全平方/立方和差
十字相乘法
x²+(p+q)x+pq=(x+p)(x+q)
二次项系数为1时最常用
十字相乘（一般）
ax²+bx+c拆成(px+q)(rx+s)
pr=a, qs=c, ps+qr=b
分组分解法
分组后各自提取公因式
四项及以上时使用
易错提醒：① (a+b)² = a²+2ab+b²，中间项有"2"！② 平方差公式中a和b可以是单项式也可以是多项式；③ 因式分解一定要分解到不能再分为止。
【例题】分解因式：(1) x²-9 (2) x²+6x+9 (3) x²-5x+6 (4) 2x²-8
解：
(1) x²-9 = (x+3)(x-3) —— 平方差公式
(2) x²+6x+9 = (x+3)² —— 完全平方公式
(3) x²-5x+6 = (x-2)(x-3) —— 十字相乘法（找两数积为6、和为-5）
(4) 2x²-8 = 2(x²-4) = 2(x+2)(x-2) —— 先提公因式再用平方差
因式分解的做题顺序口诀："一提公因式、二套公式、三十字、四分组"。按此顺序，不走弯路。
• 计算：(1) (2x+1)(2x-1) (2) (3a-2b)²
• 分解因式：(1) a²-4b² (2) x²+8x+16 (3) x²-3x-10 (4) 3x³-12x

专题三：方程与方程组

中考重点

方程是代数的核心工具——用方程来刻画数量关系，是解决实际问题的核心方法。一元二次方程的解法是重中之重。

一元一次方程

标准形式
ax + b = 0 (a≠0)
解为 x = -b/a

二元一次方程组

代入消元法
用一个未知数表示另一个
适合某系数为±1时
加减消元法
两方程加减消去一个元
系数成倍数关系时最方便

一元二次方程 — 核心公式

一般形式
ax²+bx+c=0 (a≠0)
a是二次项系数、b是一次项系数、c是常数
求根公式
x=[-b±√(b²-4ac)]/(2a)
万能法，任何一元二次方程都可用
判别式
Δ = b²-4ac
Δ>0两不等实根；Δ=0两等根；Δ<0无实根
韦达定理
x₁+x₂=-b/a, x₁x₂=c/a
根与系数的关系（Δ≥0时适用）
记忆技巧：求根公式口诀——"负b加减根号下，b方减4ac，除以2a"。韦达定理——"两根和等于负b比a，两根积等于c比a"。
【例题】解方程 x²-5x+6=0，并验证韦达定理。
解法一（因式分解）：
x²-5x+6 = (x-2)(x-3) = 0 → x₁=2, x₂=3

解法二（求根公式）：
a=1, b=-5, c=6
Δ = b²-4ac = 25-24 = 1 > 0
x = [5±1]/2 → x₁=3, x₂=2

韦达定理验证：
x₁+x₂ = 2+3 = 5 = -(-5)/1 = 5 ✓
x₁·x₂ = 2×3 = 6 = 6/1 = 6 ✓
能用因式分解的优先用因式分解（更快），不能分解的用求根公式。考题中经常要求"用两种方法解"，所以两种方法都要熟练掌握。
• 解方程：(1) x²-2x-3=0 (2) 2x²+3x-2=0 (3) x²-6x+9=0
• 已知一元二次方程 x²-4x+k=0 有两个相等的实数根，求k的值。

专题四：不等式

逻辑思维

不等式与方程最大的区别在于：方程等号两边操作一致，而不等式乘除负数时不等号方向会改变。这是最关键也是最易错的知识点。

不等式的基本性质

性质1
a>b ⇒ a±c>b±c
两边同加/减同一数，不等号不变
性质2
a>b, c>0 ⇒ ac>bc
乘正数不等号不变
性质3（关键！）
a>b, c<0 ⇒ ac
乘负数不等号改变方向！

不等式的解法

一元一次不等式
ax+b>0或<0
解法同方程，注意系数为负时变号
一元二次不等式
ax²+bx+c>0
利用二次函数图像：大于取两边、小于取中间
一元一次不等式组
两个不等式的公共部分
口诀：同大取大、同小取小、大小小大中间找
一级易错点：解不等式 -2x > 6 时，两边同除以 -2，不等号要改变方向 → x < -3。很多同学做成 x > -3 就错了！
检验方法：取一个"答案"代入原不等式，如果不成立说明方向错了。
【例题】解不等式组：{ 2x-1 > 3, x+2 ≤ 5 }
解：
由 2x-1 > 3 得 2x > 4 → x > 2
由 x+2 ≤ 5 得 x ≤ 3
∴ 不等式组的解集为 2 < x ≤ 3
数轴表示：在数轴上，2处画空心点（不含），3处画实心点（含），中间部分为解集。
记牢不等式组的"同大取大、同小取小、大小小大中间找、大大小小找不到"口诀。本题是"x>2和x≤3"，属于"大小小大中间找"。
• 解不等式：3x+7 > 2x-1
• 解不等式组：{ x-3>0, 2x+1<9 }，并用数轴表示解集。

专题五：函数公式与图像

数形结合

函数是中考的压轴题常客。一次函数、二次函数、反比例函数是初中必须掌握的三大函数，理解它们的图像特征是解题的关键。

二次函数 y = ax² + bx + c 的图像特征对比

三大函数公式

一次函数
y = kx+b (k≠0)
k>0递增、k<0递减；b为y截距
正比例函数
y = kx (k≠0)
过原点(0,0)的一次函数
反比例函数
y = k/x (k≠0)
双曲线；k>0一三象限、k<0二四象限
二次函数（一般式）
y=ax²+bx+c (a≠0)
a>0开口向上、a<0开口向下
二次函数（顶点式）
y=a(x-h)²+k
顶点(h,k)；h=-b/(2a), k=(4ac-b²)/(4a)
二次函数（交点式）
y=a(x-x₁)(x-x₂)
x₁、x₂为与x轴交点横坐标

函数图像关键特征

函数图像形状增减性对称性
一次函数直线k>0单调增；k<0单调减/
反比例函数双曲线k>0在各象限内递减；k<0递増关于原点中心对称
二次函数抛物线对称轴左侧与右侧增减相反x=-b/(2a)
关键理解：二次函数顶点式的三种变换——h控制左右平移（左加右减），k控制上下平移（上加下减），a控制开口方向和大小。
【例题】已知二次函数 y = x² - 4x + 3，(1)求顶点坐标；(2)求与x轴交点；(3)当x取何值时，y>0？
解：
(1) a=1, b=-4, c=3
对称轴 x = -b/(2a) = 4/2 = 2
顶点纵坐标 y = 2²-8+3 = -1 → 顶点(2,-1)

(2) 令y=0: x²-4x+3=0 → (x-1)(x-3)=0 → x₁=1, x₂=3
与x轴交于(1,0)和(3,0)

(3) 抛物线开口向上，与x轴交于1和3
∴ 当 x<1 或 x>3 时，y>0
二次函数题目的核心是找到"顶点、对称轴、与x轴交点"三个要素。把它们找到，函数的性质就清楚了。
• 已知二次函数 y = -x² + 2x + 3，求顶点坐标和与x轴的交点，并判断开口方向。
• 一次函数 y=2x-4 与 x 轴、y 轴的交点坐标分别是什么？

专题六：数列公式

规律探究

数列考查的是规律发现能力。等差和等比数列是两种最基本的数列模型，它们的通项公式和求和公式必须熟练掌握。

等差数列与等比数列规律对比

等差数列（差为定值）

通项公式
aₙ = a₁ + (n-1)d
d为公差。aₙ = aₘ + (n-m)d
前n项和
Sₙ = n(a₁+aₙ)/2
首项加末项乘以项数除以2
Sₙ的另一种形式
Sₙ = na₁ + n(n-1)d/2
已知a₁和d时用

等比数列（比为定值）

通项公式
aₙ = a₁ · qⁿ⁻¹
q为公比（q≠0）
前n项和（q≠1）
Sₙ = a₁(1-qⁿ)/(1-q)
注意分母符号不要写反
q=1的情况
Sₙ = n·a₁
公比为1时就是常数数列
易错提醒：① 使用等差求和公式时，注意n是项数不是末项序号；② 等比求和公式中q=1时单独处理；③ 等差数列的Sₙ是关于n的二次函数（无常数项）。
【例题】已知等差数列{an}中，a₁=3，a₅=11，求通项公式和前10项的和。
解：
a₅ = a₁ + 4d → 11 = 3 + 4d → 4d = 8 → d = 2
通项公式：aₙ = 3 + (n-1)×2 = 2n + 1

S₁₀ = 10(3+21)/2 = 10×24/2 = 120
或 S₁₀ = 10×3 + 10×9×2/2 = 30 + 90 = 120
求等差数列的关键是找到"首项a₁"和"公差d"。给了两个独立条件，列方程组即可解出。
• 等差数列 -3, 1, 5, 9, ... 的第10项是多少？前10项和是多少？
• 等比数列 2, 4, 8, 16, ... 的第6项和前6项和分别是多少？

专题七：统计与概率

数据分析

统计与概率是应用型知识，中考中常以实际情境为载体考查。核心是理解数据特征量的意义，以及概率的计算方法。

统计数据分析：均值相同方差不同、正态分布对比

数据的集中趋势（中心位置）

算术平均数
x̄ = Σxᵢ/n
所有数据之和÷数据个数。受极端值影响大
加权平均数
x̄ = Σ(wᵢxᵢ)/Σwᵢ
各数据乘以权重再求和÷总权重
中位数
排序后中间的数
n为奇数取中间；n为偶数取中间两数平均
众数
出现次数最多的数据
可以不止一个，也可以没有

数据的离散程度（波动大小）

极差
R = max - min
最大值减最小值，最简单的波动度量
方差
s² = Σ(xᵢ-x̄)²/n
各数据与平均数的差的平方和÷n。值越大越分散
标准差
s = √s²
方差的算术平方根，单位与原始数据相同

概率基础公式

等可能事件概率
P(A) = m/n
m为A包含的基本事件数，n为总事件数
互斥事件
P(A∪B)=P(A)+P(B)
A和B不能同时发生
对立事件
P(Ā) = 1 - P(A)
A不发生的概率
理解要点：方差越大→数据越分散→越不稳定。在实际应用中，方差小意味着产品质量更稳定、运动员发挥更稳定。
【例题】甲、乙两名同学的5次数学测验成绩如下：
甲：85, 90, 88, 92, 85（平均=88）
乙：70, 95, 85, 90, 100（平均=88）
两人平均分相同，谁的成绩更稳定？
解：
s²甲 = [(85-88)²+(90-88)²+(88-88)²+(92-88)²+(85-88)²]/5
= (9+4+0+16+9)/5 = 38/5 = 7.6

s²乙 = [(70-88)²+(95-88)²+(85-88)²+(90-88)²+(100-88)²]/5
= (324+49+9+4+144)/5 = 530/5 = 106

7.6 < 106，所以甲的成绩更稳定。
平均分相同的情况下，方差越小成绩越稳定。甲的方差远小于乙，说明甲是"稳定型"选手，乙是"波动型"选手。
• 一组数据为：3, 5, 7, 9, 11，求平均数、中位数、极差、方差。
• 从1、2、3三个数字中随机抽取两个组成两位数，求组成的两位数是偶数的概率。

专题八：代数综合应用与解题策略

提分必看

代数综合题往往把方程、函数、不等式融合在一起，考查学生的综合运用能力。掌握以下解题策略，能帮助你在考场上快速找到突破口。

常见综合题型与解题策略

题型特点解题策略示例
方程-函数综合给出函数图像上的点满足方程关系将点的坐标代入函数解析式构造方程抛物线上某点在直线y=2x+1上
不等式-函数综合求x取何值时函数值>0或<0转化为解二次不等式，利用图像判断求y=x²-4x+3>0的解集
实际应用题工程、行程、利润等问题设未知数→列方程/不等式→求解→检验某商品降价求最大利润
规律探究题给出数列或图形找规律列出前几项→找通项公式→代入求解摆火柴棍的规律探索

实际应用题建模流程

审题→设未知数→找等量关系→列方程(组)→解方程(组)→检验→作答
"检验"这一步最容易被忽略！要注意解的合理性（比如人数不能为负数、时间不能为负）

典型的代数易错题型

分式方程增根
解后必须检验
去分母后可能产生增根，代入公分母验证是否为0
二次方程讨论
a≠0是前提
说"二次方程"时隐含a≠0；题目没说"二次"时要讨论a=0的情况
不等式变方向
乘除负数要变号
最容易错的地方，没有之一！
【综合例题】某商店将进货单价为40元的商品按50元售出时，能卖出500件。已知每涨价1元，销售量就减少10件。问售价定为多少时，能使利润最大？最大利润是多少？
解：设涨价x元（x≥0），则售价为(50+x)元，销售量为(500-10x)件。
利润 y = (售价-进价)×销售量 = (50+x-40)(500-10x)
= (10+x)(500-10x)
= 5000 + 500x - 100x - 10x²
= -10x² + 400x + 5000
= -10(x² - 40x) + 5000
= -10(x² - 40x + 400) + 5000 + 4000
= -10(x-20)² + 9000

当x=20时，y取最大值9000元。
此时售价 = 50 + 20 = 70元。
利润最值问题本质上是二次函数求最值。关键步骤：①正确列出利润表达式；②配方求顶点坐标；③检验x是否合理（此题x=20≥0合理）。
• 将进货价为30元的商品以40元售出可卖600件，每涨价1元销量减少20件。求获得最大利润时的售价和最大利润。
• 回顾你最近做错的代数题，归类到以上"易错题型"中，针对性复习。
