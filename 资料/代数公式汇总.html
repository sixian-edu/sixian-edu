代数公式汇总

幂运算 · 乘法公式 · 因式分解 · 方程 · 函数 · 数列

幂的运算法则

同底数幂相乘
aᵐ · aⁿ = aᵐ⁺ⁿ
底数不变，指数相加
同底数幂相除
aᵐ ÷ aⁿ = aᵐ⁻ⁿ (a≠0)
底数不变，指数相减
幂的乘方
(aᵐ)ⁿ = aᵐⁿ
底数不变，指数相乘
积的乘方
(ab)ⁿ = aⁿ bⁿ
分别乘方后再相乘
零指数
a⁰ = 1 (a≠0)
任何非零数的零次幂都等于 1
负指数
a⁻ⁿ = 1/aⁿ
负指数等于正指数的倒数
分数指数
aᵐⁿ = ⁿ√aᵐ
分子为指数，分母为根指数

乘法公式

平方差公式
(a+b)(a-b) = a² - b²
两数和乘以两数差等于平方差
完全平方和
(a+b)² = a² + 2ab + b²
首平方、尾平方、二倍乘积在中央
完全平方差
(a-b)² = a² - 2ab + b²
首平方、尾平方、负二倍乘积在中央
立方和
(a+b)(a²-ab+b²) = a³ + b³
注意中间项符号
立方差
(a-b)(a²+ab+b²) = a³ - b³
注意中间项符号
完全立方和
(a+b)³ = a³+3a²b+3ab²+b³
二项式展开
完全立方差
(a-b)³ = a³-3a²b+3ab²-b³
二项式展开

因式分解方法

提公因式法
ma + mb + mc = m(a+b+c)
各项有公因式时优先提取
平方差公式
a² - b² = (a+b)(a-b)
两项均为平方且符号相反
完全平方
a²±2ab+b² = (a±b)²
三项，首尾平方中间二倍乘积
十字相乘法
x²+(p+q)x+pq = (x+p)(x+q)
二次项系数为 1
十字相乘法（一般）
ax²+bx+c = (px+q)(rx+s)
pr=a, qs=c, ps+qr=b
分组分解法
分组后提公因式或套公式
四项以上分组处理
立方和/差
a³±b³ = (a±b)(a²∓ab+b²)
按公式直接分解

一元二次方程

一般形式
ax² + bx + c = 0 (a≠0)
二次项系数不为零
求根公式
x = [-b ± √(b²-4ac)] / (2a)
万能公式，适用于所有一元二次方程
判别式
Δ = b² - 4ac
Δ>0 两不等实根；Δ=0 两等根；Δ<0 无实根
韦达定理
x₁ + x₂ = -b/a, x₁x₂ = c/a
根与系数的关系（Δ≥0 时适用）
配方法
x² + px = (x+p/2)² - (p/2)²
构造完全平方

不等式

基本性质1
a > b ⇒ a±c > b±c
两边加减同一数，不等号方向不变
基本性质2
a > b, c>0 ⇒ ac > bc
两边乘同一正数，不等号方向不变
基本性质3
a > b, c<0 ⇒ ac < bc
两边乘同一负数，不等号方向改变
一元一次不等式
ax + b > 0
解法与方程类似，注意乘除负数变号
一元二次不等式
ax² + bx + c > 0
利用二次函数图象求解（大于取两边，小于取中间）

函数公式

一次函数
y = kx + b (k≠0)
k>0 递增，k<0 递减，b 为 y 轴截距
正比例函数
y = kx (k≠0)
过原点，b=0 的特殊一次函数
反比例函数
y = k/x (k≠0)
k>0 在一三象限，k<0 在二四象限
二次函数一般式
y = ax²+bx+c (a≠0)
a>0 开口向上，a<0 开口向下
二次函数顶点式
y = a(x-h)² + k
顶点坐标为 (h, k)
二次函数交点式
y = a(x-x₁)(x-x₂)
x₁, x₂ 为与 x 轴交点横坐标
对称轴
x = -b/(2a)
二次函数对称轴方程
顶点坐标
(-b/(2a), (4ac-b²)/(4a))
二次函数顶点坐标公式

数列与统计

等差数列通项
aₙ = a₁ + (n-1)d
d 为公差
等差数列求和
Sₙ = n(a₁+aₙ)/2 = na₁ + n(n-1)d/2
首项加末项乘以项数除以 2
等比数列通项
aₙ = a₁ · qⁿ⁻¹
q 为公比
等比数列求和
Sₙ = a₁(1-qⁿ)/(1-q) (q≠1)
q=1 时 Sₙ = na₁
算术平均数
x̄ = (x₁+x₂+...+xₙ)/n
所有数据之和除以个数
方差
s² = Σ(xᵢ-x̄)²/n
反映数据波动大小，方差越小越稳定
标准差
s = √[Σ(xᵢ-x̄)²/n]
方差的算术平方根
