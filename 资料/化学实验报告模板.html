实验一：氧气的实验室制取

气体制取

实验目的

掌握实验室用加热高锰酸钾（KMnO₄）制取氧气的方法，学会排水集气法和验满操作。

实验原理

高锰酸钾受热分解：2KMnO₄ → K₂MnO₄ + MnO₂ + O₂↑（加热）

实验器材

铁架台（带铁夹）、酒精灯、大试管、单孔橡皮塞、导管、集气瓶、水槽、玻璃片、高锰酸钾、棉花。

实验步骤

  1. 查：检查装置气密性（手握试管法观察气泡）。
  2. 装：将高锰酸钾装入试管（管口放一小团棉花），塞紧带导管的橡皮塞。
  3. 定：将试管固定在铁架台上，管口略向下倾斜。
  4. 点：点燃酒精灯，先预热后集中加热。
  5. 收：气泡连续均匀放出后开始用排水法收集。
  6. 移：收集完毕后，先将导管移出水槽。
  7. 熄：再熄灭酒精灯（防止水倒吸炸裂试管）。

实验现象

试管中紫色固体逐渐变为黑色，有气体产生。带火星的木条伸入集气瓶中复燃，证明是氧气。

验满方法

排水法：集气瓶口有大气泡向外冒出说明已集满。向上排空气法：将带火星木条放在集气瓶口，木条复燃说明已满。

注意：试管口要略向下倾斜防止冷凝水回流炸裂试管。管口放棉花防止高锰酸钾粉末进入导管。实验结束先撤导管再熄酒精灯。

实验二：二氧化碳的实验室制取

气体制取

实验目的

掌握实验室用大理石（CaCO₃）和稀盐酸反应制取二氧化碳的方法。

实验原理

CaCO₃ + 2HCl → CaCl₂ + H₂O + CO₂↑

实验器材

锥形瓶（或大试管）、长颈漏斗、双孔橡皮塞、导管、集气瓶、玻璃片、大理石（石灰石）、稀盐酸。

实验步骤

  1. 检查装置气密性。
  2. 在锥形瓶中加入大理石，塞紧带长颈漏斗和导管的双孔橡皮塞。
  3. 从长颈漏斗加入稀盐酸，使液面浸没长颈漏斗下端管口。
  4. 用向上排空气法收集二氧化碳气体。
  5. 将燃着的木条放在集气瓶口，木条熄灭说明已集满。

实验现象

大理石表面有大量气泡产生，固体逐渐溶解。澄清石灰水倒入集气瓶中变浑浊（CO₂ 的检验方法）。

CO₂ 的检验

将气体通入澄清石灰水，石灰水变浑浊：CO₂ + Ca(OH)₂ → CaCO₃↓ + H₂O

注意：不能用稀硫酸代替稀盐酸（CaSO₄微溶，覆盖在大理石表面阻止反应继续）。长颈漏斗下端必须浸入液面以下防止气体逸出。

实验三：酸碱中和反应

酸碱盐

实验目的

通过实验验证酸和碱发生中和反应，理解中和反应的实质是 H⁺ + OH⁻ → H₂O。

实验原理

NaOH + HCl → NaCl + H₂O（放热反应）。中和反应实质：H⁺ + OH⁻ → H₂O

强碱滴定强酸滴定曲线：NaOH + HCl

实验器材

烧杯、量筒、胶头滴管、玻璃棒、温度计、pH试纸、NaOH稀溶液、稀盐酸、酚酞试液。

实验步骤

  1. 在烧杯中加入约5mL NaOH稀溶液，滴入2~3滴酚酞试液，溶液变红。
  2. 用胶头滴管逐滴加入稀盐酸，边加边用玻璃棒搅拌。
  3. 观察到溶液红色恰好褪去时停止滴加。
  4. 验证：用pH试纸测反应后溶液pH≈7；蒸发反应后溶液得到白色固体NaCl。

实验现象

溶液由红色逐渐变为无色（酚酞在碱性中显红色，中性时无色）。反应过程中温度升高（放热）。

指示剂变色规律

指示剂酸性中性碱性
石蕊变红紫色变蓝
酚酞无色无色变红
甲基橙变红橙色变黄
注意：中和反应放热。使用酚酞指示剂时注意：当溶液红色恰好消失时即为恰好完全反应，不要滴加过量酸。

实验四：探究金属活动性顺序

金属性质

实验目的

通过金属与酸、金属与盐溶液的反应，探究常见金属的活动性强弱。

实验原理

活动性强的金属能将活动性弱的金属从其盐溶液中置换出来。Fe + CuSO₄ → FeSO₄ + Cu

实验器材

试管、试管架、镊子、稀盐酸（或稀硫酸）、CuSO₄溶液、FeSO₄溶液、AgNO₃溶液、锌片、铁片、铜片、银片。

实验步骤

  1. 在四支试管中分别加入锌片、铁片、铜片、银片，各加入适量稀盐酸，观察现象。
  2. 将铁片放入CuSO₄溶液中，观察现象。
  3. 将铜片放入AgNO₃溶液中，观察现象。
  4. 将铜片放入FeSO₄溶液中，观察现象。

常见金属活动性顺序

K Ca Na Mg Al | Zn Fe Sn Pb (H) | Cu Hg Ag Pt Au（由强到弱）

实验结论

金属与盐酸反应与盐溶液反应活动性
Zn剧烈反应，产生大量气泡—★★★
Fe缓慢反应，有少量气泡能从CuSO₄中置换Cu★★
Cu不反应能从AgNO₃中置换Ag★
Ag不反应—☆
注意：金属与酸反应时，浓硫酸和硝酸有强氧化性，与金属反应不产生氢气。以上实验用稀盐酸或稀硫酸。

实验五：电解水实验

物质变化

实验目的

通过电解水实验验证水的组成，理解水是由氢元素和氧元素组成的。

实验原理

2H₂O → 2H₂↑ + O₂↑（通电）。水在通电条件下分解生成氢气和氧气，正极产生氧气，负极产生氢气，体积比 H₂:O₂ = 2:1。

实验器材

水电解器、直流电源、导线、试管、木条、火柴、蒸馏水（加稀硫酸或NaOH增强导电性）。

实验步骤

  1. 在水电解器中加入蒸馏水（已加入少量稀硫酸或NaOH），使液面达到适当高度。
  2. 接通直流电源，观察两个电极上的现象。
  3. 收集正负极产生的气体，记录体积比。
  4. 用带火星的木条检验正极气体（复燃→氧气）。
  5. 用点燃的木条检验负极气体（气体燃烧产生淡蓝色火焰→氢气）。

实验现象与结论

电极气体体积检验方法现象
正极（阳极）氧气 (O₂)1份带火星木条木条复燃
负极（阴极）氢气 (H₂)2份点燃淡蓝色火焰

水在通电条件下分解生成氢气和氧气，体积比 2:1，证明水是由氢、氧两种元素组成的化合物。

注意：纯水不导电，需加入少量稀硫酸或NaOH溶液增强导电性。电解使用的是直流电。H₂:O₂ 体积比理论上为 2:1，实际氧气略少（氧气微溶于水）。

实验六：粗盐的提纯

基本操作

实验目的

掌握粗盐提纯的基本操作——溶解、过滤、蒸发，学会使用玻璃棒在不同操作中的作用。

实验原理

粗盐中含有泥沙等不溶性杂质，利用过滤法除去不溶性杂质，再通过蒸发结晶得到较纯净的食盐。

实验器材

烧杯、玻璃棒、漏斗、铁架台（带铁圈）、滤纸、蒸发皿、酒精灯、坩埚钳、粗盐、水。

实验步骤

  1. 溶解：称取约5g粗盐，加入约15mL水，用玻璃棒搅拌加速溶解。
  2. 过滤：制作过滤器（一贴二低三靠），将粗盐溶液沿玻璃棒引流过滤。
  3. 蒸发：将滤液倒入蒸发皿，加热蒸发，用玻璃棒不停搅拌。
  4. 结晶：出现大量固体后停止加热，利用余热蒸干剩余水分。

玻璃棒的作用

操作玻璃棒作用
溶解搅拌，加速溶解
过滤引流，防止液体溅出
蒸发搅拌，使受热均匀，防止飞溅

实验结论

通过溶解—过滤—蒸发三个步骤，成功除去粗盐中的泥沙杂质，得到较纯净的食盐晶体。

注意：过滤操作要领：一贴（滤纸紧贴漏斗壁）、二低（滤纸边缘低于漏斗口、液面低于滤纸边缘）、三靠（烧杯嘴紧靠玻璃棒、玻璃棒轻靠三层滤纸处、漏斗下端管口紧靠烧杯内壁）。蒸发时出现大量固体就停止加热，用余热蒸干。
