⚡ 初中物理常考模型

力学·电学·热学·光学·实验模型·中考全覆盖

一、力学模型
🔹 受力分析模型
画力顺序："一重二弹三摩擦"
1. 重力 竖直向下，作用在重心（G=mg） 2. 弹力 支持力（垂直于接触面）、拉力（沿绳方向） 3. 摩擦力 与相对运动方向相反（注意有无摩擦力）
平衡状态：静止或匀速直线运动 → F合 = 0
常考场景： 水平面 G向下、N向上、F向右、f向左 斜面 G向下、N垂直斜面、f沿斜面向上/下 悬吊 G向下、绳拉力向上
🔹 运动模型
s-t图像：水平线→静止；斜向上直线→匀速直线（斜率=速度）
v-t图像：水平线→匀速；斜向上→加速；斜向下→减速
平均速度：v = s总/t总（中途停止时间要算入总时间）
惯性：只与质量有关，与速度无关。质量越大惯性越大
v = s/t  |  匀速：s = vt  |  v-t图像面积 = 路程
🔹 压强模型
固体压强：p = F/S。F=G（水平面），S取接触面积。
柱体对水平面：p = ρgh（与底面积无关，只与密度和高度有关）
液体压强：p = ρgh。深度 h 从自由液面竖直向下算。
特点：同一液体同一深度各处压强相等
大气压强：p₀ = 1.013×10⁵ Pa（托里拆利实验测出）
海拔越高气压越低 沸点随气压降低而降低 高压锅原理：限压阀→提高锅内气压→提高沸点
🔹 浮力模型
阿基米德原理：F浮 = G排 = ρ液gV排（与深度无关！）
浮沉条件：
状态条件特点
上浮F浮 > G最终漂浮（F浮=G）
悬浮F浮 = G可在液体中任意深度
下沉F浮 < G沉底后F浮+N=G
漂浮F浮 = GV排 < V物
密度计：漂浮→F浮=G→ρ液大V排小→刻度下大上小
船与潜艇：轮船漂浮→满载排水量m排=ρV排；潜艇改变自重实现浮沉
浮力四法：称重法F浮=G-F拉；阿基米德法F浮=ρ液gV排；平衡法F浮=G；压力差法F浮=F下-F上
F浮 = ρ液gV排  |  轮船：空心法  |  潜水艇：改变自重
🔹 简单机械模型
杠杆平衡：F₁L₁ = F₂L₂ 省力杠杆(L₁>L₂) 撬棍、瓶起子、钳子 费力杠杆(L₁ 镊子、钓鱼竿、扫帚
定滑轮：F=G，不省力改变方向（等臂杠杆）
动滑轮：F=G/2，省一半力不改变方向
滑轮组：F = G总/n，s = nh（n为承担重物绳子段数）
斜面：FL = Gh（理想），相同高度斜面越长越省力
🔹 功和机械能模型
功：W=Fs（力与距离同向） 不做功三情况：有力无距（搬石未动）、有距无力（惯性）、力距垂直（水平提桶）
功率：P=W/t=Fv（反映做功快慢） 机械效率：η = W有/W总 × 100% 滑轮组：η=Gh/Fs 斜面：η=Gh/FL η总小于1（额外功不可避免）
机械能守恒：动能+势能=常数（只有重力做功时） 滚摆：上升动能→重力势能，下降反之 过山车：最高点势能最大动能最小，最低点反之
W = Fs  |  P = W/t = Fv  |  η = W有/W总 × 100%
二、电学模型
🔹 电路识别模型
串并联识别法：电流路径法（唯一路径→串联，分支→并联）；节点法（用电器两端直接接电源→并联）
电流表：串联接入，正进负出，不能直接接电源
电压表：并联接入，正进负出，可以直接测电源电压
滑动变阻器："一上一下"接入，闭合开关前调到最大阻值
规律串联并联
电流I=I₁=I₂I=I₁+I₂
电压U=U₁+U₂U=U₁=U₂
电阻R=R₁+R₂R=(R₁R₂)/(R₁+R₂)
分压分流U₁/U₂=R₁/R₂I₁/I₂=R₂/R₁
🔹 欧姆定律模型
I = U/R（电流与电压成正比，与电阻成反比）
I-U图像：定值电阻→过原点直线；小灯泡→曲线（灯丝电阻随温度升高而增大）
串联分压：U₁/U₂ = R₁/R₂（电阻越大分压越多）
并联分流：I₁/I₂ = R₂/R₁（电阻越大分流越少）
动态电路分析步骤：
滑变电阻变化 → 总电阻变化 → 总电流变化 → 各支路/各元件电流电压变化
I = U/R  |  R = U/I（计算式，R与UI无关）
🔹 电功率模型
基本公式：P=W/t=UI=I²R=U²/R
电功：W=UIt=Pt  |  1度 = 1kW·h = 3.6×10⁶J
额定vs实际：灯泡亮度取决于实际功率 U实=U额 → P实=P额（正常发光） U实>U额 → 灯很亮（可能烧坏） U实
比例计算：串联时P₁/P₂=R₁/R₂；并联时P₁/P₂=R₂/R₁
动态电路极值问题：滑变在最小阻值时电流最大、总功率最大；滑变在最大阻值时电流最小、总功率最小
P = UI = I²R = U²/R  |  灯泡亮度看实际功率
🔹 电路故障分析模型
故障现象故障原因判断方法
灯不亮，IA=0断路UV=电源电压→所测部分断路
灯不亮，IA>0短路UV=0→所测部分短路
串联一灯亮一灯不亮不亮灯短路电流有通路
IA过大，烧保险/跳闸短路或超负荷总电阻过小/总功率过大
滑动变阻器调不动接成两上/两下两上=0，两下=最大不动
断路：IA=0，UV≈电源电压  |  短路：IA>0，UV=0
三、其他重点模型
🔹 光学模型
反射定律：三线共面、两线分居、两角相等（入射角=反射角）
平面镜成像：等大等距正立虚像——对称作图
折射规律：空气→水/玻璃：折射角<入射角（光从空气进入透明介质折射角变小）
凸透镜成像（重要！）： u>2f 倒立缩小实像（照相机）fu=2f 倒立等大实像（测焦距） f 倒立放大实像（投影仪）v>2f u 正立放大虚像（放大镜）
口诀："物近像远像变大，一倍焦距分虚实，二倍焦距分大小"
🔹 热学模型
六种物态变化：熔化（固→液，吸热）、凝固（液→固，放热）、汽化（液→气，吸热）、液化（气→液，放热）、升华（固→气，吸热）、凝华（气→固，放热）
晶体熔化：有熔点（冰、金属、海波），熔化时吸热温度不变
水沸腾：吸热温度不变（保持沸点），气压越高沸点越高
比热容：Q = cmΔt；c水=4.2×10³J/(kg·℃)（最大→常用于冷却剂）
热机四冲程：吸气→压缩(机械→内)→做功(内→机械)→排气
燃料热值：Q=mq；热机效率η=W有/Q放
🔹 电与磁模型
奥斯特实验：通电导体周围有磁场（电生磁）
安培定则：右手握螺线管，四指指电流方向，拇指指N极
电磁铁：磁性强弱与电流大小和线圈匝数有关
电动机：通电线圈在磁场中受力转动（电能→机械能），换向器改变电流方向
发电机：电磁感应（机械能→电能），交流电线圈转一圈电流方向改变两次
对比： 电生磁→奥斯特→电磁铁 电受力→电动机 磁生电→法拉第→发电机
