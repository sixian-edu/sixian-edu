力学
🔹 受力分析
对物体进行受力分析，按顺序：重力支持力拉力摩擦力
重力方向竖直向下，支持力垂直于接触面，摩擦力与相对运动方向相反。
G = mg  |  二力平衡：F合 = 0
🔹 重力
由于地球吸引而使物体受到的力，方向竖直向下（不是垂直向下）。
G = mgg ≈ 9.8 N/kg重心 重心是重力的作用点，形状规则且质量分布均匀的物体的重心在几何中心。
G = mg  |  重力方向：竖直向下
🔹 弹力与弹簧测力计
物体发生弹性形变时产生的力。弹力大小与形变程度有关。
弹簧测力计在弹性限度内，伸长量与拉力成正比
使用前要调零，观察量程和分度值，读数时视线与刻度盘垂直。
F = k·Δx  （在弹性限度内）
🔹 摩擦力
两个相互接触的物体发生相对运动或具有相对运动趋势时产生的力。
滑动摩擦力静摩擦力滚动摩擦力
影响因素：压力大小和接触面粗糙程度。滚动摩擦远小于滑动摩擦。
f = μN  |  增大摩擦：增大压力 / 增大粗糙度
🔹 二力平衡与牛顿第一定律
二力平衡条件：同体、等大、反向、共线。
牛顿第一定律 一切物体在不受力时，总保持静止或匀速直线运动状态。
惯性 物体保持原来运动状态的性质，只与质量有关。
F合 = 0 ⇔ 平衡状态  |  惯性只与质量有关
🔹 力的合成与平行四边形法则
两个力合成时，以表示这两个力的线段为邻边作平行四边形，对角线表示合力的大小和方向。
同向相加反向相减合力范围：|F₁−F₂| ≤ F ≤ F₁+F₂
F = √(F₁² + F₂² + 2F₁F₂·cosθ)  |  平行四边形法则
🔹 压强
物体单位面积上所受的压力。增大压强的方法：增大压力或减小受力面积。
p = F/S单位：Pa（帕斯卡）1 Pa = 1 N/m²
减小压强：增大受力面积（如书包带做宽、坦克履带）。
p = F/S  |  增大压强：增大F 或 减小S
🔹 液体压强
液体内部向各个方向都有压强，同种液体同一深度压强相等。
p = ρgh液体压强与深度成正比与液体密度成正比
连通器原理：同种液体静止时，各液面高度相同（茶壶、锅炉水位计、船闸）。
p = ρgh  |  连通器：液面相平
🔹 大气压强
大气对浸在它里面的物体产生的压强。马德堡半球实验证明了大气压的存在。
托里拆利实验 测出大气压值：p₀ = 1.013×10⁵ Pa
海拔越高，气压越低沸点随气压降低而降低
p₀ = 1.013×10⁵ Pa  |  海拔↑ 气压↓ 沸点↓
🔹 浮力 · 阿基米德原理
浸在液体中的物体受到向上的浮力，浮力大小等于物体排开液体的重力。
F浮 = ρ液 g V排浮力与深度无关
沉浮条件：F浮 > G 上浮F浮 = G 悬浮F浮 < G 下沉
F浮 = ρ液 g V排  |  轮船：空心法；潜水艇：改变自重
🔹 简单机械 · 杠杆
在力的作用下绕固定点转动的硬棒叫杠杆。
F₁L₁ = F₂L₂ 动力×动力臂 = 阻力×阻力臂
省力杠杆：L₁ > L₂（省力费距离）费力杠杆：L₁ < L₂（费力省距离）
生活中的杠杆：钳子（省力）、钓鱼竿（费力）、天平（等臂）。
F₁L₁ = F₂L₂  |  省力杠杆：L₁ > L₂
🔹 简单机械 · 滑轮
定滑轮 不省力，改变力的方向（实质为等臂杠杆）。
动滑轮 省一半力，不改变方向（实质为动力臂=2×阻力臂的杠杆）。
滑轮组 F = G/n，n为承担重物的绳子段数。
定滑轮：F = G  |  动滑轮：F = G/2  |  滑轮组：F = G/n
🔹 功与功率
功等于力与在力的方向上移动距离的乘积。
W = Fs单位：J（焦耳）
功率表示做功的快慢。P = W/t = Fv单位：W（瓦特）
不做功的三种情况：有力无距离、有距离无力、力与距离垂直。
W = Fs  |  P = W/t  |  P = Fv（匀速运动）
🔹 机械效率
有用功与总功的比值。机械效率总小于1（因为额外功不可避免）。
η = W有 / W总 × 100%
提高机械效率的方法：减小摩擦减轻机械自重
斜面：相同高度，斜面越长越省力，但机械效率越低。
η = W有/W总 × 100%  |  η < 100%
🔹 速度与运动图像
v-t 图像分析物体运动状态：
匀速直线 v恒定不变，F合=0
加速运动 v逐渐增大，合力方向与运动方向相同
减速运动 v逐渐减小，合力方向与运动方向相反
v = s/t  |  匀速：s = vt  |  v-t图面积 = 位移
热学
🔹 温度与温度计
温度表示物体的冷热程度。常用的温度计根据液体的热胀冷缩原理制成。
摄氏温度 标准大气压下冰水混合物为0℃，沸水为100℃
温度计使用 玻璃泡全部浸入、不碰容器底和壁、稳定后读数、不取出读数
t / ℃  |  液体温度计：热胀冷缩
🔹 物态变化
六种物态变化：
熔化 固态→液态（吸热）凝固 液态→固态（放热）
汽化 液态→气态（吸热）液化 气态→液态（放热）
升华 固态→气态（吸热）凝华 气态→固态（放热）
吸热：熔化、汽化、升华  |  放热：凝固、液化、凝华
🔹 晶体与非晶体熔化
晶体 有固定的熔化温度（熔点），熔化时不断吸热但温度不变。
非晶体 没有固定的熔点，熔化时温度持续上升。
常见晶体：冰、海波、各种金属；非晶体：蜡、玻璃、沥青、松香。
晶体熔化条件：①达到熔点 ②继续吸热  |  熔化时温度不变
🔹 汽化与液化
汽化的两种方式：蒸发 在任何温度下进行，发生在液体表面
沸腾 在一定温度下进行，液体表面和内部同时发生
影响蒸发快慢的因素：温度表面积空气流速
液化的两种方法：降低温度压缩体积
蒸发吸热制冷  |  液化放热
🔹 水沸腾实验
水沸腾时的特点：持续吸热，温度不变（保持沸点）。
沸点与气压有关 气压越高，沸点越高；气压越低，沸点越低。
标准大气压下水的沸点为100℃。沸腾条件：①达到沸点 ②继续吸热。
沸腾特点：吸热不升温  |  沸点随气压增大而升高
🔹 分子热运动
一切物质的分子都在不停地做无规则运动。
扩散现象 不同物质彼此进入对方的现象（温度越高越快）
分子间作用力 引力和斥力同时存在，随距离增大而减小。
扩散：温度↑ 扩散↑  |  r = r₀ 时引斥力平衡
🔹 内能与比热容
物体内部所有分子热运动的动能与分子势能的总和叫内能。
改变内能的方式 做功和热传递（等效）
比热容 c 单位质量的物质温度升高1℃所吸收的热量。
水的比热容最大：c水 = 4.2×10³ J/(kg·℃)
Q = cmΔt  |  水的比热容大 → 常用作冷却剂
🔹 热机
将内能转化为机械能的机器。常见热机：汽油机柴油机
四冲程：吸气→压缩→做功→排气，一个工作循环曲轴转2圈，活塞往复2次。
压缩冲程 机械能→内能 做功冲程 内能→机械能
四冲程：吸→压→做→排  |  做功冲程提供动力
光学
🔹 光的直线传播
光在同种均匀介质中沿直线传播。
光速 真空中 c ≈ 3×10⁸ m/s，水中约为真空中3/4，玻璃中约为2/3。
现象 影子、日食月食、小孔成像（成倒立实像）。
光速：c = 3×10⁸ m/s  |  光年 = 光在一年中走的距离
🔹 光的反射定律
反射光线、入射光线和法线在同一平面内；
反射光线和入射光线分居法线两侧；反射角等于入射角。
镜面反射 平行光入射，平行反射（表面光滑）
漫反射 平行光入射，向各方向反射（表面粗糙）
注意 反射角等于入射角（因果顺序不能颠倒）。
反射定律：三线共面、两线分居、两角相等
🔹 平面镜成像
平面镜成像特点：等大、等距、正立、虚像。
像与物大小相等像距等于物距像与物连线垂直镜面
成像原理：光的反射。用玻璃板代替平面镜（便于确定像的位置）。
平面镜成像：等大、等距、正立、虚像
🔹 光的折射
光从一种介质斜射入另一种介质时，传播方向发生偏折。
空气→水/玻璃 折射角 < 入射角（靠近法线）
水/玻璃→空气 折射角 > 入射角（远离法线）
生活现象 筷子在水中"折断"、池水看起来变浅、海市蜃楼。
光密介质中速度慢 → 光疏介质中速度快  |  垂直入射方向不变
🔹 凸透镜成像规律
凸透镜成像规律（u = 物距，f = 焦距）：
u > 2f 倒立缩小实像（f < v < 2f）— 照相机
u = 2f 倒立等大实像（v = 2f）
f < u < 2f 倒立放大实像（v > 2f）— 投影仪
u < f 正立放大虚像 — 放大镜
成像公式：1/u + 1/v = 1/f  |  一倍焦距分虚实，二倍焦距分大小
🔹 凸透镜应用与眼睛
照相机 u > 2f，成倒立缩小的实像
投影仪 f < u < 2f，成倒立放大的实像
放大镜 u < f，成正立放大的虚像
眼睛 晶状体 = 凸透镜，视网膜 = 光屏。近视眼用凹透镜矫正，远视眼用凸透镜矫正。
近视眼：像在视网膜前 → 凹透镜  |  远视眼：像在视网膜后 → 凸透镜
🔹 光的色散
白光通过三棱镜被分解成各种色光的现象叫光的色散。
七色光 红、橙、黄、绿、蓝、靛、紫
光的三原色 红、绿、蓝（等比例混合为白色）
物体的颜色 透明物体由透过它的色光决定；不透明物体由反射的色光决定。
光的三原色：红 + 绿 + 蓝 = 白  |  颜料三原色：红、黄、蓝
电学
🔹 电荷与电流
自然界只有两种电荷：正电荷和负电荷。
同种电荷相斥，异种电荷相吸
电流 电荷的定向移动形成电流。方向：正电荷移动方向。
金属导体中 自由电子定向移动形成电流，电子方向与电流方向相反。
电流方向：正极 → 负极（外部电路） |  电子方向：负极 → 正极
🔹 电路与电路图
电路由电源、开关、用电器、导线四部分组成。
通路 处处连通的电路 断路 某处断开的电路
短路 电流不经过用电器直接回到负极（可能烧坏电源）
电路图 用规定的符号表示电路连接的图。
电源：提供电能  |  用电器：消耗电能  |  开关：控制电路
🔹 串联与并联
串联 各元件首尾相连：电流只有一条路径，各用电器相互影响。
并联 各元件并列连接：电流有分支，各支路互不影响。
串联电路中：电流处处相等 I = I₁ = I₂ 并联电路中：干路电流等于各支路之和 I = I₁ + I₂
串联：I₁=I₂，U=U₁+U₂  |  并联：I=I₁+I₂，U₁=U₂
🔹 电流 · 电压 · 电阻
电流 I 单位 A（安培），电流表串联使用，正进负出。
电压 U 单位 V（伏特），电压表并联使用，可直接接电源两极。
电阻 R 单位 Ω（欧姆），导体对电流的阻碍作用。
影响电阻的因素：材料长度（越长电阻越大）横截面积（越大电阻越小）温度
R = ρL/S  |  导体电阻是导体本身的性质（与U、I无关）
🔹 欧姆定律
导体中的电流与导体两端的电压成正比，与导体的电阻成反比。
I = U/R （I—A，U—V，R—Ω）
变形公式：U = IR R = U/I
伏安法测电阻 用电压表和电流表测量电阻。
I = U/R  |  串联分压：U₁/U₂ = R₁/R₂  |  并联分流：I₁/I₂ = R₂/R₁
🔹 电阻的串并联
串联电阻 R总 = R₁ + R₂ + ...（越串越大，大于任何单个）
并联电阻 1/R总 = 1/R₁ + 1/R₂ + ...（越并越小，小于任何单个）
两电阻并联时：R = (R₁R₂)/(R₁+R₂)
串联：R = R₁ + R₂  |  并联：R = (R₁R₂)/(R₁+R₂)（两个时）
🔹 电功率
电流做的功叫电功。电流在单位时间内做的功叫电功率。
P = UI 电功率 = 电压 × 电流
W = UIt 电功 = 电压 × 电流 × 时间
额定功率 用电器在额定电压下正常工作时的功率。
P = UI = I²R = U²/R  |  1度 = 1kW·h = 3.6×10⁶ J
🔹 焦耳定律
电流通过导体产生的热量与电流的平方成正比，与导体的电阻成正比，与通电时间成正比。
Q = I²Rt （适用于任何电路）
纯电阻电路 Q = W = UIt = U²t/R（电能全部转化为内能）
电热器：利用电流热效应工作的设备（电炉、电饭锅、电热毯）。
Q = I²Rt  |  电热器：电能 → 内能
🔹 家庭电路与安全用电
家庭电路进户线：火线和零线，电压220V。
保险丝 由电阻率大、熔点低的铅锑合金制成（串联在火线上）
三孔插座 左零右火上接地 测电笔 氖管发光为火线
安全用电 不接触低压带电体，不靠近高压带电体。
家庭电路：220V  |  保险丝：电流过大时熔断切断电路
🔹 电磁现象
磁极 同名磁极相斥，异名磁极相吸。磁感线从N极出发到S极。
奥斯特实验 通电导体周围存在磁场（电流的磁效应）
电磁铁 插入铁芯后磁性增强。磁性强弱与电流大小和线圈匝数有关。
电动机 通电线圈在磁场中受力转动（电能→机械能）
发电机 电磁感应原理（机械能→电能）
电动机：电→磁→力（通电导体在磁场中受力） |  发电机：磁→电（电磁感应）
