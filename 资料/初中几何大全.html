🔷 全等与相似
🔹 手拉手模型
两个等腰三角形共顶点，旋转全等。
AB = AC AD = AE △ABD ≌ △ACE (SSS)
AB = AC, AD = AE ⇒ △ABD ≅ △ACE
🔹 一线三等角（K字型）
同一直线上三个角相等，构造相似三角形。
∠ABC = ∠ECD = ∠FDC
三等角 → 三角形相似
🔹 三垂直模型
利用三个直角关系，构造全等或相似三角形。
矩形 + 垂线 K字全等
三垂直 → △相似/全等
🔹 倍长中线模型
延长中线至一倍长度，构造平行四边形。
AM = MD 四边形ABDC为平行四边形
倍长中线 → △全等 → 平行四边
🔷 角平分线与相似
🔹 角平分线模型
角平分线上的点到角两边距离相等。
AD是角平分线 DE = DF
AD平分∠BAC ⇒ DE = DF
🔹 A字型相似
DE ∥ BC ⇒ △ADE ∼ △ABC（A字型）。
DE ∥ BC AD/AB = AE/AC = DE/BC
DE∥BC ⇒ △ADE ∽ △ABC
🔹 8字型相似
AB ∥ CD ⇒ △AOB ∼ △COD（8字型）。
AB ∥ CD 对应边成比例
AB∥CD ⇒ △AOB ∽ △COD
🔹 射影定理（母子型相似）
Rt△ABC中，AD ⊥ BC，得三个相似三角形。
∠BAC = 90° AD² = BD·CD 母子型相似
AD² = BD·CD  |  AB² = BD·BC  |  AC² = CD·BC
🔷 经典模型
🔹 半角模型
含45°角（≦90°之半）的几何问题。
∠EAF = 45° BE + DF = EF
45°半角  |  BE + DF = EF
🔹 将军饮马（最短路径）
找点P在直线l上，使AP + BP最小。
作对称点A' AP + BP = A'P + BP 最小值
作A关于l的对称点A'，连接A'B交l于P
🔹 胡不归模型
不同速度区域中的最优路径问题。
沙地速度v' 驿道速度v sinα = v'/v
sinα = v'/v  |  T点使路径最短
🔹 阿波罗尼斯圆（阿氏圆）
到两定点距离之比为定值的点P的轨迹。
PA : PB = k k > 0, k ≠ 1
PA / PB = k  |  P的轨迹为圆
🔷 拓展与中考压轴
🔹 费马点
三角形内到三顶点距离之和最小的点P。
∠APB = ∠BPC = ∠CPA = 120° PA + PB + PC 最小
PA+PB+PC最小  |  各角120°
🔹 隐圆模型
定点定角 → 点P轨迹为隐圆。
∠APB恒定 同一弧所对圆周角相等
∠APB = α ⇒ P在固定圆上
🔹 瓜豆原理（主从联动）
主动点P沿圆运动，从动点Q沿相似轨迹运动。
主动点P 从动点Q 种瓜得瓜
Q = O + k·R(θ)·(P−O)  |  Q轨迹与P相似
🔹 折叠模型（轴对称变换）
折叠前后全等，折痕是垂直平分线。
△AEF ≌ △A'EF 折痕⊥AA'且平分AA'
折叠 → 全等  |  折痕为垂直平分线
