专题一：三角形基础与勾股定理

三角形分类 · 内角和 · 勾股定理 · 特殊三角形

三角形的分类：
按边分：等边三角形（三边相等）、等腰三角形（两边相等）、不等边三角形
按角分：锐角三角形（三个角都<90°）、直角三角形（一角=90°）、钝角三角形（一角>90°）
三角形内角和定理：三角形三个内角之和 = 180°
外角定理：三角形的一个外角等于与它不相邻的两个内角之和

勾股定理（毕达哥拉斯定理）：
直角三角形两直角边a、b的平方和等于斜边c的平方：a² + b² = c²
常用勾股数：3-4-5、5-12-13、6-8-10、7-24-25、8-15-17、9-12-15
勾股逆定理：若三角形三边满足 a² + b² = c²，则该三角形是直角三角形（c为斜边）

特殊三角形的性质：
• 等腰三角形：两底角相等（等边对等角）；顶角平分线、底边中线、底边高线"三线合一"
• 等边三角形：三个内角都=60°，三边相等，面积 = √3/4 × 边长²
• 30°直角三角形：30°角所对的直角边 = 斜边的一半
• 等腰直角三角形：两直角边相等，底角=45°，斜边=直角边×√2

例题1：在△ABC中，∠A = 2∠B，∠C = ∠B + 30°，求△ABC各角的度数并判断三角形类型。
解析：
设∠B = x，则∠A = 2x，∠C = x + 30°
由三角形内角和定理：∠A + ∠B + ∠C = 180°
2x + x + (x+30°) = 180°
4x + 30° = 180° → 4x = 150° → x = 37.5°
∠A = 75°，∠B = 37.5°，∠C = 67.5°
三个角都锐角 → △ABC是锐角三角形。
注意：三角形内角和180°是几何计算的万能钥匙，遇到角度问题先想着设未知数列方程。
例题2：如图，Rt△ABC中，∠C=90°，AB=10，BC=6，求AC的长度。若以AC为轴旋转一周，求形成的几何体的体积。
解析：
(1) 由勾股定理：AC² = AB² - BC² = 10² - 6² = 100 - 36 = 64
AC = 8（取正）
(2) 以AC为轴旋转，BC为半径旋转形成圆锥：
圆锥底面半径r = BC = 6，高h = AC = 8
体积V = ⅓πr²h = ⅓ × π × 36 × 8 = 96π
勾股定理应用：已知直角三角形两边求第三边——首选勾股定理。注意分清哪边是斜边。
练习1：等腰三角形的一个外角为110°，求该等腰三角形的底角。
答案：有两种情况。①若外角是顶角的外角：顶角=180°-110°=70°，底角=(180°-70°)/2=55°。②若外角是底角的外角：底角=180°-110°=70°，顶角=180°-140°=40°。所以底角为55°或70°。
练习2：直角三角形的两边长分别为3和4，求第三边的长。
答案：有两种情况。①若3和4是直角边，斜边=√(3²+4²)=5。②若4是斜边，则第三直角边=√(4²-3²)=√7。所以第三边为5或√7。注意：没有明确斜边时要分类讨论！

专题二：全等三角形判定与模型

SSS · SAS · ASA · AAS · HL · 手拉手 · 三垂直

全等三角形的判定方法：
① SSS（边边边）：三边对应相等的两个三角形全等
② SAS（边角边）：两边和它们的夹角对应相等的两个三角形全等
③ ASA（角边角）：两角和它们的夹边对应相等的两个三角形全等
④ AAS（角角边）：两角和其中一角的对边对应相等的两个三角形全等
⑤ HL（斜边直角边）：斜边和一条直角边对应相等的两个直角三角形全等（仅限直角三角形）
注意：SSA（两边及其中一边的对角）不能判定全等！

手拉手模型（旋转全等）：
特征：两个等腰三角形共顶点，顶角相等，旋转后形成全等。
条件：AB=AC，AD=AE，∠BAC=∠DAE → △ABD ≌ △ACE（SAS）
结论：BD=CE，∠BDE=∠CED，对应边夹角等于顶角

三垂直模型（K字型全等）：
特征：三个直角在同一线上，利用同角的余角相等证明全等。
条件：∠ABC=∠BCD=∠CDE=90°（或类似配置），BC=CD → △ABC ≌ △ECD（ASA/AAS）
应用：坐标系中的几何问题、矩形折叠问题

例题3：已知AB=AC，AD=AE，∠BAC=∠DAE=90°（如图），求证：BD=CE且BD⊥CE。
解析：
这是手拉手模型的经典题。
思路：利用SAS证全等，再导角证垂直。
∵ AB=AC，AD=AE，∠BAC=∠DAE=90°
∴ ∠BAC+∠CAD = ∠DAE+∠CAD → ∠BAD=∠CAE
∴ △ABD ≌ △ACE（SAS）
∴ BD=CE（全等三角形对应边相等）
设BD与CE交于点F，BD与AC交于点G：
由全等得∠ABD=∠ACE，又∠AGB=∠CGF（对顶角）
∠BFG=∠ABD+∠BAG-∠ACE=∠BAC=90°
∴ BD⊥CE
核心规律：手拉手模型中，两条拉手线（BD和CE）的夹角 = 顶角。
例题4：在△ABC中，∠ACB=90°，AC=BC，过C在△ABC外作直线MN，AD⊥MN于D，BE⊥MN于E。求证：DE=AD+BE。
解析：
思路：通过三垂直模型构造全等三角形。
∵ AD⊥MN，BE⊥MN，∠ACB=90°
在Rt△ADC和Rt△CEB中：
∠DAC + ∠ACD = 90°（直角三角形两锐角互余）
∠ACD + ∠BCE = 180°-90°=90°（平角180°）
∴ ∠DAC = ∠BCE（同角的余角相等）
又AC=BC（已知），∠ADC=∠CEB=90°
∴ △ADC ≌ △CEB（AAS）
∴ AD=CE，CD=BE
∴ DE=CD+CE=BE+AD
三垂直模型要点：抓住"同角的余角相等"导角，再结合等边条件证全等。
练习3：如图，AB⊥BD于B，ED⊥BD于D，AC=CE且AC⊥CE。求证：△ABC ≌ △CDE。
答案：∵AB⊥BD，ED⊥BD，AC⊥CE，∠ACB+∠ECD=90°，∠ACB+∠CAB=90°→∠CAB=∠ECD。又AC=CE，∠ABC=∠CDE=90°。∴△ABC≌△CDE（AAS）。这是三垂直模型的变式。
练习4：下列条件能否判定两个三角形全等？①AB=A'B'，BC=B'C'，∠A=∠A'；②AB=A'B'，∠A=∠A'，∠C=∠C'。分别说明理由。
答案：①不能判定（SSA，但两边和其中一边的对角不一定全等）。②能判定（AAS，两角和其中一角的对边对应相等）。注意SSA不能判定全等是典型的易错点。

专题三：相似三角形判定与模型

相似判定 · A字型 · 8字型 · 比例性质

相似三角形的判定：
① AA：两角对应相等 → 两三角形相似（最常用的判定）
② SAS：两边对应成比例且夹角相等
③ SSS：三边对应成比例
④ HL：斜边和一条直角边对应成比例 → 两Rt△相似

相似三角形的性质：
• 对应角相等，对应边成比例（相似比 = k）
• 周长比 = 相似比 k
• 面积比 = 相似比的平方 k²
• 对应高、中线、角平分线的比都等于相似比

A字型相似：在△ABC中，DE∥BC → △ADE ∽ △ABC
→ AD/AB = AE/AC = DE/BC
注意：A字型有两种：正A字型（点D在AB上）和斜A字型（点D在AB延长线上，但有公共角）

8字型相似：AB∥CD → △AOB ∽ △COD
→ AO/CO = BO/DO = AB/CD
注意：8字型是"X型"——对顶角相等 + 内错角相等，得到AA相似

例题5：如图，在△ABC中，DE∥BC，AD=2cm，DB=3cm，AE=3cm，
(1) 求EC的长；(2) 若DE=4cm，求BC的长；(3) 求△ADE与△ABC的面积比。
解析：
(1) ∵ DE∥BC，由A字型相似：△ADE ∽ △ABC
AD/AB = AE/AC → 2/(2+3) = 3/(3+EC)
2/5 = 3/(3+EC) → 2(3+EC) = 15 → 6+2EC=15 → EC=4.5cm
(2) DE/BC = AD/AB = 2/5 → 4/BC=2/5 → BC=10cm
(3) 面积比 = 相似比的平方 = (2/5)² = 4/25
A字型关键：平行得相似，对应边成比例。注意AD:AB=2:5而不是AD:DB=2:3。
例题6：如图，平行四边形ABCD中，E在BA延长线上，CE交AD于F。若AB=6，AE=3，AF=2，求AD的长。
解析：
平行四边形ABCD中，AB∥CD，AD∥BC。
观察图形：AE∥CD → △AEF ∽ △DCF（8字型相似）
AE/CD = AF/DF
CD=AB=6（平行四边形对边相等），AE=3，AF=2
3/6 = 2/DF → DF=4
AD=AF+FD=2+4=6
8字型识别：当出现平行线和对顶角时，往往可以构造8字型相似。本题综合了平行四边形的性质和8字型相似。
练习5：如图，DE∥BC，AD:DB=2:3，且△ADE的面积为8cm²，求△ABC的面积。
答案：△ADE∽△ABC，相似比=AD:AB=2:5。面积比=4:25。8/S=4/25，S=50cm²。注意：面积比是相似比的平方，不是相似比本身——这是最容易犯的错误。
练习6：在△ABC中，AB=9，AC=6，点D在AB上且AD=4，点E在AC上，当AE为何值时△ADE与△ABC相似？
答案：有两种情况。①若DE∥BC，则AD/AB=AE/AC，4/9=AE/6，AE=8/3。②若∠ADE=∠C（公共角∠A，另一角相等），则AD/AC=AE/AB，4/6=AE/9，AE=6。所以AE=8/3或6。注意：题中没明确对应关系，要分类讨论。

专题四：几何辅助线专题

角平分线 · 倍长中线 · 截长补短 · 作垂线

角平分线模型：
性质：角平分线上的点到角两边距离相等。
常见辅助线：
① 过角平分线上一点向两边作垂线 → 构造全等Rt△
② 截取两边相等 → 利用SAS构造全等
③ 作平行线构造等腰三角形（角平分线+平行线→等腰）

倍长中线模型：
方法：延长三角形的中线至一倍长度，构造平行四边形或全等三角形。
条件：AD是△ABC的中线（D为BC中点），延长AD至E使AD=DE
结论：△ABD ≌ △ECD（SAS），四边形ABEC为平行四边形
作用：将分散的线段集中到同一个三角形中（常用于证明线段不等关系）

截长补短法：
用于证明"a+b=c"型的线段和差问题。
截长法：在长线段上截取一段等于某一短线段
补短法：延长短线段补上缺失部分
本质：通过构造全等三角形，转移线段位置

例题7：AD是△ABC的角平分线，∠B=2∠C。求证：AB+BD=AC。
解析：
思路（截长法）：在AC上截取AE=AB，连接DE。
∵ AD平分∠BAC → ∠BAD=∠EAD
AB=AE（所作），AD=AD（公共边）
∴ △ABD ≌ △AED（SAS）
∴ BD=DE，∠B=∠AED
∵ ∠B=2∠C → ∠AED=2∠C
又∠AED=∠C+∠EDC（外角定理）
∴ 2∠C=∠C+∠EDC → ∠EDC=∠C
∴ △EDC为等腰三角形 → DE=EC
∴ BD=DE=EC
∴ AB+BD=AE+EC=AC
截长法关键：在长线段上截取一部分等于另一短线段，构造全等三角形，将线段和差问题转化为等腰三角形问题。
例题8：在△ABC中，AD是BC边上的中线，AB=8，AC=6，求AD的取值范围。
解析：
思路（倍长中线）：延长AD至E，使DE=AD，连接CE。
∵ D为BC中点 → BD=CD
DE=AD（所作），∠ADB=∠EDC（对顶角相等）
∴ △ABD ≌ △ECD（SAS）→ CE=AB=8
在△ACE中，AC=6，CE=8
由三角形三边关系：CE-AC < AE < CE+AC
8-6 < AE < 8+6 → 2 < AE < 14
∵ AE=2AD → 2 < 2AD < 14 → 1 < AD < 7
倍长中线的作用：将不在同一个三角形的线段（AB、AC、AD）集中到△ACE中，利用三边关系求范围。
练习7：AD是△ABC的中线，求证：AB+AC > 2AD。
答案：倍长AD至E使AD=DE，连接CE。△ABD≌△ECD→CE=AB。在△ACE中：AC+CE>AE，即AC+AB>2AD。倍长中线的核心作用就是把分散的条件集中到一个三角形中。
练习8：如图，∠AOB=90°，OM是∠AOB的角平分线，P是OM上一点，PC⊥OA于C，PD⊥OB于D。求证：PC=PD。
答案：∵OM平分∠AOB→∠AOM=∠BOM=45°。PC⊥OA、PD⊥OB→∠PCO=∠PDO=90°。OP=OP（公共边）。∴△POC≌△POD（AAS）→PC=PD。这是角平分线性质的直接应用。

专题五：射影定理（母子型相似）

直角三角形·斜边高线·三个相似三角形

射影定理：
在Rt△ABC中，∠BAC=90°，AD⊥BC于D，则：
① AD² = BD·CD（高线定理——AD是BD和CD的比例中项）
② AB² = BD·BC（AB是BD和BC的比例中项）
③ AC² = CD·BC（AC是CD和BC的比例中项）

母子型相似（射影定理的相似本质）：
Rt△ABC中，斜边上的高线将原三角形分成两个小直角三角形，三个三角形两两相似：
△ABD ∽ △CAD ∽ △CBA
相似关系1：△ABD ∽ △CBA → AB/BC = BD/AB → AB² = BD·BC
相似关系2：△CAD ∽ △CBA → AC/BC = CD/AC → AC² = CD·BC
相似关系3：△ABD ∽ △CAD → AD/CD = BD/AD → AD² = BD·CD

例题9：Rt△ABC中，∠A=90°，AD⊥BC于D，BD=4cm，CD=9cm。
(1) 求AD的长；(2) 求AB和AC的长；(3) 求△ABC的面积。
解析：
(1) 由射影定理：AD² = BD·CD = 4×9 = 36 → AD = 6cm
(2) AB² = BD·BC = 4×(4+9) = 4×13 = 52 → AB = 2√13 cm
AC² = CD·BC = 9×13 = 117 → AC = 3√13 cm
(3) S△ABC = ½BC·AD = ½×13×6 = 39 cm²
验证：AB²+AC² = 52+117 = 169 = 13² = BC² ✓（勾股定理）
技巧：遇到直角三角形斜边上的高线问题，优先考虑射影定理——比用勾股定理列方程更快。
例题10：如图，矩形ABCD中，AB=3，AD=4，将矩形沿对角线BD折叠，点A的对应点为A'，A'D交BC于E。
(1) 求证：△BED是等腰三角形；
(2) 求DE的长。
解析：
(1) 折叠得∠A'BD=∠ABD（翻折全等）
矩形中AD∥BC → ∠ADB=∠CBD（内错角相等）
又∠ABD=∠A'BD → ∠CBD=∠A'BD → BE=DE
∴ △BED是等腰三角形。
(2) 设DE=BE=x，则CE=BC-BE=4-x
在Rt△CDE中，CD=AB=3，CE=4-x，DE=x
由勾股定理：x²=(4-x)²+3²
x²=16-8x+x²+9 → 8x=25 → x=25/8
折叠模型规律：折叠前后图形全等（对应边相等、对应角相等），折痕是垂直平分线。本题综合了折叠性质、等腰三角形判定和勾股定理计算。
练习9：Rt△ABC中，∠C=90°，CD⊥AB于D，AC=6，BC=8，求CD的长。
答案：由勾股定理AB=10。由射影定理AC²=AD·AB，即36=AD×10，AD=3.6。或直接用面积法：S=½×AC×BC=½×AB×CD→6×8=10×CD→CD=4.8。两种方法皆可，面积法更直接。
练习10：Rt△ABC中，CD是斜边AB上的高，求证：1/CD² = 1/AC² + 1/BC²。
答案：由射影定理CD²=AD·BD，AC²=AD·AB，BC²=BD·AB。所以1/AC²+1/BC²=1/(AD·AB)+1/(BD·AB)=(BD+AD)/(AD·BD·AB)=AB/(CD²·AB)=1/CD²。这个结论反映了直角三角形中三边的倒数平方关系。

专题六：圆的性质与定理

圆心角 · 圆周角 · 垂径定理 · 切线

圆的基本性质：
• 圆心角：顶点在圆心的角（=所对弧的度数）
• 圆周角：顶点在圆上，两边与圆相交的角
• 圆周角定理：同弧所对的圆周角 = 圆心角的一半
• 推论：直径所对的圆周角 = 90°（直径→直角，非常重要！）
• 同弧所对的圆周角相等；等弧所对的圆周角相等

垂径定理：
垂直于弦的直径平分弦，并且平分弦所对的两条弧。
推论：平分弦（非直径）的直径垂直于弦，并且平分弦所对的两条弧。

切线的性质与判定：
• 切线的性质：圆的切线垂直于过切点的半径
• 切线的判定：过半径外端且垂直于半径的直线是圆的切线
• 切线长定理：过圆外一点引圆的两条切线，切线长相等

例题11：如图，AB是⊙O的直径，C是圆上一点（不与A、B重合），D是弧AC的中点，连接OD交BC于E。
(1) 求证：OD∥AC；
(2) 若BC=8，OE=3，求⊙O的半径。
解析：
(1) ∵ D是弧AC的中点 → OD平分弧AC
由垂径定理推论：OD⊥AC且平分弦AC
但更有用的是：∠AOD = ½∠AOC（圆心角与弧对应）
又∠ABC = ½∠AOC（圆周角定理）
∴ ∠AOD = ∠ABC → OD∥BC（同位角相等）
证法二：∵D是弧AC中点 → OD平分∠AOC → ∠AOD=∠DOC
∵∠AOC=2∠ABC（圆周角=½圆心角）→ ∠AOD=∠ABC → OD∥BC
(2) ∵ AB是直径 → ∠ACB=90°（直径对直角）
由(1)得OD∥BC，O是AB中点 → OE是△ABC的中位线
AC=2OE=6（中位线=底边一半）
在Rt△ABC中，AB²=AC²+BC²=36+64=100 → AB=10
半径r=AB/2=5
题眼：直径→直角，中点+平行→中位线，这两个是圆中求线段长度的核心方法。
例题12：如图，PA、PB是⊙O的两条切线，切点为A、B，∠APB=60°，OP=6。
(1) 求PA的长；(2) 求⊙O的半径。
解析：
(1) PA、PB是切线 → PA=PB（切线长定理）
连接OA，则OA⊥PA（切线垂直于半径）
连接PO，可证△PAO≌△PBO（HL）→ PO平分∠APB → ∠APO=30°
在Rt△PAO中，∠APO=30°，OP=6
cos30°=PA/OP → √3/2 = PA/6 → PA=3√3
(2) sin30°=OA/OP → 1/2 = OA/6 → OA=3
∴ ⊙O的半径r=3
切线问题解题模板：①连接圆心与切点（得垂直）；②利用切线长定理（线段相等）；③勾股定理或三角函数求值。
练习11：⊙O中，弦AB=8，圆心O到AB的距离为3，求⊙O的半径。
答案：过O作OC⊥AB于C，由垂径定理AC=BC=4。Rt△AOC中，OC=3，AC=4，半径OA=√(3²+4²)=5。
练习12：AB是⊙O的直径，C是⊙O上一点，∠ABC=30°，AB=8，求AC的长。
答案：直径AB=8→半径=4。直径所对圆周角∠ACB=90°。Rt△ABC中，∠ABC=30°，AC=½AB=4（30°角所对直角边=斜边的一半）。

专题七：最短路径与几何最值

将军饮马 · 折叠 · 对称变换 · 最值问题

将军饮马问题（两定点一动点）：
问题：在直线l上找点P，使PA+PB最小。
方法：作A关于l的对称点A'，连接A'B交l于P点，P即所求。
原理：两点之间线段最短 + 对称性质（AP=A'P）

折叠（轴对称）模型：
折叠的本质是轴对称变换：
① 折叠前后图形全等（对应边相等、对应角相等）
② 折痕是对应点连线的垂直平分线
③ 折痕所在直线为对称轴
④ 折叠后往往产生等腰三角形（折痕两侧的对应线段相等）

其他最值思路：
• 垂线段最短：点到直线的最短距离是垂线段
• 三角形三边关系：|PA-PB| ≤ AB（当P在AB上取等号）
• 利用函数求最值：设未知数列二次函数，通过顶点求最值

例题13：在平面直角坐标系中，A(2, 5)、B(4, 1)，在x轴上找一点P使PA+PB最小，求P点坐标。
解析：
这是标准的将军饮马问题，x轴相当于直线l。
(1) 作A(2,5)关于x轴的对称点A'(2,-5)
(2) 连接A'B，求A'B与x轴的交点：
直线A'B的方程：斜率为k=(1+5)/(4-2)=3，过B(4,1)
y-1=3(x-4) → y=3x-11
(3) 令y=0：3x-11=0 → x=11/3
∴ P(11/3, 0)
验证：此时PA+PB = PA'+PB = A'B = √[(4-2)²+(1+5)²] = √(4+36)=√40=2√10，即为最小值。
例题14：矩形纸片ABCD中，AB=6，BC=8，将纸片折叠使B与D重合，折痕为EF（E在AB上，F在CD上），求折痕EF的长。
解析：
B与D重合 → 折痕EF是BD的垂直平分线。
矩形对角线BD = √(6²+8²)=10
设BD与EF交于O，则O为BD中点 → OB=OD=5
由△BOE ∽ △BAD（∠OBE共用，∠BOE=∠BAD=90°）
OE/AD = OB/AB → OE/8 = 5/6 → OE=20/3
同理OF=20/3 → EF=40/3
折叠+相似综合：折叠产生垂直平分线，再利用相似三角形求线段长度。注意折叠后对应点的连线被折痕垂直平分。
练习13：在直线l上找一点P使得PA+PB最小（A、B在l同侧），简述作法及理由。
答案：作法：作A关于l的对称点A'，连接A'B交l于P，P即所求。理由：由对称性PA=PA'，PA+PB=PA'+PB，由两点之间线段最短得A'B为最短路径。
练习14：如图，Rt△ABC中，∠C=90°，AC=4，BC=3，将△ABC折叠使A与B重合，折痕为DE（D在AB上，E在AC上），求CD的长。
答案：AB=5（勾股定理），折叠使A与B重合→DE是AB的垂直平分线，D为AB中点，AD=BD=2.5。由△ADE∽△ACB（∠A公用，∠ADE=∠C=90°→AA相似），AE/AB=AD/AC→AE/5=2.5/4→AE=25/8。CE=AC-AE=4-25/8=7/8。Rt△CDE中：CD²=CE²+DE²，但问题需要的是CD。实际上更好用：设CD=x，则DE²=AD²-(AC-CD)²...这个较复杂。

专题八：几何综合应用

综合题 · 动态几何 · 几何变换 · 中考压轴

几何三大变换：
① 平移：图形沿某方向移动，形状、大小不变，对应线段平行且相等
② 翻折（轴对称）：图形沿某直线翻折，对应点连线被对称轴垂直平分
③ 旋转：图形绕某点旋转一定角度，对应点到旋转中心距离相等（手拉手模型的本质）

几何综合题的解题策略：
① 标图：把已知条件全部标注在图上（角的度数、边的长度、平行/垂直关系）
② 联想：看到什么图形特征就联想对应的方法（看到中点想中线倍长，看到角平分线想作垂线，看到平行想相似...）
③ 导角：几何题往往先从角度关系入手，利用内角和、平行线、全等/相似推导角度
④ 设元：设未知数列方程（勾股定理方程、比例方程等）

中考几何常见模型速查：
• 半角模型：正方形中含45°角，利用旋转构造全等，结论 BE+DF=EF
• 隐圆模型：定点定角 → 点在圆上运动（定弦定角、定角定高）
• 一线三等角：一条线上出现三个等角，必构造相似或全等三角形
• 手拉手模型：共顶点等线段，旋转全等或旋转相似

例题15（综合）：正方形ABCD中，E是BC中点，F是CD上一点且CF=¼CD。
(1) 求证：△ABE ∽ △ECF。
(2) 求∠AEF的度数。
(3) 若正方形边长为4，求△AEF的面积。
解析：
设正方形边长为4，则AB=BC=CD=DA=4。
(1) E是BC中点 → BE=EC=2，CF=¼CD=1
AB/EC = 4/2 = 2，BE/CF = 2/1 = 2
∴ AB/EC = BE/CF，又∠B=∠C=90°
∴ △ABE ∽ △ECF（SAS：两边对应成比例且夹角相等）
(2) 由(1)得∠BAE=∠CEF
∠BAE+∠AEB=90°（Rt△ABE中）
∴ ∠CEF+∠AEB=90° → ∠AEF=180°-(∠CEF+∠AEB)=90°
(3) S正方形=16，S△ABE=½×4×2=4，S△ECF=½×2×1=1
DF=CD-CF=4-1=3，S△ADF=½×4×3=6
S△AEF=16-4-1-6=5
综合题三步法：①设边长便于计算；②用相似/全等导角；③面积用整体减局部。
例题16（压轴）：如图，等边△ABC边长为6，点D从A出发沿AB以1cm/s向B运动，同时E从B出发沿BC以1cm/s向C运动，连接AE、CD交于F。
(1) 求证：AE=CD。
(2) 当t为何值时，△BDE是直角三角形？
(3) 求∠DFE的度数。
解析：
(1) 等边△ABC → AB=AC=BC=6，∠A=∠B=∠C=60°
动点速度相同 → AD=BE=t
△ABE和△CAD中：AB=CA=6，∠B=∠A=60°，BE=AD=t
∴ △ABE ≌ △CAD（SAS）→ AE=CD
(2) 运动t秒，BD=6-t，BE=t
①若∠BDE=90°：∠B=60°→∠BED=30°→BD=½BE→6-t=t/2→t=4
②若∠BED=90°：∠B=60°→∠BDE=30°→BE=½BD→t=(6-t)/2→t=2
③∠DBE=60°≠90°，不可能。
∴ t=2或t=4时△BDE是直角三角形。
(3) 由(1)全等得∠BAE=∠ACD
∠DFE=∠AFC=180°-(∠ACD+∠CAF)
=180°-(∠BAE+∠CAF)=180°-∠A=120°
动态几何模板：①用t表示动点位置；②全等证明线段关系；③分类讨论特殊位置；④角度关系用全等推导。
练习15（半角模型）：正方形ABCD中，∠EAF=45°（E在BC上，F在CD上），求证：BE+DF=EF。
答案：将△ADF绕A顺时针旋转90°到△ABF'，则DF=BF'，AF=AF'，∠F'AE=∠FAD+∠BAE=90°-45°=45°=∠EAF。又AE=AE，∴△F'AE≌△FAE（SAS）→EF=EF'=BE+BF'=BE+DF。
练习16（综合）：在△ABC中，AB=AC=5，BC=6，D是BC中点，连接AD。求AD的长，及△ABC的外接圆半径。
答案：AB=AC→等腰三角形，AD⊥BC（三线合一），BD=3。Rt△ABD中AD=√(5²-3²)=4。外接圆半径：S=½×BC×AD=½×6×4=12。由正弦定理：外接圆半径R=abc/(4S)=5×5×6/(4×12)=150/48=25/8=3.125。
练习17（思考）：矩形ABCD中，AB=6，BC=8，P是AD上的动点，求PC+PB的最小值。
答案：这是将军饮马的变式。作C关于AD的对称点C'（即B关于AD的对称点...实际上矩形中，C关于AD的对称点C'在BC延长线上，且DC'=DC=AB=6）。连接BC'交AD于P。BC'=√(8²+12²)=√208=4√13。即PC+PB的最小值为4√13。
