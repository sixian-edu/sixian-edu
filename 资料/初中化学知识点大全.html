初中化学考点知识点大全人教版 · 九年级全一册

涵盖初中化学全部核心考点 · 中考复习备考专用

第一单元 走进化学世界

一、物理变化与化学变化

  • 物理变化：没有新物质生成（如：破碎、融化、挥发、蒸发）
  • 化学变化：有新物质生成（如：燃烧、生锈、变质、发酵）
  • 化学变化一定伴随物理变化（发光放热、颜色变化、生成气体沉淀）

二、实验基本操作

  • 药品取用：三不（不触、不闻、不尝）、节约原则、剩余不放回原瓶
  • 酒精灯：用外焰加热，不可用嘴吹灭（用灯帽盖灭）
  • 量筒读数：视线与凹液面最低处相平 — 仰视读数偏小，俯视读数偏大
  • 托盘天平：左物右码（药品不能直接放托盘，用纸或玻璃器皿）
  • 加热试管：液体不超过1/3，试管倾斜45°，管口不对人
  • 检查气密性：用手握法或注水法

第二单元 我们周围的空气

一、空气的成分

  • 空气中含量：N₂ 78%、O₂ 21%、稀有气体0.94%、CO₂ 0.03%
  • 测定空气中氧气含量：用红磷燃烧（P + O₂ → P₂O₅白色固体），水面上升约1/5

二、空气污染与保护

  • 有害气体：SO₂、NO₂、CO、O₃；可吸入颗粒物（PM2.5/PM10）
  • 三大环境问题：酸雨（SO₂+NO₂）、温室效应（CO₂）、臭氧空洞

三、氧气

氧气性质

物理性质：无色无味气体，密度比空气略大，不易溶于水
化学性质：助燃性（本身不燃烧），氧化性

氧气实验制取

实验室制O₂（三种方法）：

  • 2KMnO₄ →(加热) K₂MnO₄ + MnO₂ + O₂↑（固+固加热型，管口要略向下倾斜，棉花防止粉末堵塞）
  • 2H₂O₂ →(MnO₂) 2H₂O + O₂↑（固+液常温型，最简便）
  • 2KClO₃ →(MnO₂,加热) 2KCl + 3O₂↑

收集方法：向上排空气法（密度＞空气）、排水法（不易溶于水）
验满：带火星木条放瓶口，复燃即满

  • 化合反应：多变一；分解反应：一变多
  • 氧化反应：物质与氧发生的反应（剧烈氧化如燃烧，缓慢氧化如生锈）

第三单元 物质构成的奥秘

一、分子和原子

  • 分子的性质：很小、不断运动（扩散）、有间隔（热胀冷缩）
  • 原子：化学变化中的最小粒子，原子由质子、中子、电子构成
  • 分子由原子构成（如 H₂O 由 2个H 和 1个O 原子构成）

二、原子的结构

  • 原子核 = 质子（带正电）+ 中子（不带电）
  • 核外电子（带负电）分层排布，最外层电子数≤4易失电子→金属，≥4易得电子→非金属
  • 原子中：质子数 = 核电荷数 = 核外电子数
  • 相对原子质量 ≈ 质子数 + 中子数

三、元素

  • 元素：质子数相同的一类原子的总称
  • 地壳中含量前四：O、Si、Al、Fe（氧硅铝铁）
  • 元素符号：一大二小（如Na、Ca、Cl）

四、化合价

  • 常用化合价口诀：钾钠银氢+1价，钙镁钡锌+2价；氟氯溴碘-1价，氧-2来硫-2；铁有+2和+3，铝+3硅+4
  • 化合物中正负化合价代数和为零
  • 化学式：表示物质组成的式子（H₂O、NaCl、CO₂）

第四单元 自然界的水

  • 水的净化：沉淀→过滤→吸附（活性炭）→消毒（氯气）→蒸馏（最高级）
  • 过滤：一贴二低三靠—滤纸紧贴漏斗壁；滤纸低于漏斗口、液面低于滤纸边缘；烧杯靠玻璃棒、玻璃棒靠三层滤纸、漏斗下端靠烧杯壁
  • 硬水与软水：肥皂水区分（硬水泡沫少浮渣多）；硬水软化—煮沸或蒸馏
  • 电解水：正极→O₂，负极→H₂（体积比1:2）
    2H₂O →(通电) 2H₂↑ + O₂↑ — 证明水由氢、氧元素组成

第五单元 化学方程式

一、质量守恒定律

质量守恒定律

内容：化学反应前后，参加反应的各物质质量总和 = 生成的各物质质量总和

微观本质：原子种类、数目、质量三不变（只是原子重新组合）

遵循原因：化学反应是原子重新组合的过程，原子本身不变

二、化学方程式的书写

  • 书写步骤：写→配→注→等（写化学式、配平、注条件↑↓、改等号）
  • 配平方法：最小公倍数法、观察法、奇偶配平法、待定系数法
必考化学方程式

碳和碳的氧化物：
C + O₂ →(点燃) CO₂   2C + O₂ →(点燃) 2CO   CO₂ + C →(高温) 2CO
CaCO₃ + 2HCl → CaCl₂ + H₂O + CO₂↑   CO₂ + Ca(OH)₂ → CaCO₃↓ + H₂O

金属：
Fe₂O₃ + 3CO →(高温) 2Fe + 3CO₂（炼铁原理）
Fe + CuSO₄ → FeSO₄ + Cu   Zn + H₂SO₄ → ZnSO₄ + H₂↑

燃烧：
CH₄ + 2O₂ →(点燃) CO₂ + 2H₂O   C₂H₅OH + 3O₂ →(点燃) 2CO₂ + 3H₂O

第六单元 碳和碳的氧化物

一、碳单质

  • 金刚石：最硬（切割），正八面体
  • 石墨：最软之一（铅笔芯），能导电（电极），有滑腻感（润滑剂）
  • C₆₀：足球烯，由分子构成
  • 碳单质物理性质差异的原因：碳原子排列方式不同

二、碳的化学性质

  • 稳定性：常温下稳定（古代字画保存千年）
  • 可燃性：O₂充足→CO₂（放热多），O₂不足→CO（有毒）
  • 还原性：C + 2CuO →(高温) 2Cu + CO₂↑（黑色变红色）

三、二氧化碳 CO₂

  • 物理性质：无色无味，密度比空气大（可倾倒灭火），能溶于水
  • 化学性质：不燃烧不支持燃烧（灭火）；与水反应（CO₂ + H₂O → H₂CO₃，碳酸使石蕊变红）；与石灰水反应（CO₂ + Ca(OH)₂ → CaCO₃↓ + H₂O）
  • 实验室制取：CaCO₃ + 2HCl → CaCl₂ + H₂O + CO₂↑（固+液常温，向上排空气法，不用稀硫酸）
  • 验满：燃着木条放瓶口，熄灭即满

四、一氧化碳 CO

  • 有毒（与血红蛋白结合，缺氧致死）
  • 可燃性：2CO + O₂ →(点燃) 2CO₂（蓝色火焰）
  • 还原性：CO + CuO →(加热) Cu + CO₂（炼铁原理）

第七单元 燃料及其利用

  • 燃烧条件：①可燃物 ②与O₂接触 ③温度达着火点（三缺一不可）
  • 灭火原理：消除任一条件（隔离、隔绝空气、降温）
  • 化石燃料：煤（主要含C，"工业粮食"）、石油（主要含C、H，"工业血液"）、天然气（主要CH₄）
  • 新能源：氢能、风能、太阳能、核能、地热能

第八单元 金属和金属材料

一、金属材料

  • 金属的物理共性：导电导热、延展性、金属光泽
  • 合金比纯金属硬度更大、熔点更低、抗腐蚀更好
  • 常见合金：生铁（含C 2-4.3%）、钢（含C 0.03-2%，性能更优）

二、金属活动性顺序

金属活动性顺序表

K Ca Na Mg Al Zn Fe Sn Pb (H) Cu Hg Ag Pt Au

从左到右活动性逐渐减弱

  • 应用：①金属+酸→盐+H₂（前H金属）；②金属+盐→新金属+新盐（前置后）
  • 验证金属活动性：三选二法、中间金属法

三、铁的冶炼与生锈

  • 炼铁原理：Fe₂O₃ + 3CO →(高温) 2Fe + 3CO₂（还原反应）
  • 铁生锈条件：Fe + O₂ + H₂O（三者缺一不可）
  • 防锈措施：涂油刷漆、镀金属、保持干燥、制成不锈钢

第九单元 溶液

  • 溶液：均一稳定的混合物（溶质+溶剂）
  • 乳化：洗涤剂除去油污（不是溶解，是乳化作用）
  • 饱和溶液与不饱和溶液的转化：
    饱和→不饱和：加溶剂、升温（大多）
    不饱和→饱和：加溶质、蒸发溶剂、降温（大多）
  • 溶解度：一定温度下，100g溶剂中最多溶解的溶质质量
  • 溶解度曲线：交点→该温度下溶解度相等；陡→受温度影响大（如KNO₃）；缓→受温度影响小（如NaCl）
  • 溶质质量分数 = 溶质质量 / 溶液质量 × 100%
  • 配制溶液步骤：计算→称量→溶解→装瓶

第十单元 酸和碱

一、常见的酸

酸化学式性质用途
盐酸HCl无色刺激味液体，挥发白雾除锈、胃液助消化
硫酸H₂SO₄无色油状，浓硫酸有吸水性（干燥剂）和脱水性化肥、除锈、电池
硝酸HNO₃强腐蚀性、强氧化性化肥、炸药
  • 稀释浓硫酸：酸入水，沿壁倒，不断搅（放大量热，切不可水入酸！）
  • 酸的化学通性：①使指示剂变色（紫色石蕊→红）②+活泼金属→盐+H₂ ③+金属氧化物→盐+H₂O ④+碱→盐+H₂O ⑤+某些盐→新酸+新盐

二、常见的碱

碱化学式性质用途
氢氧化钠NaOH白色固体，易潮解，强腐蚀性（"烧碱、火碱、苛性钠"）造纸、肥皂、去油污
氢氧化钙Ca(OH)₂白色粉末，微溶（"熟石灰、消石灰"）改良酸性土壤、建筑
  • 碱的化学通性：①使指示剂变色（紫色石蕊→蓝，无色酚酞→红）②+非金属氧化物→盐+H₂O ③+酸→盐+H₂O ④+某些盐→新碱+新盐
  • 中和反应：酸+碱→盐+H₂O（放热），实质是H⁺+OH⁻→H₂O
  • pH：pH＜7酸性，pH=7中性，pH＞7碱性；pH试纸测整数，不能润湿

第十一单元 盐 化肥

一、常见的盐

名称化学式俗称用途
氯化钠NaCl食盐调味、防腐、生理盐水
碳酸钠Na₂CO₃纯碱、苏打玻璃、造纸、洗涤剂
碳酸氢钠NaHCO₃小苏打发酵粉、治疗胃酸过多
碳酸钙CaCO₃石灰石建筑、补钙剂

二、复分解反应

  • 通式：AB + CD → AD + CB（双交换、价不变）
  • 条件：产物中有沉淀、气体或水生成

三、化学肥料

  • 氮肥—促进叶生长（缺氮叶黄），如尿素CO(NH₂)₂
  • 磷肥—促进根和果实（缺磷暗绿），如过磷酸钙
  • 钾肥—促进抗倒伏（缺钾叶尖黄），如KCl、K₂SO₄
  • 复合肥：含两种或以上营养元素，如KNO₃、NH₄H₂PO₄
  • 鉴别铵态氮肥（NH₄⁺）：加熟石灰研磨，有氨味（刺激性气味）

第十二单元 化学与生活

  • 六大营养素：糖类、油脂、蛋白质、维生素、水、无机盐
  • 供能物质：糖类（主要）、油脂（备用）、蛋白质
  • 有机合成材料：塑料、合成纤维、合成橡胶（三大合成材料）
  • 热固性塑料—不可重塑；热塑性塑料—可重塑
  • 白色污染（塑料难降解）→ 减少使用、回收利用、使用可降解塑料

常见化学方程式汇总

化合反应：
4P + 5O₂ →(点燃) 2P₂O₅（冒白烟）
3Fe + 2O₂ →(点燃) Fe₃O₄（剧烈燃烧火星四射）
2Mg + O₂ →(点燃) 2MgO（耀眼白光）
2H₂ + O₂ →(点燃) 2H₂O（淡蓝色火焰）
S + O₂ →(点燃) SO₂（蓝紫色火焰，刺鼻）

分解反应：
2H₂O₂ →(MnO₂) 2H₂O + O₂↑   2KMnO₄ →(加热) K₂MnO₄ + MnO₂ + O₂↑
2KClO₃ →(MnO₂,加热) 2KCl + 3O₂↑   2H₂O →(通电) 2H₂↑ + O₂↑
H₂CO₃ → H₂O + CO₂↑   CaCO₃ →(高温) CaO + CO₂↑

常考物质颜色

红色：Cu（紫红）、Fe₂O₃（红棕）、红磷
黑色：C、CuO、MnO₂、Fe₃O₄、FeO
蓝色：CuSO₄·5H₂O（胆矾）、Cu(OH)₂沉淀
绿色：Cu₂(OH)₂CO₃（铜绿）
黄色：S（硫磺）、Fe³⁺溶液（棕黄）
白色：P₂O₅、MgO、CaO、Ca(OH)₂、NaCl、Na₂CO₃

初中化学必背口诀

常见元素化合价：钾钠银氢+1价，钙镁钡锌+2价；氟氯溴碘-1价，通常氧-2硫-2；铁有+2和+3，铝+3硅+4，铜有+1和+2

金属活动性顺序：嫁给那美女，心铁惜千金，统共一百斤（K Ca Na Mg Al Zn Fe Sn Pb (H) Cu Hg Ag Pt Au）

过滤操作：一贴二低三靠

稀释浓硫酸：酸入水，沿壁倒，不断搅拌
