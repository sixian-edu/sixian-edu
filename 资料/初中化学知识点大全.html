第一单元 走进化学世界

一、物理变化与化学变化

  • 物理变化：没有新物质生成（如：破碎、融化、挥发、蒸发、干冰升华、铁水铸锅、灯泡通电发光）
  • 化学变化：有新物质生成（如：燃烧、生锈、变质、发酵、酿酒、光合作用、呼吸作用）
  • 化学变化一定伴随物理变化（发光放热、颜色变化、生成气体或沉淀），但物理变化不一定伴随化学变化
  • 判断核心依据：有无新物质生成（不是看现象，而是看本质）
比较项目物理变化化学变化
本质区别无新物质生成有新物质生成
宏观表现形状、状态改变发光、放热、变色、气体、沉淀
微观实质分子间隔/运动改变,分子本身不变分子分裂成原子,原子重新组合成新分子
联系化学变化中一定伴随物理变化
实例玻璃破碎、水结冰、汽油挥发、铁丝弯曲木炭燃烧、铁生锈、食物腐败、火药爆炸
【易错陷阱】变化判断
  • ❌ 有发光放热一定是化学变化 — 电灯通电发光放热为物理变化
  • ❌ 有气体产生一定是化学变化 — 水沸腾产生水蒸气为物理变化
  • ❌ 爆炸一定是化学变化 — 轮胎爆炸为物理变化（火药爆炸才是化学变化）
  • ⚠ 口诀：现象只是表象，本质看有无新物质生成

二、化学实验基本操作

  • 药品取用原则：三不（不触、不闻、不尝）、节约原则（按需取用）、剩余药品不放回原瓶（放入指定容器）
  • 酒精灯：用外焰加热（外焰温度最高），不可用嘴吹灭（用灯帽盖灭两次），酒精量不超过2/3、不少于1/4
  • 量筒：视线与凹液面最低处相平 — 仰视读数偏小（实际偏大），俯视读数偏大（实际偏小）
  • 托盘天平：左物右码，药品不能直接放托盘（用纸或玻璃器皿），精确到0.1g；左码右物时实际质量=砝码-游码
  • 加热试管：液体不超1/3、试管倾斜45°、管口不对人、先预热后集中加热；固体试管口略下倾
  • 检查气密性：用手握法（加热法）或注水法（液差法）— 有气泡冒出/形成水柱即气密性良好
  • 仪器洗净标准：内壁附着的水既不聚成水滴也不成股流下
【例题】实验操作

下列操作中正确的是（　）
A. 用嘴吹灭酒精灯　B. 用量筒溶解固体　C. 用外焰加热　D. 用剩药品放回原瓶

答案：C。
A酒精灯必须用灯帽盖灭（两次）；B量筒只能用于量取液体，不能用于溶解或反应；D剩余药品应放入指定容器，不可放回原瓶以免污染整瓶。A、B、D均为中考常考易错项。

第二单元 我们周围的空气

一、空气的成分

  • 空气成分及含量（体积分数）：N₂ 78%、O₂ 21%、稀有气体0.94%、CO₂ 0.03%、其他0.03%
  • 测定空气中O₂含量：4P + 5O₂ →(点燃) 2P₂O₅（红磷燃烧产生大量白烟，冷却后水面上升约1/5）
  • 实验推论：①O₂约占空气体积1/5；②N₂不支持燃烧（红磷剩余）；③N₂难溶于水（水面只上升1/5）
【易错注意】氧气含量测定实验

⚠ 红磷必须足量，确保耗尽装置内全部O₂
⚠ 装置必须冷却至室温后再打开止水夹（否则气体膨胀，进入水体积偏小）
⚠ 不能用木炭、硫、铁丝代替红磷！木炭/硫生成气体（CO₂/SO₂），压强差不足；铁丝在空气中无法燃烧
⚠ 实验结果偏小的常见原因：红磷不足、未冷却打开、装置漏气

二、空气污染与保护

  • 有害气体：SO₂、NO₂、CO、O₃；可吸入颗粒物（PM2.5/PM10）
  • 三大环境问题：酸雨（SO₂+NO₂，pH＜5.6）、温室效应（CO₂→全球变暖）、臭氧空洞（氟利昂→紫外线增强）
  • 空气质量日报六项指标：SO₂、NO₂、CO、O₃、PM10、PM2.5

三、氧气

氧气性质

物理性质：无色无味气体，密度比空气略大（1.429g/L），不易溶于水（不易≠不溶，1L水溶30mL），液氧淡蓝色
化学性质：助燃性（本身不可燃）、氧化性（强氧化剂），与金属/非金属剧烈反应

氧气实验制取（三种方法对比）
制取方法化学方程式装置类型关键注意
KMnO₄分解2KMnO₄→(△)K₂MnO₄+MnO₂+O₂↑固+固加热管口略下倾,棉花防粉末堵塞
H₂O₂分解2H₂O₂→(MnO₂)2H₂O+O₂↑固+液常温最简便,不需加热,控制滴速
KClO₃分解2KClO₃→(MnO₂,△)2KCl+3O₂↑固+固加热试管口不用棉花,纯KClO₃加热易炸

收集方法：向上排空气法（密度大于空气，较干燥）；排水法（不易溶于水，更纯净）
验满：带火星木条放瓶口，复燃即满（排空气法）
检验：带火星木条伸入瓶内，复燃即证明是O₂

催化剂（触媒）

催化剂：在化学反应中能改变反应速率，而本身的质量和化学性质在反应前后都不变的物质。
记忆口诀："一变二不变" — 变的是反应速率，不变的是质量和化学性质。
⚠ 催化剂不能使生成物增多，只能缩短到达平衡的时间。
⚠ 催化剂具有专一性（一种催化剂通常只催化一种反应），如MnO₂对H₂O₂分解有效，但不能催化所有反应。

四、基本反应类型

  • 化合反应：多变一（A+B→AB），如 4P+5O₂→2P₂O₅
  • 分解反应：一变多（AB→A+B），如 2H₂O→(通电)2H₂↑+O₂↑
  • 氧化反应：物质与氧发生的反应（注意：氧化反应不是基本反应类型，是从得失氧角度分类）
    剧烈氧化：燃烧、爆炸；缓慢氧化：铁生锈、呼吸、食物腐败、农家肥腐熟
  • 缓慢氧化可能引发自燃（如白磷暴露在空气中）

第三单元 物质构成的奥秘

一、分子和原子

  • 分子的性质：很小（肉眼不可见）、不断运动（扩散现象：花香、品红扩散）、有间隔（热胀冷缩、压缩气体、10mL水+10mL酒精＜20mL）
  • 原子：化学变化中的最小粒子（化学变化中不可再分）
  • 分子由原子构成（如 H₂O 由 2个H 和 1个O 构成；CO₂由1个C和2个O构成）
比较项目分子原子
定义保持物质化学性质的最小粒子化学变化中的最小粒子
区别化学变化中可再分（分裂成原子）化学变化中不可再分
联系分子由原子构成；原子可直接构成物质（金属、稀有气体、金刚石等）
构成物质举例水（H₂O）、CO₂、H₂、O₂、CH₄Fe、Cu、Ne、He、C（金刚石）
【例题】分子原子辨析

下列有关分子和原子的说法中正确的是（　）
A. 分子是保持物质性质的最小粒子
B. 原子是不能再分的粒子
C. 分子和原子都可以直接构成物质
D. 分子一定比原子大

答案：C。
A应为"化学性质"（物理性质如颜色、状态等单个分子无法保持）；B原子在化学变化中不可再分，但在物理层面可再分（核反应）；D分子和原子大小没有绝对比较关系（如H₂分子比Fe原子小得多）。

二、原子的结构

  • 原子核 = 质子（+1，带正电）+ 中子（不带电）— 居于原子中心，体积很小但质量很大（核外电子质量可忽略）
  • 核外电子（-1，带负电）分层排布：K层（第1层）≤2个，L层（第2层）≤8个，最外层电子数≤8
  • 最外层电子数决定化学性质：≤4易失电子→金属元素；≥4易得电子→非金属元素；8（He为2）→稳定→稀有气体
  • 原子中：质子数 = 核电荷数 = 核外电子数（原子不显电性）
  • 相对原子质量 ≈ 质子数 + 中子数（精确值：Ar = 某原子质量 / 碳-12原子质量×1/12）
  • 同一原子中：质子数不一定等于中子数（如H有1个质子0个中子，C有6个质子6个中子）

三、离子

  • 离子：带电的原子（或原子团）— 原子得失电子后形成
  • 阳离子（带正电）：原子失电子 → 质子数＞电子数，如 Na⁺、Mg²⁺、Al³⁺、K⁺、Ca²⁺
  • 阴离子（带负电）：原子得电子 → 质子数＜电子数，如 Cl⁻、O²⁻、S²⁻
  • 记忆口诀："阳失阴得" — 金属失电子→阳离子，非金属得电子→阴离子
  • 常见原子团离子：OH⁻、NO₃⁻、SO₄²⁻、CO₃²⁻、NH₄⁺

四、元素

  • 元素：质子数（核电荷数）相同的一类原子总称 — 决定元素种类的是质子数
  • 地壳中含量前四：O、Si、Al、Fe（氧硅铝铁，谐音：养闺女贴）
  • 生物细胞中含量前四：O、C、H、N
  • 空气中含量最多的元素：N（78%N₂），其次为O（21%O₂）
  • 元素符号：一大二小（如Na、Ca、Cl），表示一种元素、一个原子
  • 元素周期表：横行=周期（7个），纵列=族（18列/16族），同周期电子层数相同，同族最外层电子数相同（化学性质相似）

五、化合价

  • 常用化合价口诀：钾钠银氢+1价，钙镁钡锌+2价；氟氯溴碘-1价，氧-2来硫-2；铁有+2和+3，铝+3硅+4，铜汞+2最常见
  • 常见原子团化合价：OH⁻（-1）、NO₃⁻（-1）、SO₄²⁻（-2）、CO₃²⁻（-2）、NH₄⁺（+1）
  • 规则：化合物中各元素正负化合价代数和为零（单质中元素化合价为0）
  • 化学式：表示物质组成的式子（H₂O、NaCl、CO₂、CaCO₃、NaOH）
  • 根据化合价写化学式：正左负右、标价交叉、约简检验

第四单元 自然界的水

一、水的净化

  • 净化流程：沉淀→过滤→吸附→消毒→蒸馏
  • ①沉淀：静置沉降或絮凝沉降（加明矾KAl(SO₄)₂·12H₂O，吸附悬浮杂质使其沉降）
  • ②过滤：一贴二低三靠—滤纸紧贴漏斗壁（一贴）；滤纸低于漏斗口、液面低于滤纸边缘（二低）；烧杯靠玻璃棒、玻璃棒靠三层滤纸、漏斗下端靠烧杯壁（三靠）
  • ③吸附：活性炭（疏松多孔，物理吸附—吸附色素和异味，不能软化硬水）
  • ④消毒：氯气（Cl₂）或ClO₂、O₃（化学变化，杀死微生物）
  • ⑤蒸馏：利用沸点不同分离，所得水为蒸馏水（纯净物），净化程度最高

二、硬水与软水

  • 区分方法：加肥皂水—软水泡沫多浮渣少，硬水泡沫少浮渣多
  • 硬水软化：煮沸（生活中）、蒸馏（实验室）、离子交换法（工业）
  • 硬水的危害：浪费肥皂（形成沉淀）、锅炉结垢（导热差→爆炸风险）、影响口感

三、电解水

电解水实验

2H₂O →(通电) 2H₂↑ + O₂↑

  • 正极（接电源+）→O₂（带火星木条复燃），负极（接电源-）→H₂（点燃淡蓝色火焰）
  • 体积比 V(H₂):V(O₂) = 2:1；质量比 m(H₂):m(O₂) = 1:8
  • 实验结论：水由氢元素和氧元素组成（不能说"水由氢气和氧气组成"）
  • 为增强水的导电性，可加入少量NaOH或H₂SO₄（不能加NaCl，会产生Cl₂）

第五单元 化学方程式

一、质量守恒定律

质量守恒定律

内容：参加化学反应的各物质质量总和 = 生成的各物质质量总和

微观本质：化学反应前后原子种类、数目、质量三不变（原子只是重新组合）

宏观表现：元素种类不变、元素质量不变、物质总质量不变；但物质种类一定变、分子种类一定变

⚠ "参加"二字不能漏——未反应完的剩余部分不计入

⚠ 有气体参加或生成的反应，在密闭容器中称量才守恒；敞口容器要考虑气体逸出或进入

二、化学方程式的书写与配平

  • 书写步骤：写→配→注→等（写化学式、配平、标注条件↑↓、改等号）
  • 配平方法：最小公倍数法（最常用）、观察法（适于简单反应）、奇偶配平法（氧元素）、待定系数法
  • 条件标注：加热△、点燃、高温、催化剂；生成物标↑（气体）、↓（沉淀）

三、四种基本反应类型

反应类型形式特征实例
化合反应A+B→AB多变一C+O₂→(点燃)CO₂、4P+5O₂→2P₂O₅
分解反应AB→A+B一变多2H₂O→(通电)2H₂↑+O₂↑、2KMnO₄→(△)K₂MnO₄+MnO₂+O₂↑
置换反应A+BC→AC+B单换单Fe+CuSO₄→FeSO₄+Cu、Zn+H₂SO₄→ZnSO₄+H₂↑
复分解反应AB+CD→AD+CB双交换、价不变NaCl+AgNO₃→AgCl↓+NaNO₃
【质量守恒计算】

反应 2A + B₂ → 2C 中，已知A的相对原子质量为24，B的相对原子质量为16，求C的相对分子质量。

根据质量守恒，反应前后总质量相等。
左端总质量 = 2×24 + 2×16 = 48 + 32 = 80
2个C分子的总质量为80，所以一个C分子的相对分子质量 = 80 ÷ 2 = 40。

四、化学方程式的简单计算

  • 计算步骤：设→写→标→列→解→答（设未知数、写方程式、标已知和未知、列比例式、求解、作答）
  • 注意事项：代入的必须是纯净物质量（若为溶液，代入溶质质量；若为不纯物，乘纯度）
  • 计算格式：质量比对应，单位要统一

第六单元 碳和碳的氧化物

一、碳单质的多样性

  • 金刚石：自然界最硬的物质（切割玻璃、钻探），正八面体，网状结构
  • 石墨：最软矿物之一（铅笔芯），能导电（电极），有滑腻感（润滑剂），层状结构—层间易滑动
  • C₆₀：足球烯/巴基球，由60个C原子构成球形分子（分子结构稳定）
  • 碳纳米管：管状结构，强度高、导电性好
  • 碳单质物理性质差异大的原因：碳原子的排列方式不同（结构决定性质）
  • ⚠ 金刚石和石墨都是碳单质，化学性质相同（同种元素组成），但物理性质不同

二、碳的化学性质

  • 稳定性：常温下化学性质稳定（古代字画保存千年、档案用碳素墨水）
  • 可燃性：O₂充足→CO₂（放热多），O₂不足→CO（有毒，蓝色火焰）
  • 还原性：C + 2CuO →(高温) 2Cu + CO₂↑（黑色粉末→红色固体）
    碳能还原CuO、Fe₂O₃、CO₂等

三、CO₂与CO对比

性质二氧化碳 CO₂一氧化碳 CO
物理性质无色无味,密度＞空气,能溶于水无色无味,密度≈空气,难溶于水
可燃性不燃烧,不支持燃烧（灭火）可燃烧,2CO+O₂→(点燃)2CO₂,蓝色火焰
还原性无还原性有还原性,CO+CuO→(△)Cu+CO₂
与水反应CO₂+H₂O→H₂CO₃（碳酸使石蕊变红）不反应
与石灰水CO₂+Ca(OH)₂→CaCO₃↓+H₂O（变浑）不反应
毒性无毒（但浓度高会窒息）剧毒（与血红蛋白结合,缺氧致死）
主要用途灭火、干冰人工降雨、碳酸饮料燃料、炼铁还原剂
【实验室制CO₂】

药品：石灰石（或大理石）+ 稀盐酸
CaCO₃ + 2HCl → CaCl₂ + H₂O + CO₂↑

  • ❌ 不用稀硫酸：生成CaSO₄微溶于水，覆盖在石灰石表面阻止反应持续
  • ❌ 不用浓盐酸：浓盐酸易挥发→CO₂中混有HCl气体不纯
  • ❌ 不用Na₂CO₃：反应剧烈，不易控制（一般不用于实验室制取）
  • 收集：向上排空气法（密度大于空气，能溶于水故不用排水法）
  • 验满：燃着木条放在集气瓶口，熄灭即满
【常见易错】CO与CO₂
  • CO₂通入紫色石蕊试液变红，不是因为CO₂显酸性，而是CO₂+H₂O→H₂CO₃（碳酸使石蕊变红）
  • 除去CO中的CO₂：通过NaOH溶液（CO₂+2NaOH→Na₂CO₃+H₂O，CO不反应）
  • 除去CO₂中的CO：通过灼热的CuO（CO+CuO→Cu+CO₂）
  • CO中毒急救：立即转移到通风处，严重者送医高压氧舱治疗

第七单元 燃料及其利用

一、燃烧与灭火

  • 燃烧条件：①可燃物 ②与O₂（空气）充分接触 ③温度达到着火点（三者缺一不可）
  • 着火点是物质的固有属性，不可降低（不是"降低着火点"，是"降温至着火点以下"）
  • 灭火原理（三个方向）：①清除可燃物 ②隔绝空气（O₂） ③降温至着火点以下
  • 灭火器类型：干粉灭火器（一般火灾）、CO₂灭火器（图书档案/贵重设备）、水基灭火器（木材纸张）

二、化石燃料

  • 煤：主要含C（"工业粮食"），干馏→焦炭、煤焦油、煤气（化学变化）
  • 石油：主要含C、H（"工业血液"），分馏→汽油、柴油、煤油、沥青（物理变化，利用沸点不同分离）
  • 天然气：主要成分CH₄（甲烷），CH₄ + 2O₂ →(点燃) CO₂ + 2H₂O（蓝色火焰，产生H₂O和CO₂）
  • 化石燃料是不可再生能源，燃烧产生SO₂→酸雨、CO₂→温室效应

三、新能源

  • 氢能：2H₂ + O₂ →(点燃) 2H₂O，热值高、产物无污染（最清洁燃料），但制取成本高、储存运输困难
  • 乙醇（C₂H₅OH）：C₂H₅OH + 3O₂ →(点燃) 2CO₂ + 3H₂O，可再生（粮食发酵制取）
  • 其他新能源：风能、太阳能、核能、地热能、潮汐能、生物质能

第八单元 金属和金属材料

一、金属材料

  • 金属的物理共性：导电性、导热性、延展性、金属光泽（⚠ 汞在常温下为液态，是唯一液态金属）
  • 合金：一种金属与其他金属（或非金属）熔合而成的具有金属特性的混合物
  • 合金性能优于纯金属：硬度更大、熔点更低、抗腐蚀更强
  • 常见合金：生铁（含C 2-4.3%，硬而脆）、钢（含C 0.03-2%，韧性好）、黄铜（Cu-Zn）、青铜（Cu-Sn）、不锈钢（Fe-Cr-Ni）

二、金属活动性顺序

金属活动性顺序表

K Ca Na Mg Al Zn Fe Sn Pb (H) Cu Hg Ag Pt Au

从左到右活动性逐渐减弱

  • 应用一：前H金属 + 酸 → 盐 + H₂↑（Mg、Al、Zn、Fe与酸反应产生H₂）
  • 应用二：活泼金属 + 盐溶液 → 不活泼金属 + 新盐（Fe+CuSO₄→FeSO₄+Cu）
  • 验证方法：三选二法（取三种金属两两组合）、中间金属法
  • ⚠ 特别注意：K、Ca、Na与酸反应过于剧烈，而且会先与水反应（2Na+2H₂O→2NaOH+H₂↑）

三、铁的冶炼与铁生锈

  • 高炉炼铁原理（还原反应）：Fe₂O₃ + 3CO →(高温) 2Fe + 3CO₂（CO作还原剂）
  • 铁生锈条件：Fe + O₂ + H₂O（三者缺一不可），铁锈主要成分Fe₂O₃·xH₂O（红棕色疏松多孔）
  • 防锈措施：涂漆/刷油（隔绝O₂和H₂O）、镀金属（镀锌/铬）、保持干燥、制成不锈钢（合金化）
  • 铝的自我保护：Al与O₂反应生成致密Al₂O₃氧化膜，阻止进一步氧化（铝比铁活泼但更耐腐蚀）
【例题】金属活动性判断

将X、Y、Z三种金属分别放入稀盐酸中，X、Y表面有气泡，Z无变化；将X放入YNO₃溶液中，X表面析出Y。则X、Y、Z的活动性顺序为？

分析思路：
① X、Y能与盐酸反应产生气体 → X、Y在H之前（活泼金属）
② Z不与盐酸反应 → Z在H之后（不活泼）
③ X能置换出Y（X + YNO₃ → XNO₃ + Y）→ X比Y活泼
∴ 活动性顺序：X ＞ Y ＞ H ＞ Z

第九单元 溶液

一、溶液的基本概念

  • 溶液：均一、稳定的混合物（溶质+溶剂）— 均一=各处浓度相同，稳定=不分层不沉淀
  • 乳化：洗涤剂去油污不是溶解，是乳化作用（将大油滴分散成小液滴，随水冲走）
  • 溶解时的温度变化：NaCl溶解（温度几乎不变）、NaOH/浓H₂SO₄溶解（放热）、NH₄NO₃溶解（吸热）

二、饱和溶液与不饱和溶液

  • 转化：
    不饱和→饱和：加溶质、蒸发溶剂、降温（大多数固体，气体相反）
    饱和→不饱和：加溶剂、升温（大多数固体）
  • 判断方法：继续加入该溶质，看是否溶解；或降低温度看是否有晶体析出
  • ⚠ 饱和溶液不一定是浓溶液，不饱和溶液不一定是稀溶液（如Ca(OH)₂微溶，饱和溶液浓度很小）

三、溶解度

  • 溶解度：一定温度下，100g溶剂中最多溶解的溶质质量（单位g）
  • 影响因素：内因=溶质和溶剂的本性；外因=温度（对气体还受压强影响）
  • 气体溶解度：温度↑溶解度↓，压强↑溶解度↑（打开汽水瓶盖后压强减小→CO₂逸出）
  • 溶解度曲线应用：
    — 交点→该温度下两物质溶解度相等
    — 陡升型（KNO₃）→受温度影响大，提纯用降温结晶
    — 缓升型（NaCl）→受温度影响小，提纯用蒸发结晶
    — 下降型（Ca(OH)₂）→极少数，随温度升高溶解度降低

四、溶质质量分数

核心公式

溶质质量分数 = 溶质质量 ÷ 溶液质量 × 100%

溶液质量 = 溶质质量 + 溶剂质量 = 溶液体积 × 溶液密度

稀释（配制）公式：m₁ × a% = m₂ × b%（稀释前后溶质质量不变）

饱和溶液：溶质质量分数 = S/(100+S) × 100%（S为溶解度）

  • 配制溶液步骤（以NaCl溶液为例）：计算→称量→量取→溶解→装瓶贴标签
  • 误差分析（均导致质量分数偏小）：
    溶质偏少—称量时左码右物、砝码生锈/沾污、药品潮解
    溶剂偏多—量取时仰视读数（实际量多的水）
【例题】溶液稀释计算

用100g 98%的浓硫酸配制19.6%的稀硫酸，需要加水多少克？

稀释前后H₂SO₄溶质质量不变：
m(H₂SO₄) = 100g × 98% = 98g
稀释后溶液总质量 = 98g ÷ 19.6% = 500g
需要加水 = 500g - 100g = 400g
答：需要加水400克。

一、常见的酸

【必考】浓硫酸稀释

酸入水，沿壁倒，不断搅
浓硫酸遇水放出大量热，若将水倒入浓硫酸，水迅速沸腾导致酸液飞溅，十分危险！
若不慎将浓硫酸溅到皮肤上：先用大量水冲洗，再涂3%-5%的NaHCO₃溶液。

  • 酸的化学通性（五条）：
    ① 使紫色石蕊变红（无色酚酞不变色）；② +活泼金属→盐+H₂↑；③ +金属氧化物→盐+H₂O（除锈）；④ +碱→盐+H₂O（中和反应）；⑤ +某些盐→新酸+新盐

二、常见的碱

碱化学式俗称/性质用途
氢氧化钠NaOH烧碱、火碱、苛性钠；白色固体易潮解,强腐蚀性造纸、肥皂、去油污

  • 碱的化学通性（四条）：
    ① 使紫色石蕊变蓝、无色酚酞变红；② +非金属氧化物→盐+H₂O（NaOH吸收CO₂→Na₂CO₃变质）；③ +酸→盐+H₂O；④ +某些盐→新碱+新盐

四、中和反应与pH

  • 中和反应：酸 + 碱 → 盐 + H₂O（放热）— 实质：H⁺ + OH⁻ → H₂O
  • 中和反应应用：
    — 胃酸过多：吃含Al(OH)₃的药物（Al(OH)₃+3HCl→AlCl₃+3H₂O）
    — 改良酸性土壤：加熟石灰Ca(OH)₂
    — 处理工厂废水：酸碱中和达标排放
  • pH：pH＜7酸性（越小酸性越强），pH=7中性，pH＞7碱性（越大碱性越强）
  • pH试纸使用：玻璃棒蘸取待测液涂在试纸上，与标准比色卡比较（不能润湿、只能读整数值、不能投入试液）

第十一单元 盐 化肥

一、常见的盐

名称化学式俗称用途
氯化钠NaCl食盐调味、防腐、配制生理盐水(0.9%)
碳酸钠Na₂CO₃纯碱、苏打（是盐不是碱！）玻璃、造纸、洗涤剂去油污
碳酸氢钠NaHCO₃小苏打发酵粉(产CO₂)、治疗胃酸过多

高锰酸钾KMnO₄紫黑色固体消毒杀菌、实验室制O₂
【易错提醒】纯碱不是碱

Na₂CO₃俗称"纯碱"，水溶液显碱性，但分类属于盐（由金属阳离子Na⁺和酸根阴离子CO₃²⁻构成）。
酸、碱、盐分类：酸→电离出H⁺；碱→电离出OH⁻；盐→金属阳离子+酸根阴离子。

二、复分解反应

  • 通式：AB + CD → AD + CB（双交换、价不变）
  • 反应条件（满足其一即可）：产物中有沉淀↓、气体↑或水H₂O生成
  • 常见沉淀的颜色与化学式：
    白色：AgCl↓、BaSO₄↓（不溶于酸）、CaCO₃↓、Mg(OH)₂↓、BaCO₃↓、Al(OH)₃↓
    蓝色：Cu(OH)₂↓
    红褐色：Fe(OH)₃↓
  • Cl⁻检验：加AgNO₃溶液→白色沉淀（AgCl↓），再加稀HNO₃不溶解
  • SO₄²⁻检验：加Ba(NO₃)₂溶液→白色沉淀（BaSO₄↓），再加稀HNO₃不溶解
  • CO₃²⁻检验：加稀盐酸→产生气体通入石灰水→变浑浊

第十二单元 化学与生活

一、六大营养素

  • 三大供能物质（顺序：主要→备用→构建）：
    糖类（主要供能，米饭/馒头/面条）→ C₆H₁₂O₆（葡萄糖）+ O₂ → CO₂ + H₂O + 能量
    油脂（备用能源，植物油/动物脂肪）→ 单位质量供能最多
    蛋白质（构建和修复身体，肉/蛋/奶/豆）→ 构成细胞的基本物质
  • 非供能但必不可少：
    维生素（调节生理功能，水果/蔬菜；缺VA→夜盲，缺VC→坏血病）
    水（占体重约60%-70%，参与一切代谢）
    无机盐（Ca→骨骼发育，Fe→血红蛋白，缺Fe→贫血，I→甲状腺，缺I→大脖子病）

二、有机合成材料

  • 三大合成材料：塑料、合成纤维（涤纶/腈纶/尼龙）、合成橡胶
  • 热塑性塑料（聚乙烯PE）— 加热熔化，可重塑；热固性塑料（酚醛树脂）— 加热不熔化，不可重塑
  • 天然纤维与合成纤维鉴别：灼烧法—棉花（烧纸味）、羊毛（烧焦羽毛味）、合成纤维（特殊气味，熔化球状）
  • 白色污染：废弃塑料难降解—解决措施：减少使用（少用塑料袋）、重复使用、回收利用、研发可降解塑料

中考常考题型

题型一：物质鉴别（只用一种试剂）

如何鉴别蒸馏水、食盐水、稀盐酸三种无色液体？

方案：取三种液体各少量于试管中，分别滴加紫色石蕊试液。
蒸馏水→不变色（紫色）；食盐水（NaCl中性）→不变色；稀盐酸（酸性）→变红色。
剩余两种用AgNO₃溶液鉴别：食盐水产生白色沉淀AgCl↓，蒸馏水无变化。
题型二：CO₃²⁻检验

如何证明某白色粉末中含有碳酸根离子？

方法：取少量样品于试管中，加入足量稀盐酸，将产生的气体通入澄清石灰水中。
现象：有气泡产生，石灰水变浑浊 → 证明含CO₃²⁻。
化学方程式：CO₃²⁻ + 2H⁺ → H₂O + CO₂↑；CO₂ + Ca(OH)₂ → CaCO₃↓ + H₂O
题型三：除杂题（括号内为杂质）

① CO(CO₂)   ② CO₂(CO)   ③ NaCl(Na₂CO₃)

① CO(CO₂)：气体通过NaOH溶液（CO₂+2NaOH→Na₂CO₃+H₂O），再通过浓硫酸干燥。
② CO₂(CO)：气体通过灼热CuO（CO+CuO→Cu+CO₂）。
③ NaCl(Na₂CO₃)：加适量稀盐酸（Na₂CO₃+2HCl→2NaCl+H₂O+CO₂↑），蒸发结晶。
口诀：不增不减、易分离（不引入新杂质、不减少主要成分、方法简便）。
题型四：图像分析（pH变化）

向一定量的稀盐酸中逐滴加入NaOH溶液，溶液的pH变化曲线如何？

图像特征：pH从＜7（盐酸酸性）开始 → 随着NaOH加入pH逐渐增大 → 恰好完全中和时pH=7（转折点） → 继续加NaOH则pH＞7（碱性）。
👉 曲线在pH=7处有一个"突变"区域，这是中和反应完全的特征，指示剂恰好变色（酚酞由无色→浅红）。
题型五：化学方程式计算

实验室用6.5g锌与足量稀硫酸反应（Zn + H₂SO₄ → ZnSO₄ + H₂↑），求：
(1) 生成氢气的质量是多少克？
(2) 标准状况下（H₂密度0.09g/L），生成氢气的体积是多少升？

解：设生成H₂质量为x。
Zn + H₂SO₄ → ZnSO₄ + H₂↑
65         2
6.5g        x
比例式：65/6.5 = 2/x → x = 0.2g
(1) 生成H₂质量 = 0.2g
(2) V = m/ρ = 0.2g ÷ 0.09g/L = 2.22L
答：生成氢气0.2g，体积约2.22L。

常见易错陷阱汇总

陷阱一：物质变化判断
  • ❌ 有发光放热一定是化学变化 → 电灯发光是物理变化
  • ❌ 有气体产生一定是化学变化 → 水沸腾产生水蒸气是物理变化
  • ❌ 爆炸一定是化学变化 → 轮胎爆炸是物理变化（火药爆炸才是化学变化）
  • ❌ 颜色改变一定是化学变化 → 氧气变为液氧（无色→淡蓝）是物理变化
陷阱二：溶液与溶解度
  • ❌ 均一稳定的液体都是溶液 → 水是纯净物，不是溶液
  • ❌ 溶液都是无色透明的 → CuSO₄溶液蓝色、FeCl₃溶液棕黄、KMnO₄溶液紫红
  • ❌ 溶液体积 = 溶质体积 + 溶剂体积 → 分子间有间隔，体积不具加和性（10mL水+10mL酒精＜20mL）
  • ❌ 饱和溶液一定是浓溶液，不饱和一定是稀溶液 → Ca(OH)₂微溶，饱和溶液浓度很小
  • ❌ 搅拌可以增大溶解度 → 搅拌只能加快溶解速率，不能改变溶解度
陷阱三：酸和碱
  • ❌ Na₂CO₃叫纯碱 → 是盐不是碱（水溶液显碱性是因为CO₃²⁻水解）
  • ❌ CO₂使石蕊变红 → CO₂不显酸性，是CO₂+H₂O→H₂CO₃使石蕊变红
  • ❌ pH=0表示没有酸性 → pH=0是强酸性（c(H⁺)=1mol/L很大）
  • ❌ 浓硫酸做干燥剂利用脱水性 → 利用吸水性（脱水性是化学变化，吸水性是物理变化）
  • ❌ 酸碱中和生成盐和水，则生成盐和水的反应一定是中和反应 → 酸+金属氧化物→盐+水，不是中和反应
陷阱四：金属
  • ❌ 铁与酸反应生成Fe³⁺ → 铁与稀盐酸/稀硫酸反应生成Fe²⁺（浅绿色），不是Fe³⁺（棕黄色）
  • ❌ Cu与稀盐酸反应 → Cu在H之后，不与稀盐酸/稀硫酸反应
  • ❌ Fe₂O₃→Fe是置换反应 → Fe₂O₃与CO反应不是置换（是氧化还原，生成两种化合物）
  • ❌ 合金一定由金属组成 → 生铁/钢含C（非金属），所以合金至少含一种金属但可含非金属
  • ❌ 铝耐腐蚀是因为铝不活泼 → 铝很活泼，但表面形成致密Al₂O₃氧化膜
陷阱五：实验操作
  • ❌ 量筒俯视读数偏大 → 实际量取液体偏少（浓度偏大）
  • ❌ 托盘天平左码右物 → 实际质量 = 砝码质量 - 游码质量（偏小）
  • ❌ KMnO₄制O₂结束时先撤酒精灯 → 水倒吸炸裂试管（应先撤导管）
  • ❌ 过滤时液面高于滤纸边缘 → 滤液浑浊（液体从滤纸和漏斗缝隙流下）
  • ❌ pH试纸润湿后再测 → 稀释了待测液，结果不准确（酸性偏高，碱性偏低）

附录：必背化学方程式全集

一、化合反应

4P + 5O₂ →(点燃) 2P₂O₅（大量白烟）
3Fe + 2O₂ →(点燃) Fe₃O₄（火星四射，生成黑色固体）
2Mg + O₂ →(点燃) 2MgO（耀眼白光，生成白色固体）
2H₂ + O₂ →(点燃) 2H₂O（淡蓝色火焰，唯一液态生成物）
S + O₂ →(点燃) SO₂（蓝紫色火焰，刺激性气味）
C + O₂ →(点燃) CO₂（充分燃烧）
2C + O₂ →(点燃) 2CO（不充分燃烧）
2CO + O₂ →(点燃) 2CO₂（蓝色火焰，CO剧毒）
CO₂ + C →(高温) 2CO
CaO + H₂O → Ca(OH)₂（生石灰→熟石灰，放热）

二、分解反应

2H₂O₂ →(MnO₂) 2H₂O + O₂↑
2KMnO₄ →(△) K₂MnO₄ + MnO₂ + O₂↑（紫黑色→绿色+黑色）
2KClO₃ →(MnO₂,△) 2KCl + 3O₂↑
2H₂O →(通电) 2H₂↑ + O₂↑（正O₂负H₂，体积比1:2）
H₂CO₃ → H₂O + CO₂↑（碳酸不稳定，易分解）
CaCO₃ →(高温) CaO + CO₂↑（石灰石煅烧分解）
2HgO →(△) 2Hg + O₂↑（红色粉末→银白液滴）
Cu₂(OH)₂CO₃ →(△) 2CuO + H₂O + CO₂↑（铜绿加热分解）

三、置换反应

Fe + CuSO₄ → FeSO₄ + Cu（铁表面析出红色固体）
Cu + 2AgNO₃ → Cu(NO₃)₂ + 2Ag（铜丝表面银白固体）
Zn + H₂SO₄ → ZnSO₄ + H₂↑（实验室制H₂，最常用）
Zn + 2HCl → ZnCl₂ + H₂↑
Fe + 2HCl → FeCl₂ + H₂↑（铁溶解，溶液浅绿色）
Mg + H₂SO₄ → MgSO₄ + H₂↑（剧烈反应，大量气泡）
Al + 3HCl → AlCl₃ + 3/2H₂↑（铝与酸反应）
C + 2CuO →(高温) 2Cu + CO₂↑（黑色→红色）
H₂ + CuO →(△) Cu + H₂O（黑色变红色，管壁水珠）
Fe + CuCl₂ → FeCl₂ + Cu

四、复分解反应

CaCO₃ + 2HCl → CaCl₂ + H₂O + CO₂↑（实验室制CO₂）
Na₂CO₃ + 2HCl → 2NaCl + H₂O + CO₂↑（CO₃²⁻检验）
NaHCO₃ + HCl → NaCl + H₂O + CO₂↑（小苏打治胃酸）
Na₂CO₃ + Ca(OH)₂ → CaCO₃↓ + 2NaOH（工业制NaOH）
CO₂ + Ca(OH)₂ → CaCO₃↓ + H₂O（CO₂检验，石灰水变浑）
AgNO₃ + HCl → AgCl↓ + HNO₃（Cl⁻检验，白色沉淀不溶于酸）
BaCl₂ + H₂SO₄ → BaSO₄↓ + 2HCl（SO₄²⁻检验，白色沉淀）
Ba(NO₃)₂ + H₂SO₄ → BaSO₄↓ + 2HNO₃
NaOH + HCl → NaCl + H₂O（中和反应，放热）
Ca(OH)₂ + H₂SO₄ → CaSO₄ + 2H₂O（中和反应）
Fe₂O₃ + 6HCl → 2FeCl₃ + 3H₂O（除铁锈，溶液变棕黄）
CuO + H₂SO₄ → CuSO₄ + H₂O（黑色→蓝色溶液）
Cu(OH)₂ + 2HCl → CuCl₂ + 2H₂O（蓝色沉淀溶解）

五、其他重要反应

CH₄ + 2O₂ →(点燃) CO₂ + 2H₂O（天然气燃烧）
C₂H₅OH + 3O₂ →(点燃) 2CO₂ + 3H₂O（酒精燃烧）
Fe₂O₃ + 3CO →(高温) 2Fe + 3CO₂（高炉炼铁）
CuO + H₂ →(△) Cu + H₂O
CuO + CO →(△) Cu + CO₂
2NaOH + CO₂ → Na₂CO₃ + H₂O（NaOH吸收CO₂变质）
Ca(OH)₂ + CO₂ → CaCO₃↓ + H₂O（石灰水检验CO₂）
CaO + H₂O → Ca(OH)₂（生石灰→熟石灰，放热）
Na₂CO₃ + H₂O + CO₂ → 2NaHCO₃

实验操作口诀合集

过滤：一贴二低三靠
稀释浓硫酸：酸入水，沿壁倒，不断搅拌
气密性检查：一握二看三冒泡
加热固体：试管略下倾，管口不对人，先预热后集中
排水集气：满水无泡倒立水中，导管伸入看气泡，瓶口出水已集满
配制溶液：计算称量再溶解，装瓶贴签要记清
