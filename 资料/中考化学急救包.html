03计算题模板——套进去就得分
⚠️ 计算题必知：化学计算满分6分，套对模板至少拿4分！代入质量必须是纯净物质量！
模板1 化学方程式计算（标准六步法）
❶ 设未知数
→
❷ 写方程式
→
❸ 标已知、未知
→
❹ 列比例式
→
❺ 求解
→
❻ 答
📝 示例：6g镁在O₂中燃烧，求生成MgO的质量
设生成MgO质量为x  |  写 2Mg + O₂ →点燃 2MgO
标 48 : 80 = 6g : x  |  列 48/6 = 80/x
解 x = 80×6÷48 = 10g  |  答 生成MgO质量为10g
模板2 含杂质的计算（先换算成纯净物）

公式：m(纯净物) = m(混合物) × 纯度（质量分数）

例含CaCO₃ 80%的石灰石100g高温煅烧，制得CaO多少克？
解：m(CaCO₃)=100×80%=80g。CaCO₃ →高温 CaO+CO₂↑
100 : 56 = 80 : x → x = 56×80÷100 = 44.8g
✔ 答：生成CaO 44.8g
模板3 溶液质量分数计算

公式：w = m(溶质) ÷ m(溶液) × 100%  |  稀释：m₁w₁ = m₂w₂

例将20g NaCl放入80g水中全部溶解，求质量分数。再加水20g稀释后的质量分数？
解：原来：w=20÷(20+80)=20%
稀释后：w=20÷(100+20)=16.7%
✔ 注意：稀释溶剂质量增加，溶质质量不变
易错 质量守恒定律计算

反应前后总质量不变，生成气体或沉淀时注意质量差法。

例10g CaCO₃与足量HCl反应，生成CO₂多少克？反应后溶液质量增加了多少？
解：CaCO₃+2HCl→CaCl₂+H₂O+CO₂↑
100g CaCO₃生成44g CO₂ → 10g生成4.4g CO₂
溶液增重 = 10 - 4.4 = 5.6g
✔ 生成的气体逸出，是质量减少的原因
04实验题套路——万变不离其宗
套路1 气体制取装置选择（实验题第一问必考）
🧪 三种气体发生装置对比
固固加热型
适用：加热KMnO₄、加热KClO₃+MnO₂
装置：铁架台+酒精灯+大试管(管口略向下)
注意：试管口放棉花(KMnO₄)，先移导管再熄灯
固液常温型
适用：H₂O₂+MnO₂制O₂、制CO₂、制H₂
装置：锥形瓶+长颈漏斗+双孔塞
注意：长颈漏斗末端要液封（伸入液面下）
📥 收集装置选择
排水法
O₂、H₂（不溶于水）
向上排空气
O₂、CO₂（密度 > 空气）
向下排空气
H₂（密度 < 空气）
制取装置题口诀：固固加热→选大试管｜固液常温→选锥形瓶｜密度大→向上排气
套路2 实验操作"先"与"后"（选择题必考）
🔸 制取气体
✅ 先检查气密性 后装药品
✅ 先通气体(CO/H₂) 后点燃酒精灯
✅ 先移导管 后熄灯（防倒吸）
🔸 实验操作
✅ 先预热 后集中加热
✅ 先通CO 后加热CuO（排空气防爆）
✅ 先熄灯 后停通CO（防氧化）
套路3 常见实验现象描述模板
实验现象关键词满分描述
红磷燃烧大量白烟产生大量白烟，放热
铁在O₂中燃烧火星四射剧烈燃烧，火星四射，生成黑色固体
CO还原CuO黑变红黑色粉末逐渐变红，生成使石灰水变浑浊的气体
CO₂通入石蕊变红紫色石蕊溶液变红（CO₂+H₂O→H₂CO₃）
NaOH溶液滴酚酞变红无色酚酞溶液变红（碱性）
铁与CuSO₄反应红色固体铁表面覆盖一层红色固体，溶液由蓝变浅绿
套路：现象描述 = 反应物变化 + 生成物特征 + 能量变化（放热/吸热）
05推断题突破口——颜色·现象·条件
突破口1 特征颜色速查（推断题关键线索）
黑色固体：C、CuO、Fe₃O₄、MnO₂、Fe粉
红色固体：Cu、Fe₂O₃、红磷P
蓝色溶液：CuSO₄、CuCl₂、Cu(NO₃)₂
黄色溶液：FeCl₃（含Fe³⁺）
浅绿溶液：FeSO₄、FeCl₂（含Fe²⁺）
白色沉淀：CaCO₃↓、BaSO₄↓、AgCl↓、Mg(OH)₂↓
🔥 看到"黑色+红色" → CuO/C → Cu 是中考最常考的组合
突破口2 特征反应现象
现象推断结论
使澄清石灰水变浑浊气体为CO₂
使带火星木条复燃气体为O₂
使无水CuSO₄由白变蓝产物中有水H₂O
点燃产生淡蓝色火焰气体为H₂（或CO、CH₄）
通入AgNO₃产生白色沉淀溶液中有Cl⁻
加入BaCl₂产生白色沉淀溶液中有SO₄²⁻
加稀盐酸产生使石灰水变浑浊的气体含有CO₃²⁻（碳酸盐）
突破口3 物质转化关系常见"题眼"
🔑 A →通电→ H₂ + O₂  → A是水
🔑 A →高温→ CaO + CO₂↑  → A是CaCO₃
🔑 A →MnO₂→ H₂O + O₂↑  → A是H₂O₂
🔑 最常见的液体 → H₂O ｜ 最常见的固体 → C
🔑 最常见的金属 → Fe ｜ 最常用的溶剂 → H₂O

这些陷阱在选择题中反复出现，记住套路就能一眼识破
陷阱1下列变化属于化学变化的是：A. 干冰升华 B. 酒精挥发 C. 铁生锈 D. 电灯发光
陷阱：⚠️ 混淆物理变化和化学变化。
✅ 选C。有新物质生成才是化学变化。干冰升华、酒精挥发、电灯发光都是物理变化。
✔ 口诀："生锈腐败燃烧冶炼"是化学变化，"挥发融化升华溶解"是物理变化。
陷阱2下列说法正确的是：A. 原子是不能再分的最小粒子 B. 分子是保持物质性质的最小粒子 C. 由同种元素组成的物质是单质 D. 水和冰是由同种分子构成的
陷阱：⚠️ 每个选项都有坑。
A错——原子还可以分成原子核和电子。
B错——分子保持的是化学性质，不是全部性质。
C错——同种元素可能组成混合物（如O₂和O₃混合）。
✅ D对——水和冰的化学式都是H₂O，分子相同。
陷阱3实验室制取CO₂时，下列试剂能选用的是：A. 碳酸钙和稀硫酸 B. 碳酸钠和稀盐酸 C. 大理石和稀盐酸 D. 石灰石和浓盐酸
陷阱：⚠️ 选试剂要同时考虑"反应速度"和"气体纯度"。
A错——CaCO₃+H₂SO₄反应生成CaSO₄微溶，覆盖在表面阻止反应继续。
B错——Na₂CO₃+HCl反应太快，不易控制。
✅ C对——大理石+HCl反应速度适中，适合实验室制取。
D错——浓盐酸挥发性强，使CO₂中混有HCl气体。
陷阱420℃时，将40g NaCl放入100g水中，充分溶解后得到溶液的质量为（已知20℃时NaCl溶解度为36g）：A. 140g B. 136g C. 40g D. 100g
陷阱：⚠️ 审题！给了溶解度，说明40g不能全部溶解！
20℃时100g水最多溶36g NaCl → 溶液质量=100+36=136g
✅ 选B。多出的4g NaCl没有溶解，不计算在溶液质量中。
陷阱5区别CO和H₂，最好的方法是：A. 闻气味 B. 点燃后观察火焰颜色 C. 点燃后检验产物 D. 通入紫色石蕊试液
陷阱：⚠️ CO和H₂都是无色无味（A排除），点燃都是淡蓝色火焰（B排除），都不与石蕊反应（D排除）。
✅ 选C。CO燃烧产物是CO₂（使石灰水变浑浊），H₂燃烧产物是H₂O（使无水CuSO₄变蓝）。
✔ 核心：鉴别气体要利用燃烧产物的差异！
陷阱6用pH试纸测定某溶液的pH时，规范的操作是：A. 将pH试纸浸入溶液中 B. 用玻璃棒蘸取溶液滴在pH试纸上 C. 将溶液倒在pH试纸上 D. 将pH试纸先用水润湿再测
陷阱：⚠️ pH试纸使用方法的陷阱。
A和C错——不能把试纸浸入或倒入溶液，会污染试剂。
D错——用水润湿会稀释待测液，导致pH不准（酸性偏大、碱性偏小）。
✅ 选B。用洁净干燥的玻璃棒蘸取溶液滴在pH试纸上，与标准比色卡对比。
