七年级上册

第一单元 生物和生物圈

一、认识生物

  • 生物的特征：①需要营养 ②能呼吸 ③排出废物 ④对外界刺激做出反应 ⑤生长繁殖 ⑥遗传变异 ⑦除病毒外由细胞构成
  • 调查法：科学探究常用方法，要明确目的和对象，制定合理方案

二、生物圈

  • 生物圈：地球上所有生物与其环境的总和，是最大的生态系统
  • 生态因素：非生物因素（光、温度、水、空气等）+ 生物因素（捕食、竞争、合作、寄生等）
  • 生态系统：生物部分（生产者、消费者、分解者）+ 非生物部分
  • 食物链：生产者和消费者之间吃与被吃的关系，起点是生产者，箭头指向捕食者
  • 物质和能量沿食物链流动，能量逐级递减（10%-20%）
  • 生态系统的类型：森林（"绿色水库"）、湿地（"地球之肾"）、海洋、草原、淡水、农田、城市

第二单元 细胞

一、显微镜的使用

  • 放大倍数 = 目镜倍数 × 物镜倍数
  • 目镜越长放大倍数越小，物镜越长放大倍数越大
  • 成像：倒像（上下左右均颠倒），如"b"在视野中为"q"
  • 视野中物像偏哪，玻片向哪移
  • 从低倍镜换高倍镜：视野变暗，细胞数目变少，细胞体积变大

二、细胞的结构

动植物细胞共有的结构：

细胞膜—控制物质进出；细胞质—生命活动的重要场所；细胞核—含遗传信息，生命活动的控制中心；线粒体—呼吸作用，释放能量

植物细胞特有的结构：

细胞壁—支持和保护；叶绿体—光合作用的场所；液泡—含细胞液（切水果流出的汁液）

三、细胞的生活

  • 物质由分子构成，物质进出细胞通过细胞膜
  • 能量转换器：叶绿体（光能→化学能）、线粒体（化学能→生命活动所需能量）
  • 细胞核是遗传信息库，DNA是主要的遗传物质，染色体 = DNA + 蛋白质
  • 细胞分裂：细胞核先分裂 → 细胞质分裂 → 形成新细胞膜（植物还有新细胞壁）
  • 细胞分裂过程中染色体先复制加倍再均分，新细胞与原细胞染色体数目相同

第三单元 生物圈中的绿色植物

一、植物类群

类群代表特点
藻类衣藻、水绵、海带无根茎叶分化，无输导组织
苔藓葫芦藓、地钱有茎叶（无真正的根），监测空气污染指示植物
蕨类肾蕨、贯众有根茎叶，有输导组织，靠孢子繁殖
裸子植物松、柏、杉、银杏种子裸露无果皮包被，"活化石"银杏
被子植物大多数常见植物种子有果皮包被，六大器官齐全

二、种子的萌发

  • 条件：自身条件—完整有活力的胚 + 充足的营养 + 未休眠
    环境条件—适宜的温度、一定的水分、充足的空气
  • 种子结构：胚（胚芽→茎叶、胚轴→连接、胚根→根、子叶→营养）
  • 单子叶（玉米）营养在胚乳，双子叶（菜豆）营养在子叶

三、光合作用与呼吸作用

光合作用

公式：CO₂ + H₂O →（光、叶绿体）→ 有机物 + O₂

实质：合成有机物，贮存能量；场所：叶绿体

影响因素：光照强度、CO₂浓度、温度、水

应用：合理密植、立体种植、提高大棚CO₂浓度

呼吸作用

公式：有机物 + O₂ →（线粒体）→ CO₂ + H₂O + 能量

实质：分解有机物，释放能量；场所：线粒体

影响因素：温度、水分、氧气

应用：田间松土、低温储存蔬菜水果、粮食晒干保存

光合作用与呼吸作用的关系：光合作用为呼吸作用提供有机物和氧气，呼吸作用为光合作用提供二氧化碳和能量。二者相互依存。

四、其他重要内容

  • 蒸腾作用：水分从气孔散失，拉动水和无机盐向上运输，降低叶片温度
  • 气孔：由保卫细胞控制开闭，是蒸腾作用的"门户"、气体交换的"窗口"
  • 根吸水的主要部位：根尖的成熟区（根毛区）
  • 导管：运输水和无机盐（自下而上）；筛管：运输有机物（自上而下）
  • 花的结构：花蕊最重要（雄蕊和雌蕊），子房发育成果实，胚珠发育成种子

七年级下册

第四单元 生物圈中的人

一、人的由来

  • 人类起源于森林古猿，证据：化石
  • 进化过程：直立行走（最重要的变化）→ 制造工具 → 大脑发达 → 语言
  • 生殖系统：睾丸（产生精子、分泌雄性激素）、卵巢（产生卵细胞、分泌雌性激素）
  • 受精：精子和卵细胞在输卵管结合形成受精卵
  • 胚胎发育：子宫内发育，通过胎盘和脐带从母体获取营养和氧气
  • 青春期发育特点：身高体重突增、生殖器官发育成熟、第二性征出现、大脑功能增强

二、消化系统

  • 消化道：口 → 咽 → 食道 → 胃 → 小肠 → 大肠 → 肛门
  • 消化腺：唾液腺（淀粉酶）、肝脏（最大的消化腺，分泌胆汁）、胰腺、胃腺、肠腺
  • 三大营养物质的消化：
    • 淀粉：口腔（唾液淀粉酶）→ 麦芽糖 → 小肠 → 葡萄糖
    • 蛋白质：胃（胃蛋白酶）→ 多肽 → 小肠 → 氨基酸
    • 脂肪：小肠（胆汁乳化）→ 脂肪微粒 → 小肠（酶）→ 甘油+脂肪酸
  • 小肠是消化和吸收的主要场所（长、大、薄的特点）
  • 缺乏维生素 → 夜盲症（VA）、脚气病（VB₁）、坏血病（VC）、佝偻病（VD）

三、呼吸系统

  • 组成：呼吸道（鼻→咽→喉→气管→支气管）+ 肺（呼吸系统的主要器官）
  • 呼吸运动：吸气时膈肌收缩（膈顶下降），胸廓扩大，肺内气压降低
  • 肺泡适于气体交换的特点：数量多、壁薄（一层细胞）、外有毛细血管网
  • 气体交换：扩散作用，氧气从肺泡进入血液，二氧化碳从血液进入肺泡

四、循环系统

  • 血液：血浆（运输）+ 血细胞（红细胞—运输氧气、白细胞—吞噬病菌、血小板—止血凝血）
  • 血管：动脉（离心、壁厚、血流快）、静脉（回心、有静脉瓣）、毛细血管（连通最小动脉静脉、壁极薄）
  • 心脏：四腔—左心房、左心室、右心房、右心室（左心室的壁最厚）
  • 体循环：左心室 → 主动脉 → 全身毛细血管 → 上/下腔静脉 → 右心房（动脉血→静脉血）
  • 肺循环：右心室 → 肺动脉 → 肺部毛细血管 → 肺静脉 → 左心房（静脉血→动脉血）
  • 血型与输血：以输同型血为原则，O型是"万能供血者"，AB型是"万能受血者"

五、泌尿系统

  • 组成：肾脏（形成尿液）、输尿管、膀胱（暂时贮存尿液）、尿道
  • 肾单位：肾小球 + 肾小囊 + 肾小管
  • 尿液形成：肾小球过滤（血细胞和大分子蛋白不能滤过）→ 肾小管重吸收（全部葡萄糖、大部分水、部分无机盐）

六、神经调节

  • 神经系统：中枢神经系统（脑+脊髓）+ 周围神经系统（脑神经+脊神经）
  • 神经元：神经系统的结构和功能单位
  • 反射：通过神经系统对刺激作出的规律性反应
    简单反射（天生，如膝跳反射、缩手反射）、复杂反射（后天，如望梅止渴）
  • 反射弧：感受器→传入神经→神经中枢→传出神经→效应器
  • 感觉器官：眼（晶状体相当于镜头，视网膜成像，近视配凹透镜）；耳（耳蜗内有听觉感受器）

七、激素调节

激素来源作用/异常
生长激素垂体侏儒症（幼年不足）、巨人症（幼年过多）
甲状腺激素甲状腺呆小症（幼年不足）、甲亢（过多），缺碘→大脖子病
胰岛素胰岛降低血糖，不足→糖尿病
肾上腺素肾上腺促使心跳加快、血压升高、血糖升高

八年级上册

第五单元 生物圈中的其他生物

一、动物的主要类群

无脊椎动物
  • 腔肠动物：水螅、水母、珊瑚虫 — 有口无肛门，有刺细胞
  • 扁形动物：涡虫、血吸虫 — 有口无肛门，背腹扁平
  • 线形动物：蛔虫、线虫 — 有口有肛门，体表有角质层
  • 环节动物：蚯蚓、沙蚕 — 身体分节，靠刚毛或疣足运动
  • 软体动物：河蚌、蜗牛、乌贼 — 身体柔软，外套膜，贝壳
  • 节肢动物：蝗虫、蜜蜂、蜘蛛、虾、蟹 — 最大的动物类群，体表有外骨骼，足和触角分节
脊椎动物
  • 鱼类：用鳃呼吸，用鳍游泳，体表有鳞片，侧线感知水流和方向
  • 两栖动物：青蛙、蟾蜍 — 幼体用鳃呼吸（水中），成体用肺+皮肤呼吸（陆上），变态发育
  • 爬行动物：蜥蜴、蛇、龟、鳄 — 用肺呼吸，真正陆生脊椎动物（受精卵有卵壳保护）
  • 鸟类：身体流线型，前肢变成翼，双重呼吸（气囊辅助肺呼吸），恒温
  • 哺乳动物：胎生哺乳，体表有毛，牙齿分化（门齿、犬齿、臼齿），最高等的动物类群

二、动物的运动和行为

  • 运动系统：骨骼 + 骨骼肌（骨是杠杆、关节是支点、骨骼肌提供动力）
  • 关节：关节面（关节头+关节窝）+ 关节囊 + 关节腔（含滑液）
  • 骨骼肌受神经刺激收缩牵动骨绕关节活动
  • 先天性行为：生来就有（如蜘蛛结网、蜜蜂采蜜）
  • 学习行为：后天形成，建立在遗传因素基础上（如鹦鹉学舌、猴子骑车）
  • 社会行为：群体内有组织、有分工、有等级（如蚂蚁、蜜蜂、狒狒）

三、细菌和真菌

项目细菌真菌
细胞结构无成形的细胞核（原核生物）有成形的细胞核（真核生物）
代表球菌、杆菌、螺旋菌酵母菌（单细胞）、霉菌、蘑菇
繁殖方式分裂繁殖孢子繁殖（酵母菌可出芽）
营养方式多数异养（寄生或腐生）异养（腐生或寄生）
  • 腐生：分解有机物，在自然界物质循环中起重要作用
  • 病毒：无细胞结构，由蛋白质外壳+内部遗传物质组成，必须在活细胞内寄生

第六单元 生物的多样性及其保护

  • 分类单位：界门纲目科属种（种是最基本的分类单位）
  • 亲缘关系：同种最亲，分类单位越小共同特征越多
  • 生物多样性包含三个层次：物种多样性、遗传（基因）多样性、生态系统多样性
  • 保护生物多样性最有效的措施：建立自然保护区

八年级下册

第七单元 生物圈中生命的延续和发展

一、生物的生殖和发育

  • 有性生殖：两性生殖细胞结合形成受精卵（后代有变异，适应力强）
  • 无性生殖：不经过两性生殖细胞结合（如扦插、嫁接、压条、组织培养）
    — 嫁接：接穗+砧木，成活的关键是使形成层紧密结合
  • 完全变态发育：卵→幼虫→蛹→成虫（如家蚕、蝶、蚊、蝇）
  • 不完全变态发育：卵→若虫→成虫（如蝗虫、蟋蟀、蜻蜓）
  • 两栖动物的发育：变态发育（蝌蚪→青蛙）
  • 鸟卵：胚盘（发育成雏鸟）、卵黄（营养）、卵白（营养+水分）、气室（空气）

二、生物的遗传和变异

  • 基因是有遗传效应的DNA片段，DNA在染色体上
  • 体细胞中染色体成对存在，生殖细胞中成单存在
  • 人的体细胞有23对染色体（22对常染色体+1对性染色体）
    男性：XY，女性：XX
  • 基因的显隐性：显性基因控制显性性状（大写字母），隐性基因控制隐性性状（小写字母）
    如：Aa 表现显性性状，aa 才表现隐性性状
  • 遗传图解：亲代→配子→子代基因型→子代表现型
  • 变异：可遗传变异（遗传物质改变，如太空育种）、不可遗传变异（仅环境引起）
  • 杂交水稻——袁隆平院士的贡献，利用基因多样性育种

三、生物的进化

  • 研究进化的最重要证据：化石（越古老地层生物越简单、越低等）
  • 自然选择学说（达尔文）：过度繁殖 → 生存斗争 → 遗传变异 → 适者生存
  • 自然选择的结果：生物适应环境、生物多样性的形成
  • 典型例子：长颈鹿的长颈、桦尺蛾体色变化、抗药性的形成

第八单元 健康地生活

一、传染病和免疫

  • 传染病三环节：传染源、传播途径、易感人群
  • 预防措施：控制传染源（隔离病人）、切断传播途径（戴口罩、消毒）、保护易感人群（接种疫苗）
  • 病原体：细菌、病毒、寄生虫等
  • 免疫：
    • 非特异性免疫（天生，对多种病原体有效）— 皮肤屏障、吞噬细胞、溶菌酶
    • 特异性免疫（后天，针对特定病原体）— 抗体+抗原，如接种疫苗
  • 疫苗：通常用失活的或减毒的病原体制成，属于抗原
  • 计划免疫：有计划的预防接种（如卡介苗、乙肝疫苗）

二、用药与急救

  • 安全用药：处方药（Rx）需医生开，非处方药（OTC）可自购
  • 注意药品的有效期、用法用量、禁忌
  • 急救方法：心肺复苏（胸外按压+人工呼吸）、止血包扎、骨折固定
  • 出血判断：动脉出血（鲜红、喷射）— 近心端止血；静脉出血（暗红、流出）— 远心端止血

三、健康的生活方式

  • 健康的三个维度：身体健康、心理健康、社会适应良好
  • 毒品危害极大，必须坚决抵制
  • 吸烟、酗酒危害循环系统和呼吸系统
  • 合理营养、规律作息、适度运动
