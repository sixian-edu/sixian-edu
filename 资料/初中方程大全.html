一元一次方程
🔹 一元一次方程
标准形式：ax + b = 0（a≠0），有且只有一个解。
解法步骤：去括号移项合并同类项系数化为1
ax + b = 0  ⇒  x = -b/a  （a≠0）
🔹 去分母解法
方程中含有分母时，两边同乘各分母的最小公倍数去分母。
注意 分子是多顶式时要加括号，去分母后勿漏乘不含分母的项。
x/2 + 1 = x/3 − 2  →  3x+6=2x−12  →  x = −18
二元一次方程组
🔹 二元一次方程
标准形式：ax + by = c（a,b不同时为0）。
有无数组解，每一组解对应平面直角坐标系中一条直线上的点。
ax + by = c  |  解为直线上的点
🔹 代入消元法
将一个方程中的某个未知数用含另一个未知数的式子表示，代入另一个方程消元。
变形代入回代
{ x+y=5, x−y=1 } → y=5−x → x−(5−x)=1 → x=3, y=2
🔹 加减消元法
使某一未知数的系数相等或相反，两方程相加或相减消去该未知数。
同号相减异号相加
{ 3x+2y=7, x−2y=5 } → 相加得 4x=12 → x=3, y=−1
🔹 三元一次方程组
含有三个未知数，每个方程含未知数的项次数为1。
消元策略：三元 → 二元二元 → 一元
选两个方程消去一个未知数，再选两个方程消去同一个未知数。
三元消元 → 二元一次方程组 → 一元一次方程
一元二次方程
🔹 一元二次方程·一般式
标准形式：ax² + bx + c = 0（a≠0）。
核心解法四种：配方法公式法因式分解法直接开平方法
ax² + bx + c = 0  （a≠0）
🔹 配方法
通过配方将方程化为 (x + m)² = n 的形式，再开平方求解。
移常数项两边加(b/2)²开平方
x²+6x+5=0 → (x+3)²=4 → x+3=±2 → x₁=−1, x₂=−5
🔹 公式法
利用求根公式直接求解，适用于所有一元二次方程。
Δ = b²−4acΔ ≥ 0
x = [−b ± √(b² − 4ac)] / 2a  （a≠0, Δ≥0）
🔹 因式分解法
将方程化为 (x − x₁)(x − x₂) = 0 的形式。
十字相乘法：x²+(p+q)x+pq = (x+p)(x+q)
提取公因式：ax²+bx = x(ax+b)平方差公式
(x − x₁)(x − x₂) = 0  ⇒  x = x₁ 或 x = x₂
🔹 判别式 Δ = b² − 4ac
判断一元二次方程根的情况：
Δ > 0 两个不相等的实数根
Δ = 0 两个相等的实数根（重根）
Δ < 0 无实数根（有共轭复数根）
Δ = b² − 4ac  |  Δ≥0 有实根  |  Δ<0 无实根
🔹 韦达定理（根与系数关系）
若 x₁, x₂ 为 ax²+bx+c=0 的两根，则：
x₁ + x₂ = −b/ax₁·x₂ = c/a
应用：已知一根求另一根求对称式值构造方程
x₁ + x₂ = −b/a  |  x₁·x₂ = c/a
分式方程
🔹 分式方程·解法
分母中含未知数的方程。两边同乘最简公分母化为整式方程求解。
重要 必须验根！
2/(x+1) = 3/x  →  2x = 3(x+1)  →  x = −3
🔹 分式方程·增根
去分母后整式方程的根若使最简公分母为0，则为增根，舍去。
增根不是原方程的根必须验根！
检验：代入最简公分母 ≠ 0  ⇒  是原方程的根
根式与绝对值方程
🔹 根式（无理）方程
根号内含有未知数的方程。通过两边平方去根号。
注意 平方可能产生增根，必须检验。
步骤：移项使根号单独平方解方程验根
√(2x+3) = 5  →  2x+3 = 25  →  x = 11
🔹 绝对值方程
利用绝对值的定义，分类讨论脱去绝对值符号。
|x| = a (a≥0) ⇒ x = ±a|x−a| = b (b≥0) ⇒ x = a±b
|x − 3| = 5  →  x−3 = 5 或 x−3 = −5  →  x = 8 或 x = −2
换元法与特殊方程
🔹 换元法
设辅助未知数 t 简化方程，实现降次或化繁为简。
常用于：高次方程分式方程根式方程对称式
(x²−3x)² − 2(x²−3x) − 8 = 0  →  令 t=x²−3x  →  t=4 或 t=−2
🔹 双二次方程
形式：ax⁴ + bx² + c = 0（a≠0）。
令 t = x²（t ≥ 0），化为 at² + bt + c = 0，解出 t 后回代求 x。
注意 t ≥ 0
x⁴ − 5x² + 4 = 0  →  t=x²  →  t²−5t+4=0  →  x=±1, ±2
不等式
🔹 一元一次不等式
解法与一元一次方程类似，但需注意：
乘除负数要变号！ 解集在数轴上表示：空心圈不含等实心圈含等
−2x > 4  →  x < −2  （注意变号）
🔹 不等式组
口诀：
同大取大同小取小大小小大中间找大大小小找不到
解集是各不等式解集的公共部分。
{ x > 2, x ≤ 5 }  →  2 < x ≤ 5
🔹 一元二次不等式
结合二次函数图像求解（a>0 时）：
ax²+bx+c > 0 ⇒ x < x₁ 或 x > x₂ax²+bx+c < 0 ⇒ x₁ < x < x₂
口诀：大于取两边，小于取中间。
x² − x − 6 > 0  →  (x−3)(x+2) > 0  →  x < −2 或 x > 3
含参方程
🔹 含参数方程·分类讨论
参数使最高次系数可能为0时，需分类讨论：
系数=0 降次为一元一次或更低
系数≠0 按一元二次方程求解，结合Δ讨论根的情况
(k−1)x² − 2kx + k + 2 = 0  |  k=1: 一元一次  |  k≠1: 用Δ讨论
