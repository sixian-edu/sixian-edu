专题一：静电场

静电场

电荷与库仑定律：点电荷间的作用力满足 F = k|q₁q₂|/r²（k=9×10⁹ N·m²/C²）。同种电荷相互排斥，异种电荷相互吸引。电荷守恒定律：电荷既不能创造也不能消灭，只能从一个物体转移到另一个物体。

电场强度：E = F/q（定义式），单位 N/C 或 V/m。点电荷的电场 E = kQ/r²。电场强度是矢量，方向与正电荷在该点所受电场力方向相同。

电场线：从正电荷出发终止于负电荷（或无穷远），疏密表示场强大小。电场线不相交，不闭合。匀强电场的电场线是平行等距的直线。

电势与电势差：U_AB = φ_A − φ_B = W_AB/q。电场力做功与路径无关：W_AB = qU_AB。沿着电场线方向电势逐渐降低。等势面：在同一等势面上移动电荷电场力不做功，等势面⊥电场线。

电容器与电容：C = Q/U = εS/(4πkd)（平行板电容器）。电容表征电容器容纳电荷的本领，与 Q、U 无关。平行板电容器充电后保持与电源相连则 U 不变，断开电源则 Q 不变。

点电荷的电场线：正电荷向外发散，负电荷向内汇聚
例 1：两个点电荷 q₁=2×10⁻⁸C 和 q₂=−4×10⁻⁸C 相距 0.1m，求它们之间的库仑力大小和方向。
解：F = k|q₁q₂|/r² = 9×10⁹ × (2×10⁻⁸×4×10⁻⁸)/(0.1)²
= 9×10⁹ × 8×10⁻¹⁶/0.01 = 7.2×10⁻⁴ N。
异种电荷相互吸引。
例 2：在电场中某点放入 q=4×10⁻⁹C 的试探电荷，受到 2×10⁻⁵N 的电场力，求该点场强。若换为 q=−2×10⁻⁹C，场强和电场力如何？
解：E = F/q = 2×10⁻⁵/(4×10⁻⁹) = 5000 N/C，方向与 F 相同。
场强与试探电荷无关，E 不变仍为 5000 N/C。
F' = q'E = (−2×10⁻⁹)×5000 = −1×10⁻⁵ N（负号表示方向与 E 相反）。
例 3：一带电粒子从电场中 A 点移到 B 点电场力做功 4×10⁻⁶J，粒子带正电 q=2×10⁻⁹C，求 U_AB。
解：U_AB = W_AB/q = 4×10⁻⁶/(2×10⁻⁹) = 2000 V。
U_AB > 0 → φ_A > φ_B，电势从 A 到 B 降低了 2000V。
例 4：平行板电容器 S=0.01m²，d=2mm，ε_r=1，充电到 U=100V 后断开电源，求电容 C、带电量 Q。若将板间距增大到 4mm，C 和 Q 如何变化？
解：C₀ = εS/(4πkd) = 8.85×10⁻¹²×0.01/(2×10⁻³)=4.43×10⁻¹¹F。
Q = CU = 4.43×10⁻¹¹×100 = 4.43×10⁻⁹C。
断开电源后 Q 不变。d 增大到 2 倍 → C 减半 → U=Q/C 增大到 2 倍。
• k=9×10⁹，q₁=2×10⁻⁷C，q₂=5×10⁻⁷C，r=0.05m，求库仑力 答案：0.36N
• 点电荷 Q 在 r=0.3m 处产生 E=200N/C，求 Q 答案：2×10⁻⁹C
• 电场中 A→B 电场力做功 −6×10⁻⁶J，q=−3×10⁻⁹C，求 U_AB 答案：2000V

专题二：恒定电流

恒定电流

电流：I = q/t。规定正电荷定向移动的方向为电流方向。金属导体中为自由电子的定向移动形成电流，但电流方向与电子移动方向相反。

欧姆定律：I = U/R（适用于金属导体和电解液，不适用于气体导电和半导体）。电阻定律：R = ρL/S，ρ 为电阻率，随温度升高而增大（金属）。

串并联电路：
串联：I 相等，U 分压，R = R₁+R₂+…，U₁/U₂ = R₁/R₂
并联：U 相等，I 分流，1/R = 1/R₁+1/R₂+…，I₁/I₂ = R₂/R₁

电功与电功率：W = UIt（焦耳定律：Q = I²Rt，纯电阻电路中 Q = W）。P = UI（纯电阻 P=I²R=U²/R）。电源功率：P总=EI，P出=UI，P内=I²r。

闭合电路欧姆定律：I = E/(R+r)，路端电压 U = E − Ir
例 1：电阻 R₁=6Ω，R₂=12Ω 并联后接入 12V 电路，求总电流和每个电阻的电流。
解：并联 R = (6×12)/(6+12)=72/18=4Ω。
I_total = U/R = 12/4 = 3A。
I₁ = U/R₁ = 12/6 = 2A，I₂ = U/R₂ = 12/12 = 1A。
I₁+I₂=3A 等于总电流。✓
例 2：电源 E=12V，r=1Ω，外电路 R=5Ω，求路端电压 U 和内电压 U'。
解：I = E/(R+r) = 12/(5+1) = 2A。
路端电压 U = IR = 2×5 = 10V。
内电压 U' = Ir = 2×1 = 2V。
验证：U + U' = 10+2 = 12V = E。✓
例 3：一根铜导线长 100m，横截面积 2mm²，ρ=1.7×10⁻⁸Ω·m，求电阻。
解：R = ρL/S = 1.7×10⁻⁸×100/(2×10⁻⁶) = 1.7×10⁻⁶/(2×10⁻⁶) = 0.85Ω。
例 4：额定电压 220V、功率 100W 的白炽灯正常发光时的电流和电阻。
解：I = P/U = 100/220 ≈ 0.455A。
R = U²/P = 220²/100 = 48400/100 = 484Ω（热态电阻，冷态更小）。
• R₁=4Ω，R₂=6Ω 串联，总电压 20V，求 U₁ 和 U₂ 答案：U₁=8V，U₂=12V
• E=9V，r=0.5Ω，外电路 R=4Ω，求 I 和路端电压 答案：I=2A，U=8V
• 220V·800W 电炉的电阻和电流 答案：R=60.5Ω，I≈3.64A

专题三：磁场

磁场重点

磁感应强度：B = F/(IL)（定义式），单位特斯拉（T）。方向为小磁针 N 极受力方向。磁感线：闭合曲线，外部从 N→S，内部从 S→N。

安培力：F = BIL sinθ（θ 为 B 与 I 的夹角）。方向用左手定则判断：伸开左手，使拇指与其余四指垂直，让磁感线垂直穿入手心，四指指向电流方向，拇指所指方向即为安培力方向。

洛伦兹力：F = qvB sinθ（θ 为 v 与 B 的夹角）。方向同样用左手定则判断（注意负电荷的等效电流方向与运动方向相反）。洛伦兹力永远不做功，只改变速度方向不改变速度大小。

带电粒子在匀强磁场中的运动：当 v⊥B 时，做匀速圆周运动。
轨道半径：r = mv/(qB)；周期：T = 2πm/(qB)（与速度、半径无关！）。

洛伦兹力提供向心力，粒子做匀速圆周运动。周期 T=2πm/(qB) 与速度无关
例 1：长 L=0.2m 的直导线，通有 I=3A 的电流，放在 B=0.5T 的匀强磁场中，导线与磁场方向垂直，求安培力。
解：F = BIL = 0.5×3×0.2 = 0.3 N。方向由左手定则判断。
例 2：α粒子（q=2e=3.2×10⁻¹⁹C，m=6.64×10⁻²⁷kg）以 v=2×10⁶m/s 垂直进入 B=0.5T 的匀强磁场，求轨道半径 r 和周期 T。
解：r = mv/(qB) = 6.64×10⁻²⁷×2×10⁶/(3.2×10⁻¹⁹×0.5)
= 1.328×10⁻²⁰/(1.6×10⁻¹⁹) = 0.083 m = 8.3 cm。
T = 2πm/(qB) = 2×3.14×6.64×10⁻²⁷/(3.2×10⁻¹⁹×0.5)
= 4.17×10⁻²⁶/(1.6×10⁻¹⁹) = 2.6×10⁻⁷ s。
例 3：电子以 v=4×10⁶m/s 水平射入 B=0.2T 的竖直向下的匀强磁场，求电子的轨道半径。（m_e=9.1×10⁻³¹kg，e=1.6×10⁻¹⁹C）
解：r = mv/(eB) = 9.1×10⁻³¹×4×10⁶/(1.6×10⁻¹⁹×0.2)
= 3.64×10⁻²⁴/(3.2×10⁻²⁰) = 1.14×10⁻⁴ m = 0.114 mm。
• I=2A，L=0.3m，B=0.4T，θ=30°，求安培力 答案：0.12N
• 质子 q=1.6×10⁻¹⁹C，v=3×10⁶m/s，B=0.1T，求 r 答案：0.188m
• 带电粒子在匀强磁场中做圆周运动的周期与什么有关？答案：只与 m、q、B 有关，与 v、r 无关

专题四：电磁感应

电磁感应核心

法拉第电磁感应定律：感应电动势的大小与穿过回路的磁通量的变化率成正比。
E = n·ΔΦ/Δt（n 为线圈匝数，Φ=BScosθ）。

导体切割磁感线：当导体在磁场中做切割磁感线运动时产生感应电动势。
E = BLv sinθ（θ 为 v 与 B 的夹角）。平动切割时 E=BLv（B⊥L⊥v）。

楞次定律：感应电流的磁场总要阻碍引起感应电流的磁通量的变化。口决：增反减同，来拒去留。
右手定则：伸开右手，让磁感线穿入手心，拇指指向导体运动方向，四指指向感应电流方向。

感生电动势与动生电动势：
① 感生电动势：磁场变化产生涡旋电场，E = n·S·ΔB/Δt（S 为面积）。
② 动生电动势：导体切割磁感线，E = BLv。在匀强磁场中转动时 E = ½BωL²。

自感现象：由于导体自身电流变化而产生的电磁感应现象。自感系数 L 与线圈的形状、大小、匝数以及有无铁芯有关。

导体棒切割磁感线：ε = BLv，感应电流方向由右手定则判断
楞次定律：增反减同——磁铁 N 极插入时线圈感应电流产生向上的磁场阻碍磁通量增加
例 1：线圈 n=100 匝，ΔΦ=0.02Wb 在 Δt=0.1s 内均匀变化，求感应电动势。
解：E = n·|ΔΦ/Δt| = 100×0.02/0.1 = 20 V。
例 2：导体棒 L=0.5m，以 v=4m/s 匀速切割 B=0.6T 的匀强磁场，B⊥L⊥v，求 E 和回路 R=2Ω 时的 I。
解：E = BLv = 0.6×0.5×4 = 1.2 V。
I = E/R = 1.2/2 = 0.6 A。由右手定则判断电流方向。
例 3：线圈面积 S=0.04m²，n=50 匝，磁场 B 以 0.2T/s 的变化率均匀增大，求感应电动势。
解：E = n·S·ΔB/Δt = 50×0.04×0.2 = 0.4 V。
例 4：光滑平行导轨间距 L=0.4m，电阻 R=0.5Ω，导体棒质量 m=0.1kg，B=0.5T，给棒一个水平向右的初速度 v₀=6m/s，求棒的最大感应电流和最终运动状态。
解：v₀ 时 E₀=BLv₀=0.5×0.4×6=1.2V。
I₀=E₀/R=1.2/0.5=2.4A。
安培力 F₀=BIL=0.5×2.4×0.4=0.48N，阻碍运动。
棒做减速运动，v↓→E↓→I↓→F↓，最终 趋于静止。
（若有外力维持则最终可能匀速，平衡时 F_ext = F_安 = B²L²v/R）
• n=200 匝，0.1s 内磁通量从 0.01Wb 变为 0.05Wb，求 E 答案：80V
• 导体棒 L=0.3m，v=5m/s，B=0.8T，求 E 答案：1.2V
• 矩形线圈在匀强磁场中匀速转动，产生的感应电动势按什么规律变化？答案：正弦规律 e=Eₘsinωt

专题五：交变电流

交变电流

正弦式交流电：线圈在匀强磁场中绕垂直于磁场的轴匀速转动产生。
e = Eₘ sin ωt，其中 Eₘ = nBSω（电动势峰值）。
i = Iₘ sin(ωt − φ)，u = Uₘ sin(ωt − φ′)。

有效值：交流电与直流电在相同时间内通过同一电阻产生相同热量的等效值。
E = Eₘ/√2，U = Uₘ/√2，I = Iₘ/√2（仅适用于正弦式交流电）。

变压器：U₁/U₂ = n₁/n₂。理想变压器：P₁ = P₂ → U₁I₁ = U₂I₂ → I₁/I₂ = n₂/n₁。

远距离输电：P_损 = I²R_线。提高输电电压可减小电流，从而减小线路损耗。高压输电：升压→传输→降压。

正弦式交流电：e = Eₘsinωt（电动势），i = Iₘsin(ωt−φ)（电流落后相位φ）
变压器原理：U₁/U₂ = n₁/n₂，I₁/I₂ = n₂/n₁（理想变压器无能量损耗）
例 1：正弦交流电 e=310sin(314t)V，求峰值 Eₘ、有效值 E、频率 f 和周期 T。
解：Eₘ = 310 V。E = 310/√2 ≈ 219 V。
ω=314 rad/s，f=ω/(2π)=314/6.28=50 Hz。T=1/f=0.02 s。
例 2：变压器原线圈 n₁=1100 匝，输入 U₁=220V，副线圈 n₂=50 匝，求输出 U₂。若副线圈接 R=10Ω，求 I₂ 和输入功率。
解：U₂ = U₁·n₂/n₁ = 220×50/1100 = 10 V。
I₂ = U₂/R = 10/10 = 1 A。
P₂ = U₂I₂ = 10×1 = 10W。理想变压器 P₁ = P₂ = 10 W。
I₁ = P₁/U₁ = 10/220 ≈ 0.045 A。
例 3：某交流电通过 R=5Ω 的电阻，一个周期内放热 40J，求该交流电的有效值。
解：由 Q = I²RT，得 I² = Q/(RT)。
一个周期 T=0.02s，Q=40J。
I² = 40/(5×0.02) = 400 → I = 20 A。
例 4：发电厂输出功率 P=100kW，输出电压 U₁=400V，输电线总电阻 R=10Ω，若采用 1:25 升压后输电，求线损和用户得到的电压。
解：不升压时：I=P/U₁=100000/400=250A，P_损=I²R=250²×10=625kW > 100kW（不可行！）。
升压后 U₂=400×25=10000V，I₂=P/U₂=100000/10000=10A。
P_损=I₂²R=10²×10=1 kW（占 1%）。
线路压降 ΔU=I₂R=10×10=100V，末端电压 U=10000−100=9900 V。
• 交流电 u=220√2·sin100πt(V) 的有效值和频率 答案：220V，50Hz
• 变压器 n₁:n₂=10:1，U₁=220V，R₂=5Ω，求 I₂ 和 P₁ 答案：U₂=22V，I₂=4.4A，P₁≈96.8W
• 远距离输电提高电压的好处？答案：减小电流→减小线损（P_损=I²R）

专题六：电磁综合

电磁综合压轴

带电粒子在复合场中的运动："复合场"指电场、磁场、重力场的组合。常见模型：

速度选择器：E 和 B 正交，qE = qvB → v = E/B 的粒子沿直线通过，与电荷正负和质量无关。

质谱仪：粒子在电场中加速 qU = ½mv²，进入磁场偏转 r = mv/(qB)。由 r 测量质量或比荷。
比荷 q/m = 2U/(B²r²)。

回旋加速器：D 形盒在磁场中回旋，缝隙间电场加速。粒子最终速度 v_max = qBR/m，最大动能 E_k = q²B²R²/(2m)，与加速电压无关。

电磁感应综合：导体棒在磁场中运动，涉及牛顿定律、动量定理、能量守恒。常见问题包括含容电路、含电源电路、双棒模型等。

例 1（速度选择器）：一束带电粒子（含正负电荷）以速度 v 进入正交的 E=2000V/m 和 B=0.5T 的区域，求能沿直线穿过的粒子速度。若 v 偏大粒子向哪侧偏转？
解：qE = qvB → v = E/B = 2000/0.5 = 4000 m/s。
与电荷正负、质量、电量均无关。
若 v 偏大，qvB > qE，正电荷向上偏（B 方向），负电荷向下偏。
例 2（质谱仪）：加速电压 U=1000V，磁场 B=0.2T，测得某种离子的圆周轨道半径 r=0.125m，求该离子的比荷 q/m。
解：加速过程：qU = ½mv² → v = √(2qU/m)。
偏转过程：qvB = mv²/r → v = qBr/m。
联立：√(2qU/m) = qBr/m → 2qU/m = q²B²r²/m²
q/m = 2U/(B²r²) = 2×1000/(0.2²×0.125²) = 2000/(0.04×0.015625)
= 2000/0.000625 = 3.2×10⁶ C/kg。
例 3（回旋加速器）：D 形盒半径 R=0.5m，磁场 B=1.5T，加速质子的最大动能 E_k。如果加速电压为 50kV，需要加速多少圈？（m_p=1.67×10⁻²⁷kg）
解：v_max = qBR/m = 1.6×10⁻¹⁹×1.5×0.5/(1.67×10⁻²⁷) = 1.2×10⁻¹⁹/(1.67×10⁻²⁷)
≈ 7.19×10⁷ m/s。
E_k = ½mv_max² = ½×1.67×10⁻²⁷×(7.19×10⁷)² = 4.32×10⁻¹² J。
换算为 eV：E_k = 4.32×10⁻¹²/(1.6×10⁻¹⁹) ≈ 27 MeV。
每圈加速 2 次，每次增加 2qU=2×50=100keV。
圈数 n = 27×10³/100 ≈ 270 圈。
例 4（电磁感应综合）：光滑平行导轨间距 L=0.5m，B=0.8T，电阻 R=4Ω，质量 m=0.2kg 的导体棒从静止开始水平向右在 F=1N 恒力作用下运动，求（1）最大速度；（2）最大速度时的功率。
解：（1）当安培力等于外力时速度最大。
F_安 = BIL = B·(BLv/R)·L = B²L²v/R。
F = B²L²v_m/R → v_m = FR/(B²L²) = 1×4/(0.8²×0.5²) = 4/(0.64×0.25) = 25 m/s。
（2）最大功率 P = F·v_m = 1×25 = 25 W。
此时 P 全部转化为电阻上的热功率：P = I²R = (BLv_m/R)²R = B²L²v_m²/R = (0.4)²×625/4 = 25W ✓
例 5（含容电路）：光滑导轨间距 L=0.4m，B=0.5T，电容 C=10μF，导体棒以 v=2m/s 匀速向右切割磁感线，求电容器上的电荷量。
解：E = BLv = 0.5×0.4×2 = 0.4 V。
电容器两端电压 U = E = 0.4V。
Q = CU = 10×10⁻⁶×0.4 = 4×10⁻⁶ C = 4 μC。
