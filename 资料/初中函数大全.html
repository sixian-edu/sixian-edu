📈 初中数学 · 函数大全

一次函数 · 正比例函数 · 反比例函数 · 二次函数 · 图像与性质

🔹 一次函数 y = kx + b（k≠0）
k>0 上升 k<0 下降 b>0 交y轴正半轴 b<0 交y轴负半轴
一般式：y = kx + b  |  k = Δy/Δx  |  b = y轴截距
🔹 正比例函数 y = kx（k≠0）
k>0 一三象限 k<0 二四象限 正比例函数是一次函数的特例（b=0），图像必过原点。
y = kx  |  过原点  |  |k|越大越陡
🔹 k 的几何意义
k（斜率）决定直线的倾斜程度：|k|越大直线越陡；|k|越小直线越平缓。 k>0时，从左到右上升；k<0时，从左到右下降。
🔹 反比例函数 y = k/x（k≠0）
k>0 一三象限 k<0 二四象限 图像为双曲线，与坐标轴无限接近但永不相交。
y = k/x  |  x≠0  |  双曲线
🔹 反比例函数 |k| 的几何意义
在双曲线上任取一点，向坐标轴作垂线，围成的矩形面积 = |k|。 |k|越大 双曲线离原点越远。
🔹 二次函数 y = ax² + bx + c（a≠0）
a>0 开口向上 a<0 开口向下 |a|越大开口越小
一般式：y = ax² + bx + c  |  对称轴：x = -b/2a
🔹 二次函数顶点式
顶点式 y = a(x-h)² + k，顶点坐标为 (h, k)。 对称轴为直线 x = h。
顶点式：y = a(x-h)² + k  |  顶点 (h, k)
🔹 二次函数图像平移
平移规律：左加右减上加下减
y = x² → y = (x-2)²+1：向右平移2个单位，向上平移1个单位。
🔹 判别式 Δ = b² - 4ac
Δ > 0 抛物线与x轴有两个交点
Δ = 0 抛物线与x轴相切（一个交点）
Δ < 0 抛物线与x轴无交点
🔹 函数交点问题
一次函数与二次函数的交点：联立方程 ax² + bx + c = kx + b，解得的x即为交点横坐标。 联立求解 Δ判断交点个数
