压强 三种压强对比
固体压强
p = F/S
F是压力（水平面=G）
S是受力面积
✔ 单位：Pa
液体压强
p = ρgh
与深度/密度有关
与方向无关
✔ 越深压强越大
标准大气压 p₀ = 1.013×10⁵Pa ≈ 10⁵Pa ≈ 10m水柱产生的压强
浮力 浮力四种计算方法
方法公式适用场景
称重法F浮 = G - F示弹簧测力计挂物体浸入液体
阿基米德F浮 = G排 = ρ液gV排已知V排或ρ液
漂浮/悬浮F浮 = G物物体漂浮或悬浮时
压力差法F浮 = F向上 - F向下已知上下表面压力
⚠️ 浮沉条件：ρ物 < ρ液 → 漂浮 ｜ ρ物 = ρ液 → 悬浮 ｜ ρ物 > ρ液 → 下沉
简单机械 杠杆与滑轮速记
杠杆
F₁L₁ = F₂L₂
L₁>L₂ → 省力杠杆（撬棍）
L₁<L₂ → 费力杠杆（镊子）
L₁=L₂ → 等臂杠杆（天平）
滑轮
定滑轮：不省力，改变方向
动滑轮：省一半力，不改变方向
滑轮组：F = G/n（n为绳段数）
03电学急救——电路分析是命脉
电路识别 串并联电路特点对比
🔗 串联
I = I₁ = I₂
U = U₁ + U₂
R = R₁ + R₂
U₁/U₂ = R₁/R₂
🔀 并联
I = I₁ + I₂
U = U₁ = U₂
1/R = 1/R₁ + 1/R₂
I₁/I₂ = R₂/R₁
串联分压（电阻越大分压越多）｜并联分流（电阻越大电流越小）
欧姆定律 I=U/R 的应用
🔥 已知两个求第三个：
I = U/R  →  已知电压、电阻求电流
U = IR  →  已知电流、电阻求电压
R = U/I  →  已知电压、电流求电阻
例R₁=6Ω与R₂=4Ω串联接12V电源，求电流和R₂两端电压
解：R总=6+4=10Ω，I=12/10=1.2A
U₂=IR₂=1.2×4=4.8V
✔ 串联分压，电阻越大分压越多
电功率 额定功率与实际功率
P = UI（通用）→ P = I²R（串联）→ P = U²/R（并联）
灯泡亮度由实际功率决定！
例"220V 40W"灯泡接110V电源，实际功率是多少？
解：R=U²/P=220²/40=1210Ω。P实=U实²/R=110²/1210=10W
✔ 电压减半，功率降为¼！灯丝电阻不变是关键。
例标有"6V 3W"的小灯泡正常发光时电流和电阻是多少？
解：I=P/U=3/6=0.5A，R=U/I=6/0.5=12Ω
✔ 或者用R=U²/P=36/3=12Ω

陷阱1下列物体中，质量约为50g的是：A. 一个鸡蛋 B. 一本物理书 C. 一张课桌 D. 一支铅笔
陷阱：⚠️ 对常见物体质量缺乏概念。
✅ 选A。一个鸡蛋约50g。常见估算：人50-80kg、物理书200-300g、课桌10-20kg、铅笔5-10g。
✔ 估算题技巧：用熟悉的参照物对比（如一斤=500g）
陷阱2关于惯性，下列说法正确的是：A. 物体速度越大惯性越大 B. 运动的物体有惯性，静止的没有 C. 一切物体都有惯性 D. 只有不受力时才有惯性
陷阱：⚠️ 误以为惯性与速度有关。
✅ 选C。一切物体在任何状态下都有惯性，惯性大小只与质量有关，与速度、是否受力、是否运动无关。
✔ 质量越大惯性越大，越难改变运动状态。
陷阱3甲乙两块铁，甲的体积是乙的3倍，则它们的密度之比是：A. 1:1 B. 3:1 C. 1:3 D. 9:1
陷阱：⚠️ 看到体积比就直接选密度比。
✅ 选A。密度是物质的特性，同种材料密度相同，与体积和质量无关。
✔ 规律：同种物质密度相同，不同物质密度一般不同。
陷阱4一个物体放在水平桌面上，下列哪对力是平衡力：A. 物体重力和桌面对物体的支持力 B. 物体重力和物体对桌面的压力
陷阱：⚠️ 混淆平衡力与相互作用力。
✅ 选A。平衡力：作用在同一物体上、等大、反向、共线。B中重力和压力作用在不同物体上（重力在物体上，压力在桌面上），是相互作用力。
✔ 判断口诀：同体等大反向共线是平衡力，异体等大反向共线相互作用。
陷阱5将一木块放入水中，木块漂浮在水面；若将木块露出水面的部分切去，剩余部分将：A. 下沉 B. 悬浮 C. 上浮最终漂浮 D. 无法确定
陷阱：⚠️ 以为切掉后剩余部分会下沉。
✅ 选C。木块密度<水，切掉露出部分后，剩余部分重力减小，但F浮不变（V排不变），F浮>G物，所以会上浮，最终再次漂浮。
✔ 规律：漂浮物切去露出部分后，剩余部分仍会上浮直至漂浮。
陷阱6在如图电路中，闭合开关后发现灯泡不亮，电流表无示数，电压表有示数且接近电源电压，故障可能是：A. 灯丝断了 B. 开关短路 C. 电压表坏了 D. 电流表坏了
陷阱：⚠️ 电路故障分析。
✅ 选A。电流表无示数→电路断路；电压表有示数且接近电源电压→电压表与电源连通，说明灯泡处断路（灯丝断了）。
✔ 规律：断路的灯泡两端电压=电源电压（电压表测断路处）
