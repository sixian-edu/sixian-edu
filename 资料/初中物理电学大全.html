一、电荷与电流
⚡ 电荷
自然界只有 正电荷 和 负电荷 两种。
同种电荷相互排斥，异种电荷相互吸引 用验电器检验物体是否带电

摩擦起电：用摩擦的方法使物体带电。实质是 电子的转移。
丝绸摩擦玻璃棒 → 玻璃棒带正电（失去电子） 毛皮摩擦橡胶棒 → 橡胶棒带负电（得到电子）
摩擦起电：电子得失  |  带电体能吸引轻小物体
⚡ 电流
电荷的 定向移动 形成电流。
电流方向：正电荷定向移动的方向 外部电路：从正极 → 用电器 → 负极 金属导体中：自由电子定向移动，方向与电流相反

形成持续电流的条件：① 有电源 ② 电路闭合
电流方向：正极 → 负极（外部） |  电子方向：负极 → 正极
二、电路与电路图
🔌 电路组成与状态
电路由 电源、用电器、开关、导线 四部分组成。
电源：提供电能用电器：消耗电能开关：控制电路
三种状态：通路（电器工作）→断路（无电流）→短路（危险！）
🔄 串联与并联对比
比较项串联并联
电流路径只有一条多条（有分支）
各用电器关系相互影响（一个坏全部不工作）互不影响（各支路独立）
电流规律I = I₁ = I₂（处处相等）I = I₁ + I₂（干路=支路和）
电压规律U = U₁ + U₂（分压）U = U₁ = U₂（各支路电压相等）
电阻规律R = R₁ + R₂（越串越大）R = (R₁R₂)/(R₁+R₂)（越并越小）
实例节日小彩灯、开关与灯家庭用电器、路灯
串联：I₁=I₂，U=U₁+U₂，R=R₁+R₂  |  并联：I=I₁+I₂，U₁=U₂，R=(R₁R₂)/(R₁+R₂)
例题 1：由同种材料制成的两段导体，长度相同但横截面积不同，串联在同一电路中，通过谁的电流大？
解：串联电路电流处处相等，所以通过两段导体的电流 一样大。
易错点：很多人会认为粗的电阻小电流大，但那是针对"两端电压相同时"说的。串联时电流相等。
三、电流表·电压表·滑动变阻器
📊 电流表使用规则
1
串联 — 电流表必须 串联 在被测电路中
2
正进负出 — 电流从 正接线柱 流入，从 负接线柱 流出
3
不超量程 — 被测电流不能超过电流表量程（0~0.6A 或 0~3A）
4
不允许直接接电源 — 电流表不能直接接在电源两极
试触法：先大量程试触，偏转太小换小量程
📊 电压表使用规则
1
并联 — 电压表必须 并联 在被测电路两端
2
正进负出 — 电流从正接线柱流入，负接线柱流出
3
不超量程 — 被测电压不能超过电压表量程（0~3V 或 0~15V）
4
可以直接接电源 — 电压表可以直接接在电源两极（测电源电压）
⚠️ 电压表相当于"断路"
🔧 滑动变阻器
原理：改变接入电路中电阻丝的 长度 来改变电阻。
使用：一上一下串联接入闭合前调最大
口诀："下柱在哪看哪端，远离哪端电阻大"
四、欧姆定律
📐 I = U / R
导体中的电流与导体两端的电压成正比，与导体的电阻成反比。
I = U/RU = IRR = U/I

重要：导体的电阻是导体本身的性质，不随U、I变化（定值电阻）
R = U/I 是 计算式，不是决定式！
I = U/R  |  串联分压：U₁/U₂ = R₁/R₂  |  并联分流：I₁/I₂ = R₂/R₁
🔄 动态电路分析（中考重难点）
解题思路：滑片移动 → 接入电阻变化 → 总电阻变化 → 总电流变化 → 各支路变化
口诀：串反并同 与滑变串联的部分：变化相反 与滑变并联的部分：变化相同
🔍 电路故障分析
现象原因判断
灯不亮，A无示数断路V有示数→所测部分断路
灯不亮，A有示数短路V无示数→所测部分短路
A示数过大短路或总电阻小检查短路
💡口诀
"电流表有示数→无断路；电压表有示数→所测部分无短路"
例题 2：R₁=10Ω与R₂=20Ω串联在6V电源上，求：(1)电流I (2)R₁两端电压
解：R总=10+20=30Ω，I=U/R总=6/30=0.2A，U₁=IR₁=0.2×10=2V
串联分压：U₁:U₂=R₁:R₂=1:2，6V分成2V和4V。并联分流相反：I₁:I₂=R₂:R₁。
练习 1：R₁=5Ω与R₂=10Ω并联在6V电源上，求：(1)总电阻 (2)干路电流
(1)R总=R₁R₂/(R₁+R₂)=50/15≈3.33Ω (2)I=U/R=6/3.33≈1.8A
五、伏安法测电阻
🧪 实验原理与步骤
原理：R = U/I（用电压表测U，电流表测I，计算R）
步骤：①按电路图连接（断开开关，滑变调最大）②测3组U、I值③求R平均值
多次测量取平均：减小误差 连接时开关断开，滑变调到最大
六、电功率
💡 P = UI
电功率：电流在单位时间内做的功，表示电流做功的快慢。
P = UIP = I²RP = U²/R W = UIt

额定实际
电压额定电压（正常工作电压）实际电压（可能不同）
功率额定功率实际功率（决定亮度）
P = UI = I²R = U²/R  |  1度 = 1kW·h = 3.6×10⁶ J
🧪 测小灯泡电功率
与测电阻的区别：测电阻取平均为减小误差；测小灯泡功率不取平均（不同电压下P不同）
调节滑变使电压分别为 U额、1.2U额、0.8U额
U实=U额 → 正常发光
例题 3：标有"220V 100W"的灯泡，(1)正常工作时电流多大？(2)灯丝电阻多大？(3)接110V时实际功率多大？
解：(1)I=P/U=100/220≈0.455A；(2)R=U²/P=48400/100=484Ω；(3)P实=U实²/R=12100/484=25W
灯泡的亮暗由实际功率决定。电压减半→电流减半→功率变为原来的1/4！
练习 2：标有"6V 3W"的灯泡接在3V电源上，实际功率多少？（灯丝电阻不变）
R=U²/P=36/3=12Ω；P实=U实²/R=9/12=0.75W
七、焦耳定律与家庭电路
🔥 Q = I²Rt
电流通过导体产生的热量：与电流平方成正比、与电阻成正比、与通电时间成正比。
Q = I²Rt（普遍适用）
纯电阻电路：Q=W=UIt=U²t/R=I²Rt
非纯电阻（电动机）：Q=I²Rt < W=UIt（大部分电能→机械能）
焦耳定律：Q=I²Rt  |  电热器：电能→内能
🏠 家庭电路与安全用电
进户线：火线L、零线N，电压220V。地线PE防漏电。
进户线→电能表→总开关→保险装置→各用电器（并联）
开关接火线，左零右火上接地
保险丝：电阻率大、熔点低的铅锑合金不可用铜丝铁丝代替！
人体安全电压 ≤ 36V  |  不接触低压、不靠近高压
例题 4：电炉丝热得发红，导线却不怎么热，为什么？
解：电炉丝与导线串联，I相同。根据Q=I²Rt，电炉丝电阻远大于导线，相同时间内发热更多。
串联电路中，电热与电阻成正比（Q₁:Q₂=R₁:R₂）。电炉丝电阻大→发热多。
练习 3：一个电炉电阻50Ω，通过电流4A，通电10分钟产生多少热量？
Q=I²Rt=4²×50×(10×60)=16×50×600=4.8×10⁵J
八、电与磁
🧲 磁现象与电流磁效应
磁极：同名相斥、异名相吸磁感线从N→S（外部）
奥斯特实验：通电导体周围存在磁场——电流的磁效应（第一个连接电与磁）
安培定则：右手握螺线管，四指指电流方向，拇指指N极
奥斯特：通电导体周围有磁场  |  电磁铁：电流越大磁越强、匝数越多磁越强
⚡ 电动机与发电机对比
比较项电动机发电机
原理通电线圈在磁场中受力转动电磁感应（法拉第）
能量转化电能→机械能机械能→电能
关键部件换向器滑环/换向器
生活应用风扇、电动车水力/风力发电
"电生磁"→奥斯特、电磁铁 "磁生电"→电磁感应、发电机 "电受力"→电动机
例题 5：发电机和电动机分别根据什么原理制成？能量如何转化？
解：发电机：电磁感应（机械能→电能）；电动机：通电线圈在磁场中受力（电能→机械能）
易混点：发电机"先动后电"（需要外力转动才能发电）；电动机"先电后动"（通电才能转动）。
练习 4：感应电流方向与哪些因素有关？
与磁场方向和导体切割磁感线的方向有关（其中一个反向则电流反向，两者都反向则电流不变）。
✍️ 综合练习
练习 5：R₁:R₂=1:2串联在电路中，电压比和功率比分别是多少？
串联I相同，U₁:U₂=R₁:R₂=1:2；P₁:P₂=I²R₁:I²R₂=1:2
练习 6：两灯泡L₁"220V 40W"和L₂"220V 100W"，串联在220V电路中谁更亮？
R=U²/P→R₁=1210Ω>R₂=484Ω。串联I相同，P=I²R→P₁>P₂，L₁更亮。额定功率大的反而不亮！
练习 7：家庭电路中保险丝熔断可能的原因是什么？（写两个）
①短路（火线零线直接连通）②超负荷用电（总功率过大）③用电器漏电（空气开关跳闸）
