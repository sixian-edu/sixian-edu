一、电荷与电流
⚡ 电荷
自然界只有 正电荷 和 负电荷 两种。
同种电荷相互排斥，异种电荷相互吸引 用验电器检验物体是否带电

摩擦起电：用摩擦的方法使物体带电。实质是 电子的转移。
丝绸摩擦玻璃棒 → 玻璃棒带正电（失去电子） 毛皮摩擦橡胶棒 → 橡胶棒带负电（得到电子）
摩擦起电：电子得失  |  带电体能吸引轻小物体
⚡ 电流
电荷的 定向移动 形成电流。
电流方向：正电荷定向移动的方向 外部电路：从正极 → 用电器 → 负极 金属导体中：自由电子定向移动，方向与电流相反

形成持续电流的条件：① 有电源 ② 电路闭合
电流方向：正极 → 负极（外部） |  电子方向：负极 → 正极
二、电路与电路图
🔌 电路组成
电路由 电源、用电器、开关、导线 四部分组成。

电源：提供电能（干电池、蓄电池、发电机）
用电器：消耗电能（灯泡、电动机、电炉）
开关：控制电路通断
导线：连接电路

三种电路状态：
通路（闭合电路） — 处处连通，电器工作
断路（开路） — 某处断开，无电流
短路 — 电流不经过用电器直接回负极
⚠️ 电源短路会烧坏电源，绝对要避免！
🔄 串联与并联对比
比较项串联并联
电流路径只有一条多条（有分支）
各用电器关系相互影响（一个坏全部不工作）互不影响（各支路独立）
电流规律I = I₁ = I₂（处处相等）I = I₁ + I₂（干路=支路和）
电压规律U = U₁ + U₂（分压）U = U₁ = U₂（各支路电压相等）
电阻规律R = R₁ + R₂（越串越大）R = (R₁R₂)/(R₁+R₂)（越并越小）
实例节日小彩灯、开关与灯家庭用电器、路灯
串联：I₁=I₂，U=U₁+U₂，R=R₁+R₂  |  并联：I=I₁+I₂，U₁=U₂，R=(R₁R₂)/(R₁+R₂)
三、电流表·电压表·滑动变阻器
📊 电流表使用规则
1
串联 — 电流表必须 串联 在被测电路中
2
正进负出 — 电流从 正接线柱 流入，从 负接线柱 流出
3
不超量程 — 被测电流不能超过电流表量程（0~0.6A 或 0~3A）
4
不允许直接接电源 — 电流表不能直接接在电源两极

试触法： 先大量程"试触"，偏转太小换小量程，偏转太大换更大量程
📊 电压表使用规则
1
并联 — 电压表必须 并联 在被测电路两端
2
正进负出 — 电流从正接线柱流入，负接线柱流出
3
不超量程 — 被测电压不能超过电压表量程（0~3V 或 0~15V）
4
可以直接接电源 — 电压表可以直接接在电源两极（测电源电压）

⚠️ 电压表相当于"断路" — 如果电压表串联在电路中，相当于断路，用电器不工作
🔧 滑动变阻器
原理：通过改变接入电路中电阻丝的 长度 来改变电阻。
符号：用 ——┃—— 表示。

使用规则：
一上一下 — 必须各接一个上接线柱和一个下接线柱
串联接入 — 滑动变阻器要串联在电路中
闭合前调最大 — 闭合开关前，应将滑片调到阻值最大端
看下不看上 — 接入电路的有效电阻取决于下接线柱的位置

作用：保护电路 + 改变电路中的电流和电压
📌 滑动变阻器接法判断
下接线柱接 A（左端），滑片 P 向右移 → 接入电阻变 大 → 电流变 小
下接线柱接 B（右端），滑片 P 向右移 → 接入电阻变 小 → 电流变 大
口诀："一上一下连入串，下柱在哪看哪端，远离哪端电阻大"
四、欧姆定律
📐 I = U / R
导体中的电流与导体两端的电压成正比，与导体的电阻成反比。

I = U/R U = IR R = U/I

I-U 图像分析：
定值电阻：I-U图像为过原点直线（斜率=1/R） 小灯泡：I-U图像为曲线（灯丝电阻随温度升高而增大）

重要结论：导体的电阻是导体本身的性质，不随U、I变化（定值电阻）
R = U/I 是 计算式，不是决定式（不能说R与U成正比，与I成反比）
I = U/R  |  串联分压：U₁/U₂ = R₁/R₂  |  并联分流：I₁/I₂ = R₂/R₁
🔄 动态电路分析（中考重难点）
类型一：开关通断引起的动态变化
解题思路 开关闭合/断开 → 电路结构改变 → 总电阻变化 → 总电流变化 → 各支路变化

类型二：滑动变阻器引起的动态变化
解题思路："串反并同"口诀
滑片移动 → 接入电阻变化 → 总电阻变化 → 总电流变化
与滑变串联的部分：电流电压变化与滑变变化相反
与滑变并联的部分：电流电压变化与滑变变化相同
✅ 例题解析
例：R₁与滑动变阻器R₂串联，电压表测R₁两端电压。当滑片向右移动（R₂↑），电流表和电压表示数如何变化？

分析：
① R₂↑ → R总 = R₁+R₂↑ → 增大
② I = U/R总 → U不变，R总↑ → I↓（电流表示数变小）
③ U₁ = IR₁ → I↓，R₁不变 → U₁↓（电压表示数变小）

结论：电流表↓，电压表↓
🔍 电路故障分析
常见故障原因：

故障现象可能原因判断方法
灯泡不亮，电流表无示数断路电压表有示数→电压表所测部分断路
灯泡不亮，电流表有示数短路电压表无示数→电压表所测部分短路
一灯亮一灯不亮（串联）不亮的灯短路电流表仍有示数
电流表示数过大电路短路或总电阻过小检查是否有短路
💡 故障判断口诀
"电流表有示数→无断路；电压表有示数→所测部分无短路；电压无示数→所测部分短路或所测部分以外断路"
🧪 伏安法测电阻（实验）
原理：R = U/I（用电压表测U，电流表测I，计算R）
电路连接：电流表串联，电压表并联在待测电阻两端
滑动变阻器作用：① 保护电路 ② 改变电压多次测量

步骤：
① 按电路图连接实物（断开开关，滑变调最大）
② 闭合开关，调节滑变，记录 3 组U、I值
③ 分别计算 R = U/I，求平均值
④ 整理器材

多次测量取平均值的目的是：减小误差
注意：连接时开关要断开，滑变调到最大阻值处
五、电功率
💡 P = UI
电功：电流做的功。单位：J（焦耳）
电功率：电流在单位时间内做的功。表示电流做功的快慢。

P = W/t P = UI P = I²R = U²/R W = UIt

额定功率 vs 实际功率：
额定实际
电压额定电压（正常工作的电压）实际电压（可能不同）
功率额定功率（P额=U额I额）实际功率（P实=U实I实）
灯泡亮度—取决于 实际功率
P = UI = I²R = U²/R  |  1度 = 1kW·h = 3.6×10⁶ J
📌 典型计算
例：标有"220V 100W"的灯泡，正常工作时电流多大？灯丝电阻多大？
I = P/U = 100/220 ≈ 0.455A
R = U²/P = 220²/100 = 484Ω

若实际电压为110V：（灯丝电阻不变）
P实 = U实²/R = 110²/484 = 25W（实际功率变小，灯泡变暗）
🧪 测小灯泡电功率（实验）
原理：P = UI（用电压表测U，电流表测I，计算P）

与测电阻实验的区别： 测电阻取平均值为减小误差 测小灯泡功率不取平均值（不同电压下P不同）

实验要点：
调节滑变使电压分别为 U额、1.2U额、0.8U额
分别测量对应的电流，计算三种情况下的实际功率
U实 = U额 时 → P实 = P额（正常发光）
✅ 实验常见设问
Q1：小灯泡不亮、电流表有示数、电压表无示数 → 小灯泡短路
Q2：小灯泡不亮、电流表无示数、电压表有示数（接近电源电压）→ 小灯泡断路
Q3：移动滑片，发现小灯泡忽亮忽灭 → 接触不良
Q4：小灯泡亮度变化但不明显 → 滑动变阻器规格不合适（阻值过小或过大）
Q5：小灯泡的U-I图像为什么是曲线？→ 灯丝电阻随温度升高而增大
六、焦耳定律
🔥 Q = I²Rt
电流通过导体产生的热量，与电流的平方成正比，与导体的电阻成正比，与通电时间成正比。

Q = I²Rt（普遍适用）

纯电阻电路：电能全部转化为内能
Q = W = UIt = U²t/R = I²Rt

非纯电阻电路（如电动机）：电能大部分转化为机械能，少部分转化为内能
Q = I²Rt < W = UIt
焦耳定律：Q = I²Rt  |  电热器：电能→内能
📌 电热应用
电热器：电炉、电饭锅、电热毯、电熨斗（利用电流热效应）
防止电热危害：散热风扇、散热窗、电脑CPU散热片
电炉丝与连接导线：电炉丝热得发红而导线几乎不热，因为电炉丝电阻远大于导线
根据 Q = I²Rt，串联时I相同，R越大Q越大
七、家庭电路与安全用电
🏠 家庭电路
进户线：火线（L） 和 零线（N），电压 220V。
还有 地线（PE），防止漏电伤人。

家庭电路结构（从外到内）：
进户线 → 电能表（计量用电量）→ 总开关 → 保险装置 → 各用电器

元件作用连接方式
电能表测量消耗的电能串联在干路上
保险丝电流过大时熔断，切断电路串联在火线上
空气开关电流过大或漏电时跳闸串联在火线上
用电器消耗电能并联
开关控制用电器与用电器串联，接在火线上
左零右火上接地  |  开关接火线  |  插座要接地
⚠️ 安全用电
人体安全电压：不高于 36V
测电笔：氖管发光的为火线
三孔插座：左零(N)右火(L)上接地(PE)

安全用电原则：
不接触低压带电体 不靠近高压带电体 不用湿手触摸电器 不超负荷用电

保险丝选择：电阻率 大、熔点 低 的铅锑合金
不能用铜丝或铁丝代替保险丝！
八、电与磁
🧲 磁现象
磁极：同名磁极相斥，异名磁极相吸
磁感线：从N极出发到S极（闭合曲线），不交叉，疏密表示强弱
地磁场：地磁N极在地理南极附近，指南针N极指北

奥斯特实验（1820年）：
通电导体周围存在磁场 → 电流的磁效应
导线平行于小磁针放置 电流方向改变，磁场方向也改变
奥斯特实验：通电导体周围存在磁场  |  第一个连接电与磁
🧲 电磁铁与电磁继电器
电磁铁：插入铁芯的通电螺线管。
磁性强弱与电流大小 磁性强弱与线圈匝数 有无铁芯 电流方向改变极性也改变

安培定则（右手螺旋定则）：右手握住螺线管，四指指向电流方向，大拇指指向N极

电磁铁应用：电磁起重机、电铃、电磁继电器
电磁继电器：利用电磁铁控制工作电路通断的开关。低电压控制高电压，弱电流控制强电流
⚡ 电动机与发电机对比
比较项电动机（马达）发电机
原理通电线圈在磁场中受力转动电磁感应（法拉第，1831年）
能量转化电能 → 机械能机械能 → 电能
主要构造磁体 + 线圈 + 换向器磁体 + 线圈 + 滑环/换向器
产生条件通电（有电流）闭合电路 + 部分导体切割磁感线
生活中风扇、电动车、洗衣机水力发电、风力发电

交流发电机的特点：线圈转动一周，电流方向改变两次。
感应电流方向的影响因素：磁场方向、导体切割磁感线的方向
电动机：通电导体在磁场中受力（电→磁→力） |  发电机：电磁感应（磁→电）
📣 电磁现象综合
实验/现象发现者原理应用
奥斯特实验奥斯特电流的磁效应电磁铁
通电线圈受力—磁场对电流的作用电动机
电磁感应法拉第磁生电发电机
📌 易混淆概念区分
"电生磁" → 奥斯特实验、电磁铁（有电才磁）
"磁生电" → 电磁感应、发电机（有磁有动才生电）
"电受力" → 通电导体在磁场中受力、电动机（有电有力）

练习 1
两盏灯串联在电路中，其中一盏灯灭了，另一盏灯（ ）
A. 也灭   B. 更亮   C. 不变   D. 可能灭可能不灭
✅ 答案
A。串联电路电流只有一条路径，一处断开整个电路断开。
练习 2
两个电阻 R₁:R₂ = 1:2，串联在电路中，它们的电压比和功率比是（ ）
A. 1:2，1:2   B. 2:1，2:1   C. 1:2，2:1   D. 1:1，1:1
✅ 答案
A。串联时 I 相同，U₁:U₂ = R₁:R₂ = 1:2，P₁:P₂ = I²R₁:I²R₂ = R₁:R₂ = 1:2
练习 3
一个标有"6V 3W"的小灯泡，正常发光时的电阻是多少？电流是多少？
✅ 答案
I = P/U = 3/6 = 0.5A
R = U/I = 6/0.5 = 12Ω（或 R = U²/P = 36/3 = 12Ω）
练习 4
将"6V 3W"的灯泡接在 3V 的电源上，实际功率是多少？（灯丝电阻不变）
✅ 答案
R = U额²/P额 = 36/3 = 12Ω
P实 = U实²/R = 9/12 = 0.75W
练习 5
一个电炉的电阻为 50Ω，通过电流为 4A，通电 10 分钟产生多少热量？
✅ 答案
Q = I²Rt = 4² × 50 × (10×60) = 16 × 50 × 600 = 4.8×10⁵ J
练习 6
家庭电路中，保险丝熔断的原因可能是什么？（至少写出两个）
✅ 答案
① 短路 — 火线和零线直接连通
② 超负荷用电 — 总功率过大（I = P/U 过大）
③ 用电器漏电（空气开关带漏电保护功能时会跳闸）
练习 7
发电机是根据什么原理制成的？电动机是根据什么原理制成的？
✅ 答案
发电机：电磁感应（法拉第） — 机械能→电能
电动机：通电线圈在磁场中受力转动 — 电能→机械能
