01审题思维——读三遍法
核心原则：题读三遍，其义自见。中考物理丢分的第一大原因不是"不会"，而是没看清。

第一遍·扫读 （10秒）快速浏览题目，判断所属板块：力？热？电？光？声？实验？——确定用什么知识体系来思考。
第二遍·精读 （30秒）逐字阅读，用笔圈出所有已知量和关键词：
 • "匀速" "静止" "漂浮" "悬浮" "轻质" "光滑" "忽略摩擦" → 隐含条件
 • 数字 + 单位 → 已知数据
 • "最大" "最小" "至少" "恰好" → 临界条件
第三遍·确认 （10秒）看一眼问题到底问的是什么，不要答非所问。
考场口诀：一扫情境二圈量，三看问题别跑偏。单位换算要小心，隐含条件别放掉。
常见失误：①没看到"g取10N/kg"直接用了9.8 ②cm²没换算成m² ③"光滑"意味着不计摩擦 ④"漂浮"="浮力等于重力"
例1一个质量为 50kg 的人站在水平地面上，双脚与地面的接触面积为 400cm²。求人对地面的压强。（g=10N/kg）
三步审题演示：
① 扫读：力学题 → 求压强
② 精读（圈出）：m=50kg，S=400cm²，"站"→F=G，g=10N/kg
③ 确认：求压强p，用 p=F/S

解：F=G=mg=50×10=500N，S=400cm²=0.04m²
p=F/S=500/0.04=12500Pa
易错 400cm²=0.04m²（不是0.4m²！1m²=10⁴cm²）
例2一物体在50N的水平拉力作用下沿水平面做匀速直线运动，受到的摩擦力是多少？如果拉力增大到80N，摩擦力又是多少？
审题关键："匀速直线运动" → 受力平衡 → f=F拉
第一问：f=50N（二力平衡）
第二问（易错！）：摩擦力大小只与压力和接触面粗糙程度有关，与拉力无关。压力不变、接触面不变 → f不变=50N
提分点 很多同学第二问答80N就错了——滑动摩擦力与拉力无关！
练1一木块漂浮在水面上，排开水的体积为 200cm³，求木块受到的浮力和木块的质量。（ρ水=1.0×10³kg/m³，g=10N/kg）
审题："漂浮"→F浮=G，V排=200cm³
F浮=ρ水gV排=1.0×10³×10×200×10⁻⁶=2N
漂浮时F浮=G，m=G/g=2/10=0.2kg=200g
02模型识别——这道题在考什么
核心原则：做题先"定性"再"定量"——先判断属于什么模型，再套用对应方法。模型认准了，解题方向就对了。

例1将标有"6V 3W"和"6V 6W"的两盏灯串联在 6V 的电源上，哪盏灯更亮？
模型识别三步走：
① 看到"6V 3W""6V 6W" → 电功率模型 → 用 P=UI, P=I²R, P=U²/R
② 看到"串联" → 串联电路模型 → 电流相等，电压分配与电阻成正比
③ 灯泡亮度由实际功率决定（不是额定功率！）

解：R₁=U²/P₁=6²/3=12Ω，R₂=U²/P₂=6²/6=6Ω
串联电流 I=U/R总=6/(12+6)=1/3 A
P实₁=I²R₁=(1/3)²×12≈1.33W，P实₂=I²R₂=(1/3)²×6≈0.67W
P实₁ > P实₂ → "6V 3W"的灯更亮
易错 额定功率大的灯在串联时反而不亮——串联看电阻，电阻大的实际功率大！
例2一个木块放在水平桌面上，用弹簧测力计水平拉动木块做匀速直线运动，示数为2N。若在木块上加一个砝码，再用弹簧测力计水平拉动木块做匀速直线运动，示数会如何变化？
模型识别："匀速直线运动" → 受力平衡 → 拉力 = 摩擦力
"加砝码" → 压力增大 → 滑动摩擦力增大
→ 弹簧测力计示数变大
考点 滑动摩擦力大小只与压力和接触面粗糙程度有关。
练1一个重 5N 的物体放入水中后漂浮在水面上，它排开水的体积是多少？（g=10N/kg）
模型："漂浮" → 浮力模型 → F浮=G
F浮=5N，F浮=ρ水gV排 → V排=5/(1.0×10³×10)=5×10⁻⁴m³=500cm³
03公式定位——已知什么、求什么
核心原则：从不"蒙"公式，而是"倒着推"——从问题出发，需要什么量就找什么量。

倒推法三步走：
① 明确目标：题要你求什么？（压强？电流？浮力？热量？）
② 列出公式：求这个量需要知道哪些条件？
③ 查找已知：题目给了哪些？缺的能不能间接求出来？
单位检验 代入前检查：cm²→m²，g→kg，min→s，km/h→m/s
应试技巧：公式写原始形式（如 p=F/S），不要跳步，每代一个数字都带单位，既方便检查又方便拿步骤分。
公式定位速查表
根据所求量直接查找对应公式

重力G = mg
密度ρ = m / V
压强p = F / S
液体压强p = ρ g h
浮力F浮 = ρ g V排

欧姆定律I = U / R
电功率P = U I

热学

要求什么 → 找到对应板块 → 选公式 → 代数据
例1标有"220V 100W"的灯泡接在 110V 的电路中，求它的实际功率。
公式定位三步法：
① 目标：P实（实际功率）
② 备选公式：P=UI（缺I），P=I²R（缺I），P=U²/R（R不变，U已知）→ 先用P=U²/R
③ 缺什么找什么：需要R，R不变——用额定状态求R
R=U额²/P额=220²/100=484Ω，P实=U实²/R=110²/484=25W
规律 电压减半 → 功率变为原来的1/4
易错 不能用 P实=(U实/U额)×P额 算！功率与电压的平方成正比。
例2质量为2kg的某种物质，温度从20°C升高到40°C时吸收了 1.68×10⁵J 的热量，求这种物质的比热容。
定位：目标→比热容c；公式→Q=cmΔt → c=Q/(m·Δt)
c=1.68×10⁵/(2×20)=4200 J/(kg·°C) → 这是水的比热容！
练1某电热水壶功率1500W，正常工作10分钟消耗多少电能？合多少度电？
定位：目标→电功W；公式→W=Pt
P=1.5kW，t=1/6h → W=1.5×(1/6)=0.25kWh
或 P=1500W，t=600s → W=1500×600=9×10⁵J=0.25kWh ✓
04选择题技巧——不会也能拿分
核心原则：选择题不考过程只考结果，用技巧可以在不完全确定的情况下锁定答案。宁可花10秒排除，不要花10分钟硬算。

排除法 先看有没有"绝对化表述"（都、总是、一定、任何）——物理题里绝对化表述大多是错的。
量纲法 看结果的单位是否匹配。求压强→单位应该是Pa或N/m²；求功→单位应该是J。
极限法 把某个量推向极端（0或无穷大），判断选项是否仍然合理。
特殊值法 代入1、2、10等简单数值快速验证比例关系。
图像法 遇到抽象关系，随手画个草图就能看出来。
中考选择题平均只有2分钟一题，所以——①读完题先排除1-2个明显错的 ②用技巧在剩下里选 ③不确定的先标记，做完再回看。

例1关于惯性，下列说法正确的是：
A. 物体只有静止时才有惯性   B. 物体运动速度越大惯性越大
C. 物体受力越大惯性越大   D. 一切物体在任何情况下都有惯性
排除法：A：静止才有惯性？→ 运动的也有 → ✗   B：速度越大惯性越大？→ 惯性只与质量有关 → ✗
C：受力越大惯性越大？→ 惯性是固有属性 → ✗   D：正确 → ✓
答案 D
例2甲、乙质量之比3:2，吸收相同热量后升高的温度之比4:3，则比热容之比为：
A. 1:2   B. 2:1   C. 9:8   D. 8:9
特殊值法：设m甲=3,m乙=2,Δt甲=4,Δt乙=3,Q甲=Q乙=1
c甲:c乙=(1/12):(1/6)=1:2 → 答案 A
例3（较难）滑片P向右移动时，接入电阻变大。电流表和电压表示数如何变化？
A. A↑ V↑   B. A↓ V↓   C. A↓ V↑   D. A不变 V↑
排除法+极限法：滑片右移→电阻变大→电流变小→排除A、D。极限法：假设移到最右→R₂最大→分压最大→U₂变大 → 答案 C
练1下列做法中为了减小压强的是：A.刀刃磨薄 B.铁轨铺枕木上 C.针尖做很尖 D.压路机碾子很重
排除法：A、C受力面积小→压强大✗   D压力大→压强大✗
B枕木→增大受力面积→压强减小 ✓ 答案：B
05计算题策略——步骤就是分数
核心原则：中考物理是按步骤给分！写对公式+代入数据=保底60%分，即使算不出最终结果。

计算题标准五步法（每步对应分值）：
①写公式 写出原始公式，如 G=mg（2分）
②代数据 代入数值和单位，如 G=50×10（2分）
③算结果 计算出正确结果（1分）
④写单位 结果后加单位（0.5分）
⑤作答 写"答：……"（0.5分）
重要：即使不会做完整道题，也要把能写的步骤写上去！比如读题发现是力学题，先把G=mg写上去，题目给质量了就能拿2分。千万不要空着！

例1一辆质量1.5t的小汽车在水平公路上匀速行驶，受到的阻力是车重的0.05倍。求：（1）汽车的重力 （2）汽车的牵引力
满分答卷（6分）：
已知：m=1.5t=1500kg，g=10N/kg，f=0.05G，匀速行驶
（1）公式 G=mg =1500×10=15000N   ①写公式(2分) ②代数据(2分) ③结果+单位(0.5分)
（2）匀速→受力平衡→F=f=0.05G=0.05×15000=750N   写公式+代数据(1分) 结果+单位(0.5分)
答：重力15000N，牵引力750N。总分 6/6分
例2（半会不会怎么办）把2kg水从20°C加热到100°C，求吸收的热量。[c水=4.2×10³J/(kg·°C)]
如果你忘了公式——用单位检验：
✗ Q=cm（漏了Δt）→ 单位J? ✗   ✗ Q=mΔt（漏了c）→ 数值不对 ✗
✓ Q=cmΔt=4.2×10³×2×(100-20)=6.72×10⁵J ✓
技巧 不确定公式时，检查等号两边单位是否一致！
练1一个重50N的物体在水平地面上受到20N水平拉力做匀速运动，求：（1）摩擦力 （2）若拉力减为10N，摩擦力是多少？此时物体做什么运动？
（1）匀速→f=F拉=20N
（2）摩擦力与拉力无关→f不变=20N。此时F拉=10N < f → 物体减速运动
提分点 第二问不能想当然"减速→摩擦力变小"，要先判断摩擦力是否改变！
06实验探究题套路——万变不离其宗
核心原则：实验题看似灵活，其实出题角度高度固定。把五种题型都练一遍，实验题就是送分题。

五大题型全攻略：
题型常见问法答题模板
实验原理题"本实验的原理是什么？"写出核心公式（如R=U/I、ρ=m/V）
步骤设计题"连接电路时开关应……"①断开开关 ②调到最大阻值 ③检查无误后闭合
数据分析题"根据数据可得出结论……"控制变量法：在××相同时，××越××，××越××
误差分析题"测量结果偏大/偏小原因"写出操作中哪个量偏大/偏小，代入公式分析
结论表述题"写出实验结论"用"成正比/成反比/有关/无关"表述，不说具体数值
实验题"三个一定"：①一定先写原理（公式） ②一定注意控制变量 ③一定用物理语言表述结论，不要口语化。

例1在"测量小灯泡额定功率"实验中，灯泡标有"2.5V"：
（1）实验原理？（2）连接电路时开关和滑动变阻器分别应处于什么状态？（3）闭合开关，灯不亮，电流表有示数，电压表无示数，故障原因可能是？
（1）原理：P=UI。电压表测U，电流表测I，相乘得P。
（2）规范：①开关断开 ②滑动变阻器调到阻值最大处
（3）故障：电流表有示数→电路通；电压表无示数→灯泡两端无电压→最可能灯泡短路
套路 电路故障题：电流表有示数→短路故障；电流表无示数→断路故障
例2用天平和量筒测金属块密度。测得质量78g，量筒中水30mL，放入金属块后40mL。若先测体积再称质量，密度偏大还是偏小？为什么？
（1）密度：V=10cm³，ρ=78/10=7.8g/cm³
（2）误差分析：先测体积→金属块沾水→质量偏大→ρ=m/V→分子偏大→密度偏大
模板 误差分析三步：①哪个操作变了 → ②哪个测量值偏大/偏小 → ③代入公式分析结果
练1在"研究滑动摩擦力大小与哪些因素有关"的实验中：（1）为什么用测力计水平匀速拉动木块？（2）加砝码后拉力变大，说明什么？
（1）匀速→受力平衡→拉力=摩擦力，通过测力计示数间接知道摩擦力。
（2）接触面粗糙程度相同时，压力越大，滑动摩擦力越大（控制变量法）。
练2伏安法测电阻实验中，若电压表完好但电流表损坏，能否用一个定值电阻R₀完成测量？简述方法。
特殊方法：可以——将R₀与Rx串联，用电压表分别测U₀和Ux。串联电流相等：I=U₀/R₀=Ux/Rx → Rx=(Ux/U₀)R₀。
套路 缺电流表→串联用电压表；缺电压表→并联用电流表。
07作图题与填空题——细节定成败
核心原则：作图题和填空题看似简单，但阅卷非常严格——差一点都不给分。

作图题必杀 checklist：
力的示意图 ①找受力物体 ②画在重心 ③标字母和大小（G、F、f、N）④线段长短表示力的大小
光路图 ①法线用虚线 ②光线带箭头 ③反射角=入射角 ④折射：空气→水中角变小
电路图 ①元件符号规范 ②导线不交叉 ③电流从正极出发 ④开关处于断开状态
杠杆 ①支点标O ②力的作用线用虚线延长 ③力臂用大括号标出 ④标L₁、L₂
考场口诀：作图题、尺规画，虚线实线看分明，箭头标号不能少，擦干净了再画下一道。
常见扣分点：①力的示意图没标大小 ②光路图没画箭头 ③电路图导线交叉 ④力臂没标直角符号 ⑤用铅笔画的太轻扫描不出来
例1一个重5N的物体静止在水平桌面上，画出它对桌面的压力示意图。
规范作图：
① 压力的作用点画在接触面上（不是画在物体重心！）
② 方向：垂直于桌面向下
③ 标字母 F 和大小 F=5N
④ 箭头清晰，线段长度适当
易错 "压力示意图"和"重力示意图"的区别——压力作用点在接触面，重力在重心！
例2（填空题）光从空气斜射入水中时，折射角________入射角（选填"大于""小于""等于"）；当入射角增大时，折射角________（选填"增大""减小""不变"）。
填空题技巧：看清填空位置前后的文字，确定用词是否准确。
① 光从空气→水中：折射角小于入射角（空气→水，速度变慢，偏向法线）
② 入射角增大→折射角也增大（但不一定是正比）
套路 填空题填"增大""减小"时，注意是否需要加"逐渐"或"先……后……"。
练1（填空）家庭电路中，控制电灯的开关应接在________线上；三孔插座的第三个孔（最上面的孔）接________线。
①开关接火线（断开开关后灯不带电，安全）②三孔插座上孔接地线（左零右火上接地）
08考场实战——时间分配与检查策略
核心原则：中考物理不仅是知识考试，也是"时间管理"考试。很多同学不是不会做，是时间不够或节奏乱了。

中考物理考试时间分配（以60分钟为例）：
题型题量建议用时策略
选择题10-12题15分钟每题不超过2分钟，不会的先跳过
填空题6-8题10分钟注意单位和用词准确
作图题2-3题5分钟尺规作图，规范清晰
实验探究题2-3题12分钟原理→步骤→数据→误差→结论
计算题2-3题13分钟写公式→代数据→结果，难的写前半部分
检查—5分钟查漏补缺、核对单位、确认填涂
三级做题策略：
第一遍（30分钟）：做所有会做的题，确保全对。
第二遍（20分钟）：做不确定的题，把能写的步骤都写上去。
第三遍（5分钟）：做完全不会的题，选择题用排除法蒙，计算题写公式。
最后5分钟：检查答题卡填涂、单位、姓名考号。
血的教训：千万不要在一道题上死磕超过5分钟！一道选择题耽误了，后面三道大题的时间就没了。
考场模拟考试还剩最后10分钟，你还有一道计算题（6分）和一道实验题（5分）没做。有思路但不确定对不对，怎么分配时间？
最优策略：
① 先花1分钟审题——判断哪道题更容易拿分
② 如果计算题思路清晰：先写公式和代数据（保底4分），再去对付实验题
③ 如果实验题更简单：先快速写完实验题（写对原理和结论就有3-4分），剩下的时间回来补计算题的步骤
核心 最后10分钟的目标是"多道题都拿部分分"而不是"一道题拿全分"。
思维练习你在考试中发现某道选择题选项A"一定"、B"可能"、C"不可能"、D"总是"，哪几个选项出问题的可能性最大？为什么？
快速判断：A"一定"和D"总是"有问题可能性最大！物理题中，绝对化表述（都、总是、一定、任何、所有）往往是错的，因为物理规律大多有条件限制。
B"可能"最可能是正确答案（温和表述，留有余地）。
09出题人陷阱——专治各种想当然
核心原则：中考物理出题人最擅长“挖坑”——看着简单，一做就错。这些坑都是有规律的，提前踩一遍，考场上就不会再掉进去。

常见陷阱类型
① 单位陷阱：故意给非标准单位（cm、g、min），不换算就直接代公式→全错。
② 隐含条件忽略：“光滑”“轻质”“匀速”“漂浮”“忽略温度对电阻的影响”——这些不是白写的！
③ 概念混淆：把压力和重力混为一谈、额定功率当成实际功率、质量重量分不清。
④ 定势思维：平时做的题都是这样，考试换个问法就不认识了。
⑤ 多余条件干扰：故意给一个用不上的条件，看你敢不敢忽略它。
防坑口诀：先看单位再看条件，概念不清一定完蛋；出题人比你狡，多留个眼心准没亏。
陷阱1一个质量为50g的鸡蛋，受到的重力是多少？（g=10N/kg）
大部分人的第一反应：50×10=500N ✗
单位坑 g和kg！质量单位是g，重力公式G=mg中m的单位必须是kg！
正确：m=50g=0.05kg，G=mg=0.05×10=0.5N
错误答案500N：差了1000倍，一个鸡蛋500N相当于50kg的物体——想想就知道不可能！
防坑 看到质量先看单位！g→kg要÷1000，t→kg要×1000。
陷阱2用10N的水平推力推一个重100N的箱子，箱子没动。此时箱子受到的摩擦力是多少？
概念坑 静摩擦力和重力无关！很多同学看到100N就条件反射去算摩擦力。
"没动"→静止→受力平衡→水平方向：推力=静摩擦力
→ f = F推 = 10N（和100N的物重半毛钱关系都没有！）
防坑 受力分析要分开看：水平方向和竖直方向互不干扰。
陷阱3一物体在20N的水平拉力下沿水平面匀速运动，若将拉力增大到30N，此时物体受到的摩擦力是多少？
条件变化坑 滑动摩擦力与拉力无关！只要压力和接触面不变，滑动摩擦力就不变。
匀速时：f=F拉=20N
拉力增到30N时：压力不变、接触面不变 → f不变，仍为20N
此时物体受力不平衡，将做加速运动
防坑 记死：滑动摩擦力的大小由压力和粗糙程度决定，跟拉力大小、速度大小都没有关系！
陷阱4甲、乙两个实心球，甲的密度是乙的2倍，乙的质量是甲的4倍，则甲的体积是乙的：
A. 1/8   B. 1/2   C. 2倍   D. 8倍
比例坑 很多人被“甲的是乙的”“乙的是甲的”绕晕。
设值法：设ρ甲=2, ρ乙=1；m乙=4, m甲=1
V甲=m甲/ρ甲=1/2=0.5；V乙=m乙/ρ乙=4/1=4
V甲/V乙=0.5/4=1/8 → 答案A
防坑 比例题谁绕谁设值，设成简单整数秒出答案。
陷阱5将一铁块从水中提出来，在铁块逐渐露出水面的过程中，铁块受到的浮力如何变化？
定势思维坑 很多人背过“浮力与深度无关”就套用到这里。
但“与深度无关”的前提是V排不变（完全浸没时）！
铁块露出水面 → V排减小 → F浮=ρgV排 → 浮力逐渐变小
防坑 浮力看V排！ V排不变→浮力不变；V排变→浮力变。不要死记“浮力与深度无关”。
陷阱6一个标有“6V 3W”的灯泡，如果接在3V的电源上，它的实际功率是多少？
公式坑 功率与电压是平方关系，不是正比关系！
正确：先求电阻 R=U²/P=6²/3=12Ω
P实=U实²/R=3²/12=0.75W
电压减半→功率变为1/4，不是1/2！
防坑 用 P=U²/R 这个形式时，R不变，P与U²成正比。
陷阱7当物距为20cm时在光屏上得到倒立放大的实像，则癏距可能是：
A. 5cm   B. 10cm   C. 15cm   D. 20cm
推理坑 “倒立放大实像” → f < u < 2f
已知u=20cm，所以：f < 20 < 2f → 10 < f < 20
四个选项中，只有C. 15cm符合条件
防坑 记牢：一焦分虚实，二焦分大小；物近像远像变大，物远像近像变小。
陷阱8一个人用50N的力提着重30N的物体在水平地面上匀速走了5m，他对物体做了多少功？
做功条件坑 做功的两个条件：有力、在力的方向上有距离。
提力方向是竖直向上，走的距离是水平方向。
提力方向与运动方向垂直 → 提力方向上没有移动距离 → 不做功！
正确答案：0J
防坑 看到“做功”克问自己：方向呢？力跟距离方向垂直→不做功。
陷阱9将“220V 40W”和“220V 100W”并联接入220V电路中，哪个更亮？
串并联混淆坑 前面讲过串联时电阻大的更亮，但现在是并联！
并联时各支路电压相等（都是220V）→ 都正常发光 → 100W的更亮（额定功率越大越亮）
串联看电阻（电阻大的亮），并联看功率（功率大的亮），完全不同！
防坑 拿到题先看是串联还是并联！两个结论完全不同，千万别混。
陷阱10两个完全相同的烧杯，分别装有质量相等的水和酒精。将同一个木块放入水中漂浮，放入酒精中沉底。木块在两种液体中受到的浮力大小关系是？
A. 水中大   B. 酒精中大   C. 一样大   D. 无法判断
综合坑 看到“同一个木块”→质量相同
水中：漂浮 → F浮水 = G木
酒精中：沉底 → F浮酒 < G木
→ F浮水 > F浮酒 → A. 水中大
防坑 浮力不只看液体密度，更要看物体在液体中的状态。漂浮时F浮=G，这是个超好用的等量关系！
