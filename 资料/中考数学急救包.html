分式 分式有条件限制
🔥 中考最爱考这个：分式有意义的条件
分式 A/B 有意义的条件：分母 ≠ 0！
分式 A/B 值为0的条件：分子 = 0 且 分母 ≠ 0（两个条件缺一不可！）
例分式 (x²-4)/(x-2) 的值为0时，x的值为：A. ±2 B. 2 C. -2 D. 0
解：分子=0 → x²-4=0 → x=±2。还需分母≠0 → x-2≠0 → x≠2。
所以只能取x=-2，选C。
✔ 易错：忘记验分母！x=2时分母为0分式无意义。

大数：n=位数-1。如 23000 = 2.3×10⁴
小数：n=负的连续0的个数。如 0.00056 = 5.6×10⁻⁴
易错：万=10⁴，亿=10⁸，10万=10⁵

04方程与不等式——解题基本功
一元二次 一元二次方程 ax²+bx+c=0（a≠0）
三种解法
① 因式分解：x²-5x+6=0 → (x-2)(x-3)=0 → x=2或3
② 公式法：x = [-b ± √(b²-4ac)] / 2a
③ 配方法：x²-6x+8=0 → (x-3)²=1 → x=4或2
判别式 Δ = b²-4ac
Δ > 0 → 两个不等实根
Δ = 0 → 两个相等实根
Δ < 0 → 无实根
韦达定理：x₁+x₂ = -b/a，x₁x₂ = c/a（知道一根就可求另一根！）
不等式 不等式性质（最易错！）
🔥 不等号方向改变——唯一触发条件：
两边同时乘以或除以一个负数，不等号方向必须反转！
例解不等式：-2x + 4 > 6
解：-2x + 4 > 6 → -2x > 2 → x < -1
✔ 除以-2时不等号反转！> 变成 <

二次函数 二次函数顶点与最值
三种表达式
① 一般式：y=ax²+bx+c
② 顶点式：y=a(x-h)²+k → 顶点(h,k)
③ 交点式：y=a(x-x₁)(x-x₂)
顶点公式
h = -b/(2a)（对称轴）
k = (4ac-b²)/(4a)（最值）
a>0有最小值，a<0有最大值
平移口诀：左加右减（x），上加下减（y）
例抛物线 y = x² - 4x + 3 的顶点坐标和对称轴是？
解：a=1, b=-4, c=3
对称轴 x = -b/2a = 4/2 = 2
顶点 y = 2²-4×2+3 = 4-8+3 = -1
顶点坐标 (2, -1)
✔ 也可用配方法：y=(x-2)²-1，直接看出顶点(2,-1)

陷阱1√16 的平方根是：A. 4 B. ±4 C. 2 D. ±2
陷阱：⚠️ 分不清"√16"和"√16的平方根"。
√16 = 4（算术平方根），4的平方根 = ±2
✅ 选D。
✔ 规律：先算出√16的值再求平方根。负数没有平方根！
陷阱2若关于x的一元二次方程 kx² - 6x + 9 = 0 有两个不相等的实数根，则k的取值范围：A. k<1 B. k≠0 C. k<1且k≠0 D. k≤1且k≠0
陷阱：⚠️ 只考虑Δ却忘了二次项系数≠0。
Δ = 36-36k > 0 → k < 1。且二次项系数 k ≠ 0。
✅ 选C。
✔ 注意：凡是"一元二次"方程，隐含a≠0条件！
陷阱3等腰三角形一边等于5，另一边等于8，则它的周长是：A. 18 B. 21 C. 18或21 D. 13
陷阱：⚠️ 没分类讨论哪条是腰。
① 腰=5，底=8 → 周长=5+5+8=18
② 腰=8，底=5 → 周长=8+8+5=21
✅ 选C。
✔ 等腰三角形必分类：谁是腰谁是底，还要验证两边之和大于第三边！
陷阱4下列运算正确的是：A. √2 + √3 = √5 B. 2√3 - √3 = 2 C. √9 = ±3 D. √8 = 2√2
陷阱：⚠️ 根式运算法则混淆。
A错：√2+√3不能合并，√5≠√2+√3
B错：2√3-√3 = √3，不是2
C错：√9 = 3（算术平方根只取非负），±3是9的平方根
✅ 选D。√8 = 2√2
✔ 根式运算：同类根式才能合并，算术平方根只有一个非负值
陷阱5一次函数y=kx+b的图像经过第一、二、四象限，则：A. k>0,b>0 B. k>0,b<0 C. k<0,b>0 D. k<0,b<0
陷阱：⚠️ 各象限对应的k、b符号记反。
一二四象限 → 图像从左向右下降（k<0），与y轴正半轴相交（b>0）
✅ 选C。
✔ 口诀：k>0过一三，k<0过二四；b>0交正半轴，b<0交负半轴
陷阱6已知|a| = 3, |b| = 5，且a·b < 0，求a+b的值：A. 8 B. -8 C. 2 D. ±2
陷阱：⚠️ 忽略了负数的可能性。
|a|=3 → a=±3；|b|=5 → b=±5
a·b<0 → a,b异号
a+b = 3+(-5) = -2 或 (-3)+5 = 2
✅ 选D。
✔ 绝对值题：看到绝对值就要想到正负两种情况，再看条件排除。
