初三物理知识点大全

中考总复习 · 系统梳理

一、机械运动

长度与时间测量：长度的基本单位是米（m），常用单位有千米（km）、分米（dm）、厘米（cm）、毫米（mm）、微米（μm）、纳米（nm）。刻度尺读数时需估读到分度值的下一位。时间的国际单位是秒（s），常用单位有分钟（min）、小时（h）。

误差：测量值与真实值之间的差异。误差不可消除，只能减小——多次测量取平均值、选用精密仪器、改进测量方法。

参照物与运动：物体位置随时间的变化叫机械运动。判断运动需选参照物，参照物假定不动。运动和静止是相对的。速度是描述物体运动快慢的物理量，v = s / t基本单位 m/s，常用 km/h，1 m/s = 3.6 km/h。匀速直线运动：速度不变，路线为直线。

二、声现象

声音的产生与传播：声音由物体振动产生，振动停止发声停止。声音通过介质（固体、液体、气体）传播，真空不能传声。15℃空气中声速 340 m/s。声速固体＞液体＞气体。

声音的特性：音调（频率决定，频率越高音调越高）、响度（振幅决定，振幅越大响度越大，还与距离有关）、音色（发声体本身决定，用于区分不同声源）。

超声波与次声波：频率高于 20000 Hz 为超声波（B 超、声呐、倒车雷达），低于 20 Hz 为次声波（地震、海啸预警）。噪声的控制：在声源处减弱、在传播途中减弱、在人耳处减弱。

三、光现象

光的直线传播：光在同种均匀介质中沿直线传播。实例：影子、日食月食、小孔成像。光速 c = 3×10⁸ m/s（真空中），水中约为真空的 3/4，玻璃中约为 2/3。

光的反射：反射光线、入射光线和法线在同一平面内（三线共面），反射光线和入射光线分居法线两侧（两线分居），反射角等于入射角（两角相等）。镜面反射与漫反射都遵循反射定律。平面镜成像特点：等大、等距、正立、虚像。

光的折射：光从一种介质斜射入另一种介质时传播方向发生偏折。折射角随入射角增大而增大。光从空气射入水中时折射角小于入射角；从水中射入空气时折射角大于入射角。

光的色散：白光通过三棱镜分解为红橙黄绿蓝靛紫七色光。光的三原色：红、绿、蓝。红外线用于遥控、夜视；紫外线用于验钞、灭菌。

四、透镜及其应用

凸透镜与凹透镜：凸透镜对光有会聚作用，凹透镜对光有发散作用。主光轴、光心（O）、焦点（F）、焦距（f）。

凸透镜成像规律（u = 物距，v = 像距）：

u > 2f
f < v < 2f，倒立缩小的实像——照相机
u = 2f
v = 2f，倒立等大的实像——测焦距
f < u < 2f
v > 2f，倒立放大的实像——投影仪
u = f
不成像——平行光源
u < f
正立放大的虚像——放大镜

实像倒立、虚像正立。眼睛成像相当于凸透镜，近视眼配凹透镜矫正，远视眼配凸透镜矫正。显微镜由目镜和物镜组成（均凸透镜），望远镜亦然。

五、质量与密度

质量：物体所含物质的多少，不随形状、状态、位置、温度而变。单位：kg、g、mg、t。天平使用时"左物右码"。

密度：单位体积内某种物质的质量。ρ = m / V。单位 kg/m³、g/cm³（1 g/cm³ = 1000 kg/m³）。水的密度 1.0×10³ kg/m³。密度是物质的一种特性，与质量和体积无关。测密度：用天平测质量、量筒（排水法）测体积。

六、力

力的作用效果：力可以改变物体的形状（形变），也可以改变物体的运动状态（速度大小或方向变化）。力的三要素：大小、方向、作用点。力的作用是相互的。

弹力：物体发生弹性形变后对使其发生形变的物体产生的力。弹簧测力计的原理：在弹性限度内，弹簧的伸长量与受到的拉力成正比。

重力：地球附近的物体由于地球吸引而受到的力。G = mg（g = 9.8 N/kg）。重力的方向竖直向下，作用点在重心。

摩擦力：两个接触面粗糙的物体之间有相对运动（或相对运动趋势）时产生的阻碍相对运动的力。滑动摩擦力大小与压力大小和接触面粗糙程度有关。增大摩擦：增大压力、增大接触面粗糙程度。减小摩擦：减小压力、减小粗糙程度、加润滑油、变滑动为滚动。

七、运动和力

牛顿第一定律（惯性定律）：一切物体在没有受到力的作用时，总保持静止状态或匀速直线运动状态。牛顿第一定律是通过实验加推理得出的。

惯性：物体保持原来运动状态不变的性质。一切物体都有惯性，惯性大小只与质量有关。利用惯性（跳远、锤头）和防范惯性（安全带、保持车距）。

二力平衡：作用在同一物体上的两个力，大小相等、方向相反、作用在同一直线上。物体受平衡力时保持静止或匀速直线运动。平衡力与相互作用力的区别：平衡力作用在同一物体上，相互作用力作用在不同物体上。

八、压强

压力与压强：垂直作用在物体表面上的力叫压力。压强表示压力的作用效果，p = F / S。单位 Pa（N/m²）。增大压强：增大压力、减小受力面积。减小压强反之。

液体压强：液体内部朝各个方向都有压强。同一深度各方向压强相等。深度越大压强越大。p = ρgh。连通器原理：同种液体不流动时各液面相平（茶壶、船闸、锅炉水位计）。

大气压强：大气对浸在它里面的物体产生的压强。标准大气压 p₀ = 1.013×10⁵ Pa ≈ 760 mmHg。马德堡半球实验证明大气压存在，托里拆利实验测出大气压值。大气压随海拔升高而减小。沸点随气压降低而降低。

流体压强与流速的关系：流速越大的位置压强越小（飞机机翼、火车站安全线）。

九、浮力

浮力：浸在液体（或气体）中的物体受到的竖直向上的托力。浮力方向竖直向上。浮力产生的原因：上下表面的压力差。称重法测浮力：F浮 = G - F示。

阿基米德原理：浸在液体中的物体受到向上的浮力，浮力大小等于物体排开的液体所受的重力。F浮 = G排 = ρ液 g V排。浮力大小只与液体密度和排开液体体积有关，与深度无关。

物体的浮沉条件：F浮＞G 上浮，F浮 = G 悬浮，F浮＜G 下沉。ρ液＞ρ物 上浮，ρ液 = ρ物 悬浮，ρ液＜ρ物 下沉。轮船（空心增大V排）、潜水艇（改变自重）、气球飞艇（改变浮力）、密度计（漂浮，示数下大上小）。

十、功和机械能

功：力与物体在力的方向上移动距离的乘积。W = F s。单位 J。做功的两个必要条件：有力作用在物体上，物体在力的方向上移动了距离。不做功的三种情况：有力无距、有距无力、力与距离垂直。

功率：做功的快慢。P = W / t，单位 W（瓦特）。

机械能：动能 + 势能。动能大小与质量和速度有关（质量越大、速度越大，动能越大）。重力势能与质量和高度有关。弹性势能与弹性形变程度有关。机械能守恒：只有动能和势能相互转化时，机械能总量不变。

十一、简单机械

杠杆：在力的作用下绕固定点转动的硬棒。五要素：支点（O）、动力（F₁）、阻力（F₂）、动力臂（l₁）、阻力臂（l₂）。杠杆平衡条件（杠杆原理）：F₁ l₁ = F₂ l₂。省力杠杆（l₁＞l₂，省力费距离）、费力杠杆（l₁＜l₂，费力省距离）、等臂杠杆。

滑轮：定滑轮（不省力，改变方向）、动滑轮（省一半力，不改变方向）。滑轮组：F = G总 / n（n 为承担重物的绳子段数），s = n h。

机械效率：有用功与总功的比值。η = W有 / W总 × 100%。提高机械效率的方法：减小额外功（减轻机械自重、加润滑油）。

十二、内能

分子热运动：一切物质的分子都在永不停息地做无规则运动。扩散现象（分子运动的结果）。温度越高，分子热运动越剧烈。

内能：物体内部所有分子热运动的动能和分子势能的总和。一切物体都有内能。改变内能的方式：做功（实质是能量的转化）和热传递（实质是能量的转移）。

比热容：单位质量的某种物质温度升高（或降低）1℃所吸收（或放出）的热量。c = Q / (mΔt)，单位 J/(kg·℃)。水的比热容最大 4.2×10³ J/(kg·℃)。Q吸 = cm(t - t₀)，Q放 = cm(t₀ - t)。

热值：燃料完全燃烧放出的热量与其质量之比。q = Q / m，单位 J/kg。Q放 = m q。热机四冲程：吸气、压缩、做功、排气，其中做功冲程将内能转化为机械能。

十三、电流和电路

电荷：摩擦起电的本质是电子的转移。正电荷（丝绸摩擦玻璃棒）、负电荷（毛皮摩擦橡胶棒）。同种电荷相互排斥，异种电荷相互吸引。验电器原理：同种电荷相互排斥。

电流与电路：电荷的定向移动形成电流。把正电荷定向移动的方向规定为电流方向。电路由电源、用电器、开关、导线组成。三种状态：通路、断路、短路。串联（电流只有一条路径，各用电器相互影响）和并联（电流有多条路径，各用电器独立工作）。

电流：I = Q / t，单位 A（安培）。电流表串联使用，正进负出。串并联电路电流特点：串联 I = I₁ = I₂ = ...；并联 I = I₁ + I₂ + ...。

十四、电压和电阻

电压：使电荷定向移动形成电流的原因。单位 V（伏特）。常见电压：一节干电池 1.5 V，家庭电路 220 V，人体安全电压不高于 36 V。电压表并联使用，正进负出。串并联电压特点：串联 U = U₁ + U₂ + ...；并联 U = U₁ = U₂ = ...。

电阻：导体对电流的阻碍作用。R = U / I，单位 Ω（欧姆）。电阻是导体本身的性质，与材料、长度（成正比）、横截面积（成反比）和温度有关。金属电阻随温度升高而增大（超导：电阻为零）。

滑动变阻器：通过改变接入电路的电阻丝长度来改变电阻。原理：R = ρL / S。使用：一上一下串联接入电路，闭合开关前滑片移至阻值最大处。

十五、欧姆定律

欧姆定律：导体中的电流，跟导体两端的电压成正比，跟导体的电阻成反比。I = U / R。U = I R，R = U / I。注意：R 不随 U、I 改变。

串并联电路电阻规律：串联 R总 = R₁ + R₂ + ...（越串越大）；并联 1/R总 = 1/R₁ + 1/R₂ + ...（越并越小）。串联电阻分压（电压与电阻成正比），并联电阻分流（电流与电阻成反比）。

伏安法测电阻：原理 R = U / I，用电压表测 U、电流表测 I。多次测量取平均值减小误差。

十六、电功率

电能与电功：电流做功的过程就是电能转化为其他形式能的过程。W = U I t，单位 J（焦耳），常用 kW·h（度），1 kW·h = 3.6×10⁶ J。电能表（电度表）计量电能。

电功率：电流做功的快慢。P = W / t = U I。单位 W（瓦特）。由 P = U² / R 和 P = I² R 推导得出。用电器额定功率与实际功率的区别：额定电压下功率为额定功率。

焦耳定律：电流通过导体产生的热量跟电流的平方成正比，跟导体的电阻成正比，跟通电时间成正比。Q = I² R t。纯电阻电路 Q = W，非纯电阻电路 Q＜W。

家庭电路：进户线（火线零线）、电能表、总开关、保险装置、用电器并联。三孔插座接地保护。测电笔辨别火线零线。安全用电：不接触低压带电体，不靠近高压带电体。

十七、电与磁

磁现象：磁体有磁性，磁极间同名相斥异名相吸。磁感线从 N 极出发回到 S 极（外部），是假想的曲线，不交叉不中断。地磁场的北极在地理南极附近。

电生磁：奥斯特实验证明通电导体周围存在磁场。安培定则（右手螺旋定则）：用右手握住螺线管，四指指向电流方向，拇指指向 N 极。电磁铁——插入铁芯使磁性增强。影响电磁铁磁性强弱的因素：电流大小、线圈匝数、有无铁芯。电磁铁应用：电磁起重机、电铃、电磁继电器。

磁场对电流的作用：通电导体在磁场中受到力的作用，受力方向与电流方向和磁场方向有关（左手定则）。电动机原理：通电线圈在磁场中受力转动，将电能转化为机械能。

磁生电：法拉第发现电磁感应——闭合电路的部分导体在磁场中做切割磁感线运动时产生感应电流。感应电流方向与导体运动方向和磁场方向有关（右手定则）。发电机原理：电磁感应，将机械能转化为电能。交流电：周期性地改变方向的电流，我国 220 V 50 Hz。

十八、能源与可持续发展

能源分类：一次能源（直接从自然界获取：煤、石油、天然气、太阳能、风能、水能）和二次能源（经过加工转换：电能、汽油、氢气）。可再生能源（太阳能、风能、水能）和不可再生能源（煤、石油、天然气、核能）。

核能：核裂变（链式反应——原子弹、核电站）和核聚变（热核反应——氢弹、太阳）。核电站原理：核能→内能→机械能→电能。

能源与环保：化石能源的大量使用导致温室效应、酸雨、雾霾。未来的能源趋势：开发新能源（太阳能、风能、核能、氢能），提高能源利用率，节能减排。
