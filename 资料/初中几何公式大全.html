专题一：平面图形面积公式

基础必会 高频考点

平面图形面积是初中几何的基础板块，也是中考的必考内容。掌握以下公式的同时，要理解每个公式的推导过程和适用条件。

常见平面图形面积公式一览
正方形
S = a²
边长平方。周长 C = 4a
长方形
S = ab
长 × 宽。对角线 d = √(a²+b²)
三角形
S = ½ ah
底 × 高 ÷ 2。注意底与高的对应关系
海伦公式
S = √[p(p-a)(p-b)(p-c)]
p = (a+b+c)/2，已知三边求面积
平行四边形
S = ah
底 × 高。注意高是垂直于底的
梯形
S = ½(a+b)h
(上底+下底) × 高 ÷ 2。中位线 m = (a+b)/2
菱形
S = ½ d₁d₂ = ah
对角线乘积的一半；也可用底×高
圆
S = πr²
π≈3.14。周长 C = 2πr = πd
扇形
S = nπr²/360 = ½lr
n°扇形面积；l弧长。l = nπr/180
弓形
S = S扇 - S△
扇形面积减去三角形面积
记忆技巧：三角形、平行四边形、梯形面积公式的核心都是"底×高"，区别在于要不要乘以½，因为三角形是平行四边形的一半，梯形要取平均底长。
【例题】一个梯形上底为6cm，下底为10cm，高为8cm，求面积。若将该梯形改成等高的平行四边形，其面积有何变化？
解：S梯形 = ½(6+10)×8 = ½×16×8 = 64 (cm²)
等高平行四边形面积 S = 底×高。若底=6（取上底），S=48cm²；若底=10（取下底），S=80cm²。
结论：梯形面积介于两个平行四边形面积之间。梯形相当于"上下底取平均"的平行四边形。
梯形公式的核心思想就是"取上下底的平均"。记住这一点，即使考试时一时忘记，也可以从平行四边形推导出来。
• 一个圆的半径扩大到原来的2倍，面积扩大到原来的几倍？
• 已知三角形三边长分别为5、12、13，求面积（提示：先用勾股定理判断三角形类型）。

专题二：立体图形表面积与体积

空间思维

立体几何在初中阶段主要考查表面积和体积的计算，是中考填空题和解答题的高频考点。注意区分"侧面积"和"全面积"。

常见立体图形体积对比（同底面积下）

1. 柱体（圆柱/棱柱）

圆柱侧面积
S侧 = 2πrh
底面周长 × 高
圆柱全面积
S全 = 2πr² + 2πrh
两个底面积 + 侧面积
圆柱体积
V = πr²h
底面积 × 高
棱柱体积
V = Sh
任意棱柱：底面积 × 高

2. 锥体（圆锥/棱锥）

圆锥侧面积
S侧 = πrl
l 为母线长。l² = r² + h²
圆锥全面积
S全 = πr² + πrl
底面积 + 侧面积
圆锥体积
V = ⅓πr²h
等底等高的圆柱的 1/3
棱锥体积
V = ⅓Sh
底面积 × 高 ÷ 3

3. 球体与台体

球表面积
S = 4πR²
半径为R的球体表面积
球体积
V = ⁴⁄₃πR³
半径为R的球体体积
圆台体积
V = ⅓πh(R²+Rr+r²)
h高，R下底半径，r上底半径
易错提醒：① 圆锥体积公式中的"⅓"容易漏掉；② 球体积公式中R³而非R²，注意区分；③ 计算表面积时看清要求"侧面积"还是"全面积"。
【例题】一个圆锥的底面半径为3cm，高为4cm，求该圆锥的侧面积和体积。
解：
母线 l = √(r²+h²) = √(3²+4²) = 5 (cm)
侧面积 S侧 = πrl = π×3×5 = 15π ≈ 47.1 (cm²)
体积 V = ⅓πr²h = ⅓×π×9×4 = 12π ≈ 37.68 (cm³)
关键步骤：求侧面积前必须先求母线长！母线与底面半径、高构成直角三角形，用勾股定理求解。
• 一个圆柱的底面直径是8cm，高是10cm，求它的全面积和体积。
• 一个球的体积为36π cm³，求它的表面积。

专题三：三角形重要定理与性质

核心定理

三角形是初中几何的核心构件——几乎所有几何问题最终都转化为三角形问题来解决。掌握三角形的定理体系，是学好几何的关键。

三角形分类关系图

三角形基础性质

内角和定理
∠A+∠B+∠C = 180°
任意三角形内角和恒为180°
外角定理
∠外 = ∠A + ∠B
外角等于不相邻两内角和
三边关系
a+b>c, |a-b|
任意两边之和>第三边，差<第三边
边角关系
大边对大角
较长边所对角较大，反之亦然

重要线段公式

中线公式
mₐ = ½√(2b²+2c²-a²)
顶点到对边中点的线段长度
重心
重心分中线 2:1
重心到顶点的距离:到中点=2:1
角平分线定理
BD/DC = AB/AC
角平分线分对边成比例
中位线
DE∥BC, DE = ½BC
两边中点连线平行于第三边且为其一半
易错提醒：三边关系判断是中考选择题的常考点——不仅要检查"两边之和>第三边"，还需要检查"两边之差<第三边"，两者缺一不可。
【例题】在△ABC中，AB=8，AC=6，第三边BC为整数，求BC的可能长度。
解：由三边关系：
8-6 < BC < 8+6 → 2 < BC < 14
BC为整数，所以 BC = 3,4,5,...,13，共11种可能。

拓展：若要求△ABC为直角三角形，则BC = √(8²-6²) = √28 ≈ 5.29（舍去整数要求），或BC = √(8²+6²) = 10。所以BC=10时，构成直角三角形。
三边关系题型的变式：题干改为"周长为整数"、"面积为最大"等条件，本质上还是在考察三边关系的不等式。同时要养成检查"两边之差"的习惯。
• 在△ABC中，∠A=50°，∠B=70°，求∠C的度数，并判断这个三角形按角分类属于哪一类。
• 三角形的三边长分别为3、4、x，若x为整数，求x的所有可能值。

专题四：勾股定理与三角函数

中考重点

勾股定理是初中几何最重要的定理之一，也是中考的必考内容。它揭示了直角三角形三边之间的数量关系：a² + b² = c²（c为斜边）。

勾股定理可视化：a² + b² = c²

勾股定理与逆定理

勾股定理
a² + b² = c²
直角三角形中，两直角边的平方和等于斜边平方
勾股逆定理
a²+b²=c² ⇒ 直角
若三角形三边满足此关系，则为直角三角形
常见勾股数
3-4-5, 5-12-13, 8-15-17
常见的整数勾股组合，牢记可提速
勾股数扩倍
3k-4k-5k
勾股数同时乘以k仍然成立

特殊直角三角形

30°-60°-90°
1:√3:2
30°所对边:60°所对边:斜边
45°-45°-90°
1:1:√2
等腰直角三角形，两腰相等

锐角三角函数

角度sincostan记忆口诀
30°½√3/2√3/31-2-√3，sin30最小
45°√2/2√2/21两个√2/2，tan45=1
60°√3/2½√3与30°正好相反
记忆技巧：sin值从30°→45°→60°依次为 ½ → √2/2 → √3/2（递增），cos值正好相反（递减）。tan = sin/cos。
【例题】如图，一棵树被风吹倒，树尖接触地面时距离树干底部6米，树干与地面夹角为30°，求树的高度。
解：树干、地面和树梢构成直角三角形。
树干与地面夹角30°，所以30°的对边（树尖到地面的高度h）与斜边（树长L）的关系：
tan30° = h/6 → h = 6×tan30° = 6×√3/3 = 2√3 ≈ 3.46 (m)
树长 L = h/sin30° = 2√3÷½ = 4√3 ≈ 6.93 (m)
或者先用cos30°求斜边：L = 6/cos30° = 6÷(√3/2) = 12/√3 = 4√3
解直角三角形问题的关键是找到已知边与所求边之间的三角函数关系。记住：sin=对/斜，cos=邻/斜，tan=对/邻。
• 直角三角形两直角边分别为5和12，求斜边长度。
• 已知sinA=3/5，且∠A为锐角，求cosA和tanA的值。

专题五：圆的公式与性质

几何综合

圆的相关定理是中考几何的综合应用部分，常与三角形、四边形结合考查。掌握圆的性质是解决综合性问题的前提。

圆的基本元素示意图

圆的基本量计算

圆周长
C = 2πr = πd
π取3.14或保留π
圆面积
S = πr²
牢记！区分周长和面积公式
弧长
l = nπr/180
n°圆心角所对弧长
扇形面积
S = nπr²/360 = lr/2
n°扇形。lr/2类比三角形

重要定理

垂径定理
CD⊥AB ⇒ AE=EB
直径垂直于弦则平分弦及其所对弧
圆周角定理
∠C = ½∠AOB
同弧所对圆周角是圆心角一半
直径对直角
∠ACB = 90°
直径所对圆周角=90°
切线性质
OA⊥PA
切线垂直于过切点的半径
切线长定理
PA = PB
过圆外一点的两条切线长相等
易错提醒：圆周角定理使用时要确认"同弧"条件。两个圆周角必须对着同一个弧才能相等，不能只看角度大小相同。
【例题】如图，AB是⊙O的直径，∠CAB=40°，求∠ABC的度数。
解：
∵ AB是直径
∴ ∠ACB = 90°（直径所对圆周角为直角）
在△ABC中，∠CAB + ∠ABC + ∠ACB = 180°
∴ 40° + ∠ABC + 90° = 180°
∴ ∠ABC = 50°
看到"直径"二字，第一反应就是"直径所对的圆周角等于90°"。这是圆的题目中最常用的定理之一。
• 一个扇形的圆心角为72°，半径为5cm，求扇形面积。（π取3.14）
• 圆的半径为10cm，一条弦长为12cm，求圆心到该弦的距离。

专题六：四边形与多边形

进阶必会

四边形是中考几何的综合载体，特别是平行四边形、矩形、菱形、正方形之间的性质与判定，是解答题的高频命题方向。

平行四边形

面积
S = ah
底×高。注意h是垂直距离
对角线和
d₁²+d₂²=2(a²+b²)
对角线平方和等于四边平方和
性质
对边平行且相等
对角相等，邻角互补，对角线互相平分

特殊平行四边形

图形额外性质对角线特点面积补充
矩形四个角=90°对角线相等S = ab
菱形四条边相等对角线互相垂直S = ½d₁d₂
正方形兼具矩形和菱形性质相等且垂直S = a² = ½d²

多边形一般公式

n边形内角和
(n-2) × 180°
从三角形开始，每多一边增加180°
n边形外角和
360°
任意凸多边形外角和恒为360°
正n边形内角
(n-2)×180°/n
每个内角大小相等
正n边形对角线数
n(n-3)/2
n个顶点，每个连(n-3)条
记忆技巧：内角和公式记作"边数减2乘180"——因为n边形可以分成(n-2)个三角形。外角和不需记忆，因为外角=180°-内角，取和即可。
【例题】一个正多边形的每个内角都是150°，求这个多边形的边数。
解法一：设边数为n，则 (n-2)×180° = n×150°
180n - 360 = 150n → 30n = 360 → n = 12

解法二：每个外角 = 180°-150° = 30°
外角和360° → 360°÷30° = 12 → n = 12
该多边形为正十二边形。
解法二利用"外角和=360°"更快捷。记住：已知内角求边数时，用外角法最快——外角=180°-内角，360°÷外角=边数。
• 四边形ABCD中，∠A:∠B:∠C:∠D=1:2:3:4，求各角度数。
• 一个多边形的内角和是外角和的3倍，求边数。

专题七：相似与全等

证明基础

全等和相似是初中几何证明题的两种核心方法。全等意味着"形状和大小都相同"（对应边相等），相似意味着"形状相同但大小不同"（对应边成比例）。

全等三角形判定条件

SSS（边边边）
三边对应相等
三条边都对应相等
SAS（边角边）
两边及其夹角相等
必须是夹角！
ASA（角边角）
两角及其夹边相等
两角和它们的夹边
AAS（角角边）
两角及一邻边相等
两角和任意一边
HL（斜边直角边）
R△中斜边+一直角边
仅直角三角形可用

相似三角形判定条件

AA（角角）
两角对应相等
最常用的相似判定
SAS（两边成比例+夹角）
两边成比例且夹角相等
比例相等是重点
SSS（三边成比例）
三边对应成比例
比例相同即相似

相似比与面积比

周长比
C₁:C₂ = k
周长比等于相似比
面积比
S₁:S₂ = k²
面积比等于相似比的平方！关键！
对应高线比
h₁:h₂ = k
高的比也等于相似比
易错提醒：全等证明中"SSA"（两边及其中一边的对角）不能判定全等。不要因为直觉上"感觉对"就使用。面积比等于相似比的平方，不是等于相似比！
【例题】两个相似三角形的相似比为2:3，其中小三角形的面积为12cm²，求大三角形的面积。
解：面积比 = k² = (2/3)² = 4/9
小面:大面 = 4:9 → 12:S大 = 4:9
4S大 = 12×9 = 108 → S大 = 27 (cm²)
牢记"面积比是相似比的平方"！这是一级易错点。设未知量时最好用比例方程，不容易出错。
• 已知△ABC≌△DEF，AB=5cm，BC=7cm，AC=9cm，求△DEF的周长。
• 两个相似三角形面积比为9:25，求它们的相似比。

专题八：几何解题模型与策略

模型名称特征核心方法常见题型

学会"倒推"——从求证出发，思考要得到结论需要什么条件，一步步往前推
【综合例题】矩形ABCD中，AB=6，AD=8，E是AD的中点，F是BC上一点且BF=2，连接EF交对角线AC于点G，求AG:GC。
解：
构造辅助线：延长FE交BA的延长线于点H。
∵ AE=ED=4，AE∥BC → △AHE∽△BFE
AH/BF = AE/BE(不需要) → 需通过坐标法或相似求解

方法：建立坐标系
设A(0,0), B(6,0), C(6,8), D(0,8)
E(0,4), F(6,2)
直线EF: 过E(0,4)和F(6,2) → 斜率=(2-4)/(6-0) = -1/3 → y = -x/3 + 4
直线AC: 过A(0,0)和C(6,8) → y = 4x/3
交点G: -x/3+4 = 4x/3 → 5x/3=4 → x=12/5=2.4, y=4×2.4/3=3.2
AG = √(2.4²+3.2²) = √(5.76+10.24) = √16 = 4
AC = √(6²+8²) = 10
∴ AG:GC = 4:6 = 2:3
当几何方法感觉困难时，建立坐标系用解析法是很好的替代方案。中考几何中，有时用"坐标法"比纯几何证明更快更准。
• 总结你遇到的几何题中，哪些"辅助线"的添加方法最常用？整理成自己的笔记。
• 回顾最近做过的几何难题，归类到上述"五大模型"中，确认每种模型的典型特征。
