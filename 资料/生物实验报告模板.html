实验一：显微镜的使用

观察工具

实验目的

认识显微镜的基本结构，学会规范使用显微镜观察玻片标本。

光学显微镜结构示意图

实验器材

光学显微镜、永久玻片标本（或临时装片）、擦镜纸、纱布。

显微镜结构

显微镜主要由机械部分（镜座、镜柱、镜臂、镜筒、粗准焦螺旋、细准焦螺旋、载物台、压片夹）和光学部分（目镜、物镜、反光镜、遮光器）组成。

操作步骤

  1. 取镜和安放：右手握镜臂，左手托镜座，将显微镜放在实验台偏左距边缘约7cm处。
  2. 对光：转动转换器使低倍物镜对准通光孔，转动遮光器选大光圈，左眼注视目镜，右眼睁开，转动反光镜看到明亮圆形视野。
  3. 安放玻片：将玻片标本放在载物台上，用压片夹压住，使标本正对通光孔中心。
  4. 调焦：转动粗准焦螺旋使镜筒下降至物镜接近玻片（从侧面看），左眼看目镜缓慢上升镜筒至看到物像，再调细准焦螺旋使物像清晰。
  5. 观察与收镜：移动玻片寻找目标，观察完毕升高镜筒取下玻片，将显微镜放回原处。

放大倍数

显微镜放大倍数 = 目镜放大倍数 × 物镜放大倍数。常用目镜有 5×、10×、16×，物镜有 10×、40×（油镜 100×）。

注意：取镜时必须一手握镜臂一手托镜座。对光时要用低倍物镜和大光圈。观察时先粗调再细调。物像为倒像，移动玻片时物像移动方向与玻片移动方向相反。

实验二：观察叶片的结构

植物学

实验目的

通过制作叶片横切面临时装片，观察叶片的基本结构。

叶片横切面结构（左）与叶绿体结构放大（右）

实验器材

新鲜叶片（如菠菜叶）、显微镜、双面刀片、镊子、载玻片、盖玻片、培养皿、清水、滴管、碘液。

叶片结构

结构层次组成功能
表皮（上/下）一层排列紧密的细胞（无色透明）保护作用，气孔调节蒸腾作用与气体交换
栅栏组织靠近上表皮，细胞长柱形，排列整齐含大量叶绿体，光合作用主要场所
海绵组织靠近下表皮，细胞不规则，排列疏松含少量叶绿体，细胞间隙利于气体交换
叶脉维管束（导管+筛管）导管运输水和无机盐，筛管运输有机物

实验步骤

  1. 将叶片平放在载玻片上，用刀片沿与主脉垂直方向切割（夹在两个刀片之间快速切割）。
  2. 用毛笔蘸取最薄的切片，放在滴有清水的载玻片上。
  3. 盖上盖玻片（一侧先接触液面再缓缓放下，避免产生气泡），制成临时装片。
  4. 在低倍镜下观察叶片结构，识别表皮、叶肉、叶脉。
  5. 换高倍镜观察栅栏组织和海绵组织中的叶绿体。
注意：切片越薄越好（透光），制成临时装片时要避免产生气泡。盖盖玻片时用镊子夹住一侧倾斜45°缓慢放下。

实验三：探究唾液对淀粉的消化作用

生理学

实验目的

探究唾液淀粉酶对淀粉的消化作用，理解淀粉在口腔中的初步消化。

实验原理

淀粉遇碘变蓝。唾液中含有唾液淀粉酶，能将淀粉分解为麦芽糖（麦芽糖遇碘不变蓝）。

实验器材

试管（2支）、烧杯、量筒、温度计、滴管、碘液、淀粉糊、唾液、37℃温水。

实验步骤

  1. 取两支试管编号 A、B，各加入等量（约2mL）淀粉糊。
  2. A试管加入2mL唾液，B试管加入2mL清水（对照）。
  3. 将两支试管同时放入37℃温水中水浴加热5~10分钟。
  4. 取出试管，各滴入2滴碘液，摇匀观察颜色变化。

实验现象与结论

试管加入物质水浴后加碘现象结论
A淀粉糊 + 唾液37℃ 10min不变蓝淀粉已被唾液淀粉酶分解
B（对照）淀粉糊 + 清水37℃ 10min变蓝淀粉没有被分解

唾液中的唾液淀粉酶能将淀粉分解为麦芽糖，淀粉的消化从口腔开始。

注意：水浴温度37℃：模拟口腔温度，温度过高（>80℃）酶会变性失活，温度过低抑制酶活性。设置对照实验是为了排除其他因素的干扰。

实验四：观察人的口腔上皮细胞

细胞学

实验目的

制作人的口腔上皮细胞临时装片，观察动物细胞的基本结构。

实验器材

显微镜、载玻片、盖玻片、生理盐水（0.9% NaCl）、稀碘液、消毒牙签、吸水纸、纱布。

实验步骤

  1. 用洁净纱布擦拭载玻片和盖玻片。
  2. 在载玻片中央滴一滴生理盐水。
  3. 用消毒牙签在口腔内侧壁轻轻刮取细胞（不要用力）。
  4. 将牙签在载玻片上的生理盐水中涂抹几下。
  5. 盖上盖玻片（一侧先接触液面再缓缓放下）。
  6. 在盖玻片一侧滴加碘液，另一侧用吸水纸吸引染色。
  7. 在显微镜下观察（先用低倍镜找到细胞，再换高倍镜观察结构）。

观察到的结构

结构特点
细胞膜最外侧，极薄，控制物质进出
细胞质细胞膜以内、细胞核以外的透明胶状物质
细胞核位于细胞中央，染色最深（含遗传物质）

动物细胞 vs 植物细胞

比较项目动物细胞植物细胞
细胞壁无有（纤维素）
细胞膜有有
细胞质有有
细胞核有有
液泡无（或微小）有（中央大液泡）
叶绿体无有（绿色部分）
注意：用生理盐水代替清水是为了维持细胞正常形态（防止细胞吸水涨破）。刮取口腔上皮细胞时不要用力过猛以免出血。染色可使细胞核更清晰可见。

实验五：探究光合作用的产物

植物学

实验目的

验证光合作用产生淀粉（有机物）和氧气，理解光合作用的条件和意义。

实验原理

6CO₂ + 6H₂O → C₆H₁₂O₆ + 6O₂（光、叶绿体）。光合作用在光下产生淀粉（遇碘变蓝）和氧气（使带火星木条复燃）。

实验器材

盆栽天竺葵（或黑藻）、烧杯、酒精灯、三脚架、石棉网、酒精、碘液、大烧杯、小烧杯、镊子、火柴、木条、黑纸片、回形针。

实验步骤

  1. 暗处理：将天竺葵放在黑暗处一昼夜（消耗原有淀粉）。
  2. 遮光对照：用黑纸片将部分叶片从上下两面遮盖，移至光下照射2~4小时。
  3. 脱色处理：摘下叶片放入酒精中隔水加热（酒精变绿，叶片变黄白色）。
  4. 漂洗：用清水漂洗叶片。
  5. 染色：在叶片上滴加碘液。
  6. 观察：遮光部分不变蓝（未进行光合作用），见光部分变蓝（产生了淀粉）。

实验现象与结论

处理现象结论
见光部分滴碘液变蓝有淀粉产生，光照是光合作用的必要条件
遮光部分滴碘液不变蓝没有淀粉产生

光合作用与呼吸作用对比

比较光合作用呼吸作用
场所叶绿体线粒体
条件光有光/无光均可
原料CO₂ + H₂O有机物 + O₂
产物有机物 + O₂CO₂ + H₂O + 能量
实质合成有机物储存能量分解有机物释放能量
注意：暗处理的目的是消耗原有淀粉，避免干扰实验结果。隔水加热酒精脱色：酒精易燃不能直接加热，用热水浴加热。遮光对照：同一叶片形成对照，保证变量唯一。
