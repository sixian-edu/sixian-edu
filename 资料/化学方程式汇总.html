化学方程式汇总

化合 · 分解 · 置换 · 复分解 · 离子检验 · 金属活动性 · 有机 · 工业

记忆口诀：化学方程式书写步骤——"写配注等"：写化学式、配平、注条件（Δ/↑/↓）、检查等号两边原子个数是否相等。

化合反应（A + B → AB）

反应物化学方程式现象/说明
碳 + 氧气（充足）C + O₂ → CO₂发出白光，生成使石灰水变浑浊的气体
碳 + 氧气（不足）2C + O₂ → 2CO生成一氧化碳（不完全燃烧）
硫 + 氧气S + O₂ → SO₂蓝紫色火焰，有刺激性气味
磷 + 氧气4P + 5O₂ → 2P₂O₅产生大量白烟（五氧化二磷）
镁 + 氧气2Mg + O₂ → 2MgO发出耀眼白光，生成白色固体
铁 + 氧气3Fe + 2O₂ → Fe₃O₄火星四射，生成黑色固体
氢气 + 氧气2H₂ + O₂ → 2H₂O淡蓝色火焰，产物为水
钠 + 氧气（常温）4Na + O₂ → 2Na₂O银白色固体表面变暗，生成氧化钠
钠 + 氧气（点燃）2Na + O₂ → Na₂O₂黄色火焰，生成淡黄色过氧化钠
铝 + 氧气4Al + 3O₂ → 2Al₂O₃铝表面形成致密氧化膜（钝化）
铜 + 氧气（加热）2Cu + O₂ → 2CuO红色铜丝加热变黑
一氧化碳 + 氧气2CO + O₂ → 2CO₂蓝色火焰，产物使石灰水变浑浊
二氧化碳 + 水CO₂ + H₂O → H₂CO₃碳酸使紫色石蕊试液变红
二氧化硫 + 水SO₂ + H₂O → H₂SO₃亚硫酸，酸性氧化物与水反应
三氧化硫 + 水SO₃ + H₂O → H₂SO₄工业制硫酸的关键一步
氧化钙 + 水CaO + H₂O → Ca(OH)₂放热反应（生石灰变熟石灰）
氧化钠 + 水Na₂O + H₂O → 2NaOH碱性氧化物与水反应生成碱
氧化钾 + 水K₂O + H₂O → 2KOH生成强碱氢氧化钾
氨气 + 氯化氢NH₃ + HCl → NH₄Cl产生大量白烟
氢气 + 氯气（点燃）H₂ + Cl₂ → 2HCl苍白色火焰，瓶口有白雾
铁 + 硫（加热）Fe + S → FeS黑色固体生成（硫化亚铁）
一氧化氮 + 氧气2NO + O₂ → 2NO₂无色气体变为红棕色
氮气 + 氢气N₂ + 3H₂ ⇌ 2NH₃工业合成氨，高温高压催化剂
二氧化氮聚合2NO₂ ⇌ N₂O₄红棕色变无色（降温正向）

分解反应（AB → A + B）

反应物化学方程式现象/说明
水通电分解2H₂O → 2H₂↑ + O₂↑正极 O₂（使带火星木条复燃），负极 H₂（可燃烧）
过氧化氢分解2H₂O₂ → 2H₂O + O₂↑MnO₂ 催化，实验室制 O₂
高锰酸钾分解2KMnO₄ → K₂MnO₄ + MnO₂ + O₂↑加热，实验室制 O₂
氯酸钾分解2KClO₃ → 2KCl + 3O₂↑MnO₂ 催化加热
碳酸分解H₂CO₃ → H₂O + CO₂↑碳酸不稳定，易分解
碳酸钙高温分解CaCO₃ → CaO + CO₂↑高温煅烧石灰石，工业制生石灰
氧化汞分解2HgO → 2Hg + O₂↑加热，红色粉末变银白色液体
氢氧化铁分解2Fe(OH)₃ → Fe₂O₃ + 3H₂O红褐色沉淀加热分解
氢氧化铜分解Cu(OH)₂ → CuO + H₂O蓝色沉淀加热变黑
氢氧化铝分解2Al(OH)₃ → Al₂O₃ + 3H₂O白色沉淀加热分解
碳酸氢铵分解NH₄HCO₃ → NH₃↑ + H₂O + CO₂↑产生刺激性气味气体
碳酸氢钠分解2NaHCO₃ → Na₂CO₃ + H₂O + CO₂↑加热固体质量减少，用于制纯碱
碳酸铵分解(NH₄)₂CO₃ → 2NH₃↑ + H₂O + CO₂↑产生两种气体
碱式碳酸铜分解Cu₂(OH)₂CO₃ → 2CuO + H₂O + CO₂↑绿色粉末加热变黑（铜绿分解）
碳酸氢钙分解Ca(HCO₃)₂ → CaCO₃↓ + H₂O + CO₂↑水垢形成原理
氧化银分解2Ag₂O → 4Ag + O₂↑加热，黑色粉末变银白色
氢氧化镁分解Mg(OH)₂ → MgO + H₂O白色沉淀加热分解
氢氧化钙分解Ca(OH)₂ → CaO + H₂O高温分解

置换反应（A + BC → AC + B）

反应物化学方程式现象/说明
锌 + 稀硫酸Zn + H₂SO₄ → ZnSO₄ + H₂↑锌粒表面有气泡，实验室制 H₂
锌 + 稀盐酸Zn + 2HCl → ZnCl₂ + H₂↑产生大量气泡
铁 + 稀硫酸Fe + H₂SO₄ → FeSO₄ + H₂↑溶液变浅绿色，有气泡
铁 + 稀盐酸Fe + 2HCl → FeCl₂ + H₂↑有气泡，溶液变浅绿色
镁 + 稀盐酸Mg + 2HCl → MgCl₂ + H₂↑剧烈反应，大量气泡
镁 + 稀硫酸Mg + H₂SO₄ → MgSO₄ + H₂↑剧烈反应，放热
铝 + 稀硫酸2Al + 3H₂SO₄ → Al₂(SO₄)₃ + 3H₂↑铝表面有气泡
铝 + 稀盐酸2Al + 6HCl → 2AlCl₃ + 3H₂↑铝溶解，产生气泡
铁 + 硫酸铜Fe + CuSO₄ → FeSO₄ + Cu铁表面覆盖红色铜，溶液由蓝变浅绿
锌 + 硫酸铜Zn + CuSO₄ → ZnSO₄ + Cu锌表面析出红色铜
镁 + 硫酸铜Mg + CuSO₄ → MgSO₄ + Cu镁表面析出红色铜，溶液蓝色变浅
铜 + 硝酸银Cu + 2AgNO₃ → Cu(NO₃)₂ + 2Ag铜表面析出银白色银
铁 + 硝酸银Fe + 2AgNO₃ → Fe(NO₃)₂ + 2Ag铁表面析出银白色银
锌 + 硫酸亚铁Zn + FeSO₄ → ZnSO₄ + Fe锌表面析出黑色铁
镁 + 硫酸亚铁Mg + FeSO₄ → MgSO₄ + Fe镁表面析出黑色铁
碳 + 氧化铜C + 2CuO → 2Cu + CO₂↑高温，黑色粉末变红
碳 + 氧化铁3C + 2Fe₂O₃ → 4Fe + 3CO₂↑高温，工业炼铁原理
氢气 + 氧化铜H₂ + CuO → Cu + H₂O加热，黑色变红，管壁有水珠
铝 + 氧化铁（铝热反应）2Al + Fe₂O₃ → 2Fe + Al₂O₃高温，剧烈放热，焊接铁轨

复分解反应（AB + CD → AD + CB）

复分解反应发生的条件：生成物中有沉淀、气体或水。

酸 + 碱（中和反应）

反应物化学方程式说明
盐酸 + 氢氧化钠HCl + NaOH → NaCl + H₂O无明显现象（借助指示剂判断）
硫酸 + 氢氧化钠H₂SO₄ + 2NaOH → Na₂SO₄ + 2H₂O放热，pH 趋向 7
盐酸 + 氢氧化钙Ca(OH)₂ + 2HCl → CaCl₂ + 2H₂O常用于中和酸性土壤
硝酸 + 氢氧化钾HNO₃ + KOH → KNO₃ + H₂O中和反应
硫酸 + 氢氧化钡H₂SO₄ + Ba(OH)₂ → BaSO₄↓ + 2H₂O白色沉淀 + 生成水
盐酸 + 氨水HCl + NH₃·H₂O → NH₄Cl + H₂O氨水与盐酸反应

酸 + 盐

反应物化学方程式说明
盐酸 + 硝酸银HCl + AgNO₃ → AgCl↓ + HNO₃白色沉淀（不溶于稀硝酸）
硫酸 + 氯化钡H₂SO₄ + BaCl₂ → BaSO₄↓ + 2HCl白色沉淀（不溶于酸）
盐酸 + 碳酸钙2HCl + CaCO₃ → CaCl₂ + H₂O + CO₂↑实验室制 CO₂
硫酸 + 碳酸钠H₂SO₄ + Na₂CO₃ → Na₂SO₄ + H₂O + CO₂↑产生气泡
盐酸 + 碳酸氢钠HCl + NaHCO₃ → NaCl + H₂O + CO₂↑小苏打与盐酸反应（胃药原理）
硫酸 + 亚硫酸钠H₂SO₄ + Na₂SO₃ → Na₂SO₄ + H₂O + SO₂↑产生刺激性气体 SO₂

碱 + 盐

反应物化学方程式说明
氢氧化钠 + 硫酸铜2NaOH + CuSO₄ → Cu(OH)₂↓ + Na₂SO₄产生蓝色沉淀
氢氧化钠 + 氯化铁3NaOH + FeCl₃ → Fe(OH)₃↓ + 3NaCl产生红褐色沉淀
氢氧化钙 + 碳酸钠Ca(OH)₂ + Na₂CO₃ → CaCO₃↓ + 2NaOH工业制 NaOH 原理
氢氧化钡 + 硫酸钠Ba(OH)₂ + Na₂SO₄ → BaSO₄↓ + 2NaOH白色沉淀
氢氧化钠 + 氯化镁2NaOH + MgCl₂ → Mg(OH)₂↓ + 2NaCl白色沉淀
氢氧化钙 + 氯化镁Ca(OH)₂ + MgCl₂ → Mg(OH)₂↓ + CaCl₂海水制镁原理
氢氧化钠 + 氯化铜2NaOH + CuCl₂ → Cu(OH)₂↓ + 2NaCl蓝色沉淀

盐 + 盐

反应物化学方程式说明
氯化钠 + 硝酸银NaCl + AgNO₃ → AgCl↓ + NaNO₃白色沉淀（检验 Cl⁻）
硫酸钠 + 氯化钡Na₂SO₄ + BaCl₂ → BaSO₄↓ + 2NaCl白色沉淀（检验 SO₄²⁻）
氯化钙 + 碳酸钠CaCl₂ + Na₂CO₃ → CaCO₃↓ + 2NaCl白色沉淀
硝酸钡 + 硫酸钠Ba(NO₃)₂ + Na₂SO₄ → BaSO₄↓ + 2NaNO₃白色沉淀
硝酸银 + 氯化铁3AgNO₃ + FeCl₃ → 3AgCl↓ + Fe(NO₃)₃白色沉淀

常见离子检验

离子检验试剂现象化学方程式（示例）
H⁺（酸）紫色石蕊试液石蕊变红—
OH⁻（碱）无色酚酞试液酚酞变红—
Cl⁻AgNO₃ 溶液 + 稀 HNO₃白色沉淀Ag⁺ + Cl⁻ → AgCl↓
SO₄²⁻BaCl₂ 溶液 + 稀 HNO₃白色沉淀Ba²⁺ + SO₄²⁻ → BaSO₄↓
CO₃²⁻稀 HCl + 澄清石灰水气泡使石灰水变浑浊2H⁺ + CO₃²⁻ → H₂O + CO₂↑
NH₄⁺NaOH 溶液 + 加热刺激性气味气体NH₄⁺ + OH⁻ → NH₃↑ + H₂O

Cu²⁺NaOH 溶液蓝色沉淀Cu²⁺ + 2OH⁻ → Cu(OH)₂↓
Mg²⁺NaOH 溶液白色沉淀Mg²⁺ + 2OH⁻ → Mg(OH)₂↓
Al³⁺NaOH 溶液（过量）先沉淀后溶解Al³⁺ + 3OH⁻ → Al(OH)₃↓ → AlO₂⁻

金属活动性顺序

K Ca Na Mg Al Zn Fe Sn Pb (H) Cu Hg Ag Pt Au

金属活动性由强到弱，排在 H 前面的金属能置换出酸中的氢，排在前面的金属能把后面的金属从其盐溶液中置换出来。

特例：K、Ca、Na 与盐溶液反应时，先与水反应生成碱和氢气，碱再与盐发生复分解反应。

工业重要反应

工业制硫酸（接触法）

步骤化学方程式条件
① 制 SO₂S + O₂ → SO₂（或 4FeS₂ + 11O₂ → 2Fe₂O₃ + 8SO₂）点燃
② SO₂ 氧化2SO₂ + O₂ ⇌ 2SO₃V₂O₅ 催化，450°C
③ 吸收SO₃ + H₂O → H₂SO₄用浓硫酸吸收

工业制硝酸（氨氧化法）

步骤化学方程式条件
① 氨氧化4NH₃ + 5O₂ → 4NO + 6H₂OPt-Rh 催化，800°C
② NO 氧化2NO + O₂ → 2NO₂常温
③ 吸收3NO₂ + H₂O → 2HNO₃ + NONO 循环利用

其他工业反应

工艺化学方程式说明
高炉炼铁Fe₂O₃ + 3CO → 2Fe + 3CO₂CO 还原铁矿石
氯碱工业2NaCl + 2H₂O → 2NaOH + Cl₂↑ + H₂↑电解饱和食盐水
工业制盐酸H₂ + Cl₂ → 2HCl点燃后用水吸收
工业制纯碱（侯氏制碱）NaCl + NH₃ + H₂O + CO₂ → NaHCO₃↓ + NH₄Cl利用溶解度差异
铝热反应（焊接）2Al + Fe₂O₃ → 2Fe + Al₂O₃镁条引燃，剧烈放热
化学方程式配平方法：（1）最小公倍数法——找出左右各出现一次且原子数差异较大的元素，求最小公倍数；（2）观察法——从原子数最复杂的化学式入手；（3）奇数配偶法——将奇数变偶数。
常见反应条件：Δ 表示加热（温度低于酒精灯火焰温度）；高温指温度高于酒精灯火焰所能达到的温度；↑ 表示气体生成；↓ 表示沉淀生成。⇌ 表示可逆反应。
与量有关的反应记忆：（1）CO₂ 与碱反应——少量生成碳酸盐，过量生成碳酸氢盐；（2）多元酸与碱反应——注意物质的量之比；（3）Fe 与 HNO₃——铁过量生成 Fe²⁺，硝酸过量生成 Fe³⁺。
