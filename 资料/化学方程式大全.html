🧪 初中化学 · 方程式大全

化合 · 分解 · 置换 · 复分解 · 配平 · 计算 · 中考必备

化合反应（A + B → AB）
🔹 氢气的燃烧
点燃 氢气在氧气中燃烧，产生淡蓝色火焰。
产物只有水，是最清洁的燃料。
2H₂ + O₂ →(点燃) 2H₂O
🔹 碳的燃烧
充分燃烧 碳在氧气中充分燃烧生成二氧化碳
不充分燃烧 氧气不足时生成一氧化碳（有毒）
C + O₂ →(点燃) CO₂  |  2C + O₂ →(点燃) 2CO
🔹 一氧化碳的燃烧
点燃 CO在空气中燃烧，产生蓝色火焰。
CO是常见燃料（水煤气的主要成分之一）。
2CO + O₂ →(点燃) 2CO₂
🔹 镁与氧气的反应
点燃 镁条在空气中剧烈燃烧，发出耀眼白光，生成白色固体。
用于照明弹、烟花。
2Mg + O₂ →(点燃) 2MgO
🔹 铁与氧气的反应
点燃 铁丝在氧气中剧烈燃烧，火星四射，生成黑色固体。
注意 集气瓶底要放少量水或细沙（防止炸裂）。
3Fe + 2O₂ →(点燃) Fe₃O₄
🔹 铝与氧气的反应
点燃 铝在氧气中燃烧，发出耀眼白光。
铝在空气中表面会形成一层致密的氧化铝保护膜（Al₂O₃），因此铝有良好的抗腐蚀性。
4Al + 3O₂ →(点燃) 2Al₂O₃
🔹 磷与硫的燃烧
磷燃烧 产生大量白烟（P₂O₅固体），用于测定空气中氧气含量
硫燃烧 发出明亮的蓝紫色火焰（纯氧中），生成有刺激性气味的气体
4P + 5O₂ →(点燃) 2P₂O₅  |  S + O₂ →(点燃) SO₂
🔹 金属与氯气的反应
钠 剧烈燃烧，产生白烟，生成白色固体
铁 剧烈燃烧，产生棕褐色烟
2Na + Cl₂ →(点燃) 2NaCl  |  2Fe + 3Cl₂ →(点燃) 2FeCl₃
🔹 碱性氧化物与水
碱性氧化物与水反应生成对应的碱。
注意 只有可溶性碱对应的碱性氧化物才能与水直接反应（如CuO不能与水反应）。
CaO + H₂O → Ca(OH)₂  |  Na₂O + H₂O → 2NaOH  |  K₂O + H₂O → 2KOH
🔹 酸性氧化物与水
酸性氧化物与水反应生成对应的酸（酸酐的水化反应）。
注意 SiO₂不与水反应（H₂SiO₃须由可溶性硅酸盐与酸反应制得）。
CO₂ + H₂O → H₂CO₃  |  SO₂ + H₂O → H₂SO₃  |  SO₃ + H₂O → H₂SO₄
分解反应（AB → A + B）
🔹 水的电解
通电 正极产生氧气，负极产生氢气，体积比为 1 : 2。
证明水由氢元素和氧元素组成。
2H₂O →(通电) 2H₂↑ + O₂↑
🔹 加热高锰酸钾制氧气
加热 紫黑色固体高锰酸钾分解。
试管口要略向下倾斜，管口放一团棉花（防止粉末堵塞导管）。
2KMnO₄ →(△) K₂MnO₄ + MnO₂ + O₂↑
🔹 加热氯酸钾制氧气
MnO₂催化 氯酸钾加热分解。
MnO₂在此反应中作为催化剂，反应前后质量和化学性质不变。
2KClO₃ →(MnO₂△) 2KCl + 3O₂↑
🔹 过氧化氢分解制氧气
MnO₂催化 过氧化氢溶液在常温下分解。
无需加热，操作简便安全，是实验室制氧气最常用的方法之一。
2H₂O₂ →(MnO₂) 2H₂O + O₂↑
🔹 碳酸钙高温分解
高温 石灰石（主要成分CaCO₃）高温分解。
这是工业制取生石灰（CaO）的原理。生石灰与水反应放热。
CaCO₃ →(高温) CaO + CO₂↑
🔹 碳酸/亚硫酸分解
不稳定的酸容易分解为水和对应的酸性氧化物。
碳酸饮料 打开瓶盖后冒出的气泡就是CO₂。
H₂CO₃ → H₂O + CO₂↑  |  H₂SO₃ → H₂O + SO₂↑
🔹 铵盐的分解
碳酸氢铵（碳铵）不稳定，受热易分解。
铵态氮肥应密封储存于阴凉处，避免高温和暴晒。
NH₄HCO₃ →(△) NH₃↑ + H₂O + CO₂↑
🔹 碱式碳酸铜分解
铜绿（碱式碳酸铜）受热分解，绿色粉末变为黑色固体。
可用于鉴别碱式碳酸铜。
Cu₂(OH)₂CO₃ →(△) 2CuO + H₂O + CO₂↑
🔹 氧化汞与氧化铝分解
氧化汞 红色粉末加热分解（拉瓦锡实验）
氧化铝 电解熔融氧化铝制金属铝
2HgO →(△) 2Hg + O₂↑  |  2Al₂O₃ →(通电) 4Al + 3O₂↑
置换反应（A + BC → AC + B）
🔹 活泼金属与酸反应制氢气
Mg 剧烈反应，产生大量气泡
Zn 反应较剧烈（实验室制H₂常用）
Fe 反应缓慢，溶液变浅绿色
Al 反应剧烈
Zn + H₂SO₄ → ZnSO₄ + H₂↑  |  Mg + 2HCl → MgCl₂ + H₂↑
Fe + H₂SO₄ → FeSO₄ + H₂↑  |  2Al + 6HCl → 2AlCl₃ + 3H₂↑
🔹 铁与铜盐溶液的反应
湿法冶铜 铁与硫酸铜溶液反应，铁表面析出红色固体。
溶液由蓝色逐渐变为浅绿色。铁的活动性比铜强。
Fe + CuSO₄ → FeSO₄ + Cu
🔹 铜与银盐溶液的反应
铜丝浸入硝酸银溶液 表面析出银白色固体。
溶液由无色变为蓝色。铜的活动性比银强。
Cu + 2AgNO₃ → Cu(NO₃)₂ + 2Ag
🔹 其他金属与盐的置换
金属活动性顺序中，排在前面的金属能把后面的金属从它的盐溶液中置换出来。
注意 K、Ca、Na等极活泼金属会先与水反应，不能置换盐溶液中的金属。
Mg + CuSO₄ → MgSO₄ + Cu  |  2Al + 3CuSO₄ → Al₂(SO₄)₃ + 3Cu
🔹 碳还原金属氧化物
高温 碳具有还原性，可将金属氧化物还原为金属单质。
古代冶铜、冶铁都利用了碳的还原性。
C + 2CuO →(高温) 2Cu + CO₂↑  |  2Fe₂O₃ + 3C →(高温) 4Fe + 3CO₂↑
🔹 氢气还原金属氧化物
加热 H₂具有还原性，能将CuO等还原为金属单质。
实验步骤：先通氢，后加热；先撤灯，后停氢（防止爆炸和金属重新氧化）。
H₂ + CuO →(△) Cu + H₂O  |  3H₂ + Fe₂O₃ →(高温) 2Fe + 3H₂O
复分解反应（AB + CD → AD + CB）
🔹 酸碱中和反应
中和反应 酸和碱生成盐和水，属于复分解反应。
实质：H⁺ + OH⁻ → H₂O。农业上常用熟石灰改良酸性土壤。
NaOH + HCl → NaCl + H₂O  |  Ca(OH)₂ + 2HCl → CaCl₂ + 2H₂O
2NaOH + H₂SO₄ → Na₂SO₄ + 2H₂O  |  Ba(OH)₂ + H₂SO₄ → BaSO₄↓ + 2H₂O
🔹 酸与碳酸盐反应
产生气泡 碳酸盐与酸反应生成CO₂气体。
用于实验室制CO₂(大理石+稀盐酸)、碳酸盐的鉴别。
CaCO₃ + 2HCl → CaCl₂ + H₂O + CO₂↑  |  Na₂CO₃ + 2HCl → 2NaCl + H₂O + CO₂↑
🔹 酸与碳酸氢盐反应
小苏打 碳酸氢钠与酸反应产生CO₂。
小苏打可用于治疗胃酸过多、做发酵粉。
NaHCO₃ + HCl → NaCl + H₂O + CO₂↑
🔹 碱与盐反应（生成新碱和新盐）
反应物必须都可溶，且生成物中有沉淀或气体。
蓝色沉淀 Cu(OH)₂   红褐色沉淀 Fe(OH)₃
CuSO₄ + 2NaOH → Cu(OH)₂↓ + Na₂SO₄  |  FeCl₃ + 3NaOH → Fe(OH)₃↓ + 3NaCl
Na₂CO₃ + Ca(OH)₂ → CaCO₃↓ + 2NaOH （工业制烧碱）
🔹 盐与盐反应（生成两种新盐）
反应物都可溶，生成物中有沉淀。
用于鉴别Cl⁻、SO₄²⁻、CO₃²⁻等离子。
NaCl + AgNO₃ → AgCl↓ + NaNO₃  |  Na₂SO₄ + BaCl₂ → BaSO₄↓ + 2NaCl
🔹 酸与盐反应（生成新酸和新盐）
强酸制弱酸，或生成沉淀/气体。
用于检验Cl⁻（AgNO₃溶液）和SO₄²⁻（BaCl₂溶液）。
AgNO₃ + HCl → AgCl↓ + HNO₃  |  BaCl₂ + H₂SO₄ → BaSO₄↓ + 2HCl
配平方法
🔹 最小公倍数法
找出原子个数的最小公倍数，调整化学计量数。
适用 反应物和生成物各只有一种元素原子个数不同的反应。
P + O₂ → P₂O₅：左O=2，右O=5，最小公倍数10，4P + 5O₂ → 2P₂O₅
🔹 观察法
从组成较复杂的化学式入手，逐步推导其他物质的化学计量数。
适用 有明显特征的复杂化合物参与的反应。
Fe₂O₃ + 3CO →(高温) 2Fe + 3CO₂  |  从Fe₂O₃入手：Fe₂O₃中有2个Fe，所以Fe前配2
🔹 奇偶配平法
找出出现次数多的元素，先将其配平为偶数，再调整其他系数。
适用 某元素在方程式中出现次数多的反应。
C₂H₂ + O₂ → CO₂ + H₂O：左O=2(偶)，右O=3(奇)→乘2，2C₂H₂+5O₂→4CO₂+2H₂O
🔹 待定系数法
设各化学式前系数为未知数，列方程求解。
适用 较复杂的氧化还原反应。
aFe₂O₃ + bCO → cFe + dCO₂ → Fe₂O₃ + 3CO → 2Fe + 3CO₂
化学计算
🔹 质量守恒定律
参加化学反应的各物质的质量总和等于反应后生成的各物质的质量总和。
微观本质 反应前后原子的种类、数目、质量均不变。
注意 质量守恒 ≠ 体积守恒。
反应物质量总和 = 生成物质量总和  |  原子守恒
🔹 根据化学方程式计算
解题步骤：设未知数 写方程式 列比例 求结果
关键是找出已知量和未知量的质量关系。单位要统一，代入纯物质质量。
各物质质量比 = 化学计量数 × 相对分子质量之比
🔹 溶质质量分数与化学方程式
涉及溶液的反应，代入计算的是溶质的质量，不是溶液的质量。
m(溶质) = m(溶液) × ω
反应后溶液质量 = 反应前总质量 − 气体↑ − 沉淀↓。
ω = m(溶质) / m(溶液) × 100%  |  反应后溶液质量 = 反应物总质量 − 气体 − 沉淀
