中考数学知识点大全

中考总复习 · 系统梳理 · 济南考区适用

一、数与式

1. 实数

分类：按定义分有理数（整数、分数）和无理数（无限不循环小数）。按正负分正数、零、负数。

数轴：规定了原点、正方向和单位长度的直线。实数与数轴上的点一一对应。数轴上右边的数总比左边的大。

相反数：只有符号不同的两个数互为相反数。0 的相反数是 0。a+b=0 ⇔ a、b 互为相反数。

绝对值：数轴上表示数 a 的点与原点的距离，记作 |a|。性质：|a|≥0；|a|=a（a≥0），|a|=-a（a<0）。

科学记数法：a×10ⁿ（1≤|a|<10，n 为整数）。大数 n=位数-1；小数 n=第一个非零数前零的个数的相反数。

2. 实数的运算

运算律：加法交换律 a+b=b+a；加法结合律 (a+b)+c=a+(b+c)；乘法交换律 ab=ba；乘法结合律 (ab)c=a(bc)；乘法分配律 a(b+c)=ab+ac。

乘方与开方：aⁿ 表示 n 个 a 相乘。a² 的平方根为 ±√a（a≥0），算术平方根为 √a（a≥0）。立方根 ∛a。

混合运算顺序：先乘方开方，再乘除，后加减；有括号先算括号内。同级运算从左到右。

3. 整式与因式分解

整式运算：合并同类项（系数相加，字母及指数不变）；去括号法则（括号前是+"不变号"，是"﹣"全变号）。

幂的运算：aᵐ·aⁿ = aᵐ⁺ⁿ    (aᵐ)ⁿ = aᵐⁿ    (ab)ⁿ = aⁿbⁿ    aᵐ÷aⁿ = aᵐ⁻ⁿ（a≠0）

乘法公式：

平方差公式
(a+b)(a-b) = a²-b²
完全平方公式
(a±b)² = a²±2ab+b²

因式分解：提取公因式法 ma+mb+mc=m(a+b+c)；公式法（平方差 a²-b²=(a+b)(a-b)，完全平方 a²±2ab+b²=(a±b)²）；十字相乘法 x²+(p+q)x+pq=(x+p)(x+q)。

4. 分式与二次根式

分式：分母含字母的式子。分母≠0 时有意义。分式值为 0 ⇔ 分子=0 且分母≠0。基本性质：分子分母同乘（除）非零整式，值不变。

二次根式：√a（a≥0）。性质：(√a)²=a（a≥0）；√a²=|a|；√ab=√a·√b（a,b≥0）；√(a/b)=√a/√b（a≥0,b>0）。最简二次根式：被开方数不含分母、不含开得尽的因式。

二、方程与不等式

1. 一元一次方程

形式 ax+b=0（a≠0）。解法：去分母→去括号→移项→合并→系数化 1。应用题步骤：审→设→列→解→验→答。

2. 二元一次方程组

代入消元法：将一式变形成 y=... 代入另一式。加减消元法：两式加减消去一个未知数。解的一般形式：{x=a, y=b}

3. 分式方程

去分母（乘最简公分母）化为整式方程。⭐必验根：使最简公分母=0 的根为增根，舍去。

4. 一元二次方程

一般形式 ax²+bx+c=0（a≠0）。

解法：直接开平方、配方法、公式法、因式分解法（十字相乘）。

求根公式：x = (-b ± √(b²-4ac)) / (2a)

判别式 Δ=b²-4ac：Δ>0 两个不等实根；Δ=0 两个相等实根；Δ<0 无实根。

韦达定理：x₁+x₂=-b/a，x₁x₂=c/a（Δ≥0 时适用）。

5. 不等式（组）

基本性质：加减同一数不等号方向不变；乘除正数方向不变；乘除负数方向改变。

解集在数轴表示：≥或≤用实心点，>或<用空心圈。不等式组解集：同大取大、同小取小、大小小大中间找、大大小小无解。

三、函数

1. 平面直角坐标系

各象限符号：第一象限（+，+）、第二象限（﹣，+）、第三象限（﹣，﹣）、第四象限（+，﹣）。坐标轴上的点不属于任何象限。点 P(a,b) 到 x 轴距离 |b|，到 y 轴距离 |a|。

2. 一次函数 y=kx+b（k≠0）

k>0
过一、三象限，y 随 x 增大而增大
k<0
过二、四象限，y 随 x 增大而减小
b>0
与 y 轴正半轴相交
b<0
与 y 轴负半轴相交

正比例函数 y=kx（b=0，k≠0），图象过原点。求解析式：待定系数法（找两个点）。

3. 反比例函数 y=k/x（k≠0）

k>0
一、三象限，每个象限内 y 随 x 增大而减小
k<0
二、四象限，每个象限内 y 随 x 增大而增大

图象是双曲线，与坐标轴无交点。|k| 的几何意义：过双曲线上任一点作 x、y 轴垂线，矩形面积为 |k|。

4. 二次函数 y=ax²+bx+c（a≠0）

顶点式：y=a(x-h)²+k，顶点 (h,k)。一般式顶点坐标：(-b/(2a), (4ac-b²)/(4a))。

对称轴：x=-b/(2a)。a>0 开口向上，有最小值；a<0 开口向下，有最大值。

与 x 轴交点：Δ>0 两个交点；Δ=0 一个交点（顶点在 x 轴上）；Δ<0 无交点。

平移规律：左加右减（x），上加下减（y）。

四、几何

1. 线与角

平行线性质与判定：同位角相等⇔两直线平行；内错角相等⇔两直线平行；同旁内角互补⇔两直线平行。

垂线：过一点有且只有一条直线与已知直线垂直。垂线段最短（垂线段长度即点到直线的距离）。

角平分线：角平分线上的点到角两边距离相等。到角两边距离相等的点在角平分线上。

2. 三角形

内角和：180°。外角等于不相邻两内角和。任意两边之和＞第三边，任意两边之差＜第三边。

全等判定：SSS、SAS、ASA、AAS、HL（Rt△）。性质：对应边相等、对应角相等、对应线段（中线、高、角平分线）相等。

相似判定：两角对应相等、两边对应成比例且夹角相等、三边对应成比例。性质：对应角相等、对应边成比例、周长比=相似比、面积比=相似比的平方。

特殊三角形：等腰三角形（两腰相等、两底角相等、三线合一）；等边三角形（三边相等、三角均60°）；直角三角形（30°所对直角边=斜边一半、中线等于斜边一半）。

勾股定理：a² + b² = c²（直角三角形两直角边平方和等于斜边平方）。勾股数：（3,4,5）、（5,12,13）、（6,8,10）、（7,24,25）、（8,15,17）。

3. 四边形

平行四边形
对边平行且相等；对角相等；对角线互相平分
矩形
四个角90°；对角线相等且互相平分
菱形
四边相等；对角线互相垂直平分；每条对角线平分一组对角
正方形
四边相等、四个角90°；对角线相等垂直互相平分

中位线定理：三角形中位线平行第三边且等于第三边一半。梯形中位线平行两底且等于上下底和的一半。

n 边形内角和=(n-2)×180°，外角和=360°。正 n 边形每个内角=(n-2)×180°/n。

4. 圆

基本性质：垂径定理（过圆心垂直弦的直径平分弦及所对弧）。同弧所对圆周角相等，等于圆心角一半。直径所对圆周角=90°。

位置关系：点与圆（d<r 在内、d=r 在上、d>r 在外）。直线与圆（d<r 相交、d=r 相切、d>r 相离）。

切线的判定与性质：过半径外端且垂直于半径的直线是切线。切线垂直于过切点的半径。切线长定理：从圆外一点引两条切线长相等。

弧长与扇形：弧长 l=nπr/180。扇形面积 S=nπr²/360=lr/2。圆锥侧面积 S=πrl（l 为母线）。

5. 解直角三角形

锐角三角函数：sinA=对边/斜边   cosA=邻边/斜边   tanA=对边/邻边

特殊角三角函数值：

30°
sin=½，cos=√3/2，tan=√3/3
45°
sin=√2/2，cos=√2/2，tan=1
60°
sin=√3/2，cos=½，tan=√3

6. 图形的变换

平移：形状大小不变，各点平移方向距离相同。轴对称（翻折）：对应点连线被对称轴垂直平分。旋转：对应点到旋转中心距离相等，旋转角相等。中心对称：旋转180°，对应点连线过对称中心且被平分。

位似：对应点连线交于一点，位似比=相似比。位似图形面积比=位似比的平方。

五、统计与概率

1. 统计

数据的收集：普查（全面调查）与抽样调查（总体、个体、样本、样本容量）。

统计图：条形图（展示具体数量）、折线图（展示变化趋势）、扇形图（展示比例）。

统计量：平均数（x̄=(x₁+x₂+...+xₙ)/n）、中位数（排序后中间的数）、众数（出现次数最多的数）。方差 S²=Σ(xᵢ-x̄)²/n——反映数据波动大小，方差越大波动越大。

2. 概率

概率公式：P(A) = 事件 A 发生的结果数 / 所有等可能结果总数

取值范围：0≤P(A)≤1。P=0 为不可能事件，P=1 为必然事件。

计算方法：列举法（树状图、列表法）。频率估计概率：大量重复试验时，频率稳定于概率。

六、常用数学思想方法

数形结合：将代数问题转化为几何图形，或将几何问题用代数方法解决。

分类讨论：当问题有多种可能时，分情况讨论，不重不漏。

整体代入：将一个代数式视为整体代入求值，简化计算。

转化思想：将未知转化为已知，将复杂转化为简单，如分式方程化为整式方程。

方程思想：设未知数列方程，通过解方程解决问题。

函数思想：建立函数关系，利用函数性质解决问题（最值、增减性等）。

建模思想：将实际问题抽象为数学问题（方程模型、函数模型、几何模型等）。
