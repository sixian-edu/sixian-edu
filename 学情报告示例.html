一、错题分析

题目：解方程 x² - 5x + 6 = 0

学生作答：(x - 2)(x + 3) = 0 → x = 2 或 x = -3

正确答案：(x - 2)(x - 3) = 0 → x = 2 或 x = 3

错误定位

因式分解符号错误。(x - 2)(x + 3) 展开得 x² + x - 6，与原式 x² - 5x + 6 对比，一次项系数和常数项均不匹配。

错误根源

对十字相乘的符号规律理解不牢固。常数项为正时两个因式常数项同号，一次项系数为负时两个因式常数项都为负。此外缺少验算习惯——展开后与原式不符却没有发现。

正确解题步骤

  • 确定系数：a = 1, b = -5, c = 6
  • 找两数相乘得 +6、相加得 -5 → 同为负 → -2 和 -3
  • 因式分解：(x - 2)(x - 3) = 0
  • 零积性质：x - 2 = 0 或 x - 3 = 0
  • 解得：x = 2 或 x = 3
  • 验算：2² - 5×2 + 6 = 0 ✓，3² - 5×3 + 6 = 0 ✓
