## 七、Share Wall 模式（所有页面统一）

所有页面有完全一致的 share overlay 实现：

```
localStorage 键:
  sx_shared   — 是否已分享（解锁联系方式）
  sx_copy_time — 复制文案的时间戳（判断是否过了10秒等待）

流程：
  点击"获取联系方式" → 弹窗显示分享文案
  → 点击"复制分享文案" → 存入 sx_copy_time + 启动10秒计时器
  → 10秒后解锁"我已分享"按钮
  → 点击"我已分享" → 存入 sx_shared → 所有 .contact-trigger 替换为电话号码
```

⚠️ **新增页面时**，必须复制完整的 share overlay HTML + JS，否则联系方式无法解锁。
